  1. Plugins

    Activate Plugins

    Plugin activation

    Plugins are included in the osfinancials installation. By default, some Plugins are not available from the Plugins menu on the Setup ribbon. 

    To add plugins to the Plugins menu, you need to activate the plugins, you require. This will update the "Plugins load list". Activated Plugins will be available on the Plugins menu.

    To activate plugins:

    1. On the Setup ribbon, select Tools → Activate plugins. 
    2. On the "Plugin load list" screen, you may select to activate / deactivate, the plugins you have purchased on the list.

    1. Click on the Close button.
    2. The selected plugins will be added to the Plugins menu on the Setup ribbon. 
    3. Plugins needs a licence before these can be activated. Some plugins, if not registered (purchased on the Webshop), can be used for a number of transactions or records only.

    Using plugins 

    Once the plug-ins are activated, you must enable (enable) the plug-ins you purchased in the Plugins menu (Setup ribbon). 

    Some plug-ins must be activated for each Set of Books in osFinancials.

    All plug-ins are enabled in the Tools → Activate plugins (Setup ribbon) and will be available in the Plug-ins menu. However, there are two exceptions:

    1. Bank Import Plugin - This plugin must be enabled in the Setup → System parameters (Setup ribbon).
    2. Payroll Import Plugin - This plugin is accessed in the Process batch option (F9:Process), provided you have a valid, and you have a valid input file from a supported payroll software.

    Start osFinancials again to fully activate the enabled plugins.

    Retrieving plugin licences

    To retrieve plugin licences:

    1. Go to the Start ribbon (tab). Then click on Online registration. 
    2. On the “Register your osFinancials” screen, click on the Get it online button.
    3. On the next screen, enter your username and password for this website. 
    4. Then click Send (bottom of screen) and your plugin codes are retrieved from our servers.
    5. You will see all the plugins you purchased or plugins you have asked for a trial. 

    At the bottom of the "Your registration is successful!" with your registration codes section of the "Register your osFinancials" screen, your purchased plugin(s) will show the name of the plug-in(s) well as the maximum number of times you can retrieve the plug-in(s) and the end date on which the licence expires (in the case of time licence plug-ins) are displayed. 

    If the counter of the number of remaining times to renew the licence is at 0, then the plugin is blocked. To reset the counter, you can send us an email with the request to reactivate the plugins.

    If the plugins are about to expire, you will be notified. osFinancials will automatically send a quotation for extending the usage period. 

    This will happen before the actual expiry date.

    So if the quotation is paid, the plugin(s) are extended in period after the original expired date.

    So you won't need to see the notice messages until the expiry date.
    1. Click on the Close button.

    You only can use plugins with the serial number/account of the osFinancials account that you purchased the plugins with.

    The plugins can not be combined with other plugins purchased using a different account.

    This could happen if you have registered multiple-times.
