    Reports - userreports - Graphs file names

    User reports menu 

    These user reports are stored in the “...\plug_ins\reports\userreports” folder of your installation directory. These user reports are included in the osFinancials5.1 installation. 

    When printing reports using the “Material” theme, the icons in the “Material” theme folder will be displayed in menus and the icons in the “Classic” theme folder will be displayed on the report parameters screens and in "Report man". 

    For your convenience, a list of the file names and structure of the reports in the User reports menu (Reports ribbon) is as follows:
