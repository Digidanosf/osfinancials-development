    Exported documents - Headers files in Spreadsheets

    If you have saved the documents for the selected periods as a "MS Excel SpreadSheet (*.XLS)" file type, you can open this export file for the document header ("Header") in a spreadsheet.

    An example of the exported file opened in "LibreOffice Calc" spreadsheet, is as follows: 


    The exported document header data will be exported in forty-two (42) columns:

    Column

    Column heading

    Source / fields

    Notes

    A

    WDOCID

    Document ID 

    Integer (Auto generated for each new document created or imported).

    B

    WTYPEID

    Document type ID

    Integer

    10 = Invoice

    11 = Credit note

    12 = Purchase

    13 = Supplier return

    14 = Quote

    15 = Order

    16 = Stock adjustment

    Date

    Document no.

    String (8) Prefixes added in templates followed by 6 numeric digits.

    QU = Quote

    OR = Order 

    ST = Stock adjustments (Hard coded)

    F

    SREFERENCE

    Your reference

    0 = Document never been printed
