    Spreadsheet - Import - Stock items

    This option will allow you to import your stock code file from a valid an Excel spreadsheet in the Excel Workbook (xlsx) file format. You may then add new stock items, edit existing stock items and import it back into your stock code file.

    To import stock items from a Excel Workbook (xlsx) file: 

    1. Open the Set of Books in which, you wish to import stock items from a valid a spreadsheet export file.
    2. On the Setup ribbon, select Import. The "Import Data into osFinanacials" screen is displayed: 

     

    1. Select Stock as the "Data type to import".
    2. Click on the Excel import export button.  The "Data Import Export" screen will be displayed with the column headings for the Stock items (without any data).

    1. On the "Data Import Export" screen, click on the Open icon to select a valid Excel Workbook (*.xlsx) file. 

    You may click on the "Show preview pane" icon on the "Open" screen to display a preview of the selected file to open (import) in the "Data Import Export" screen..
    1. The data of the Stock items included in the valid Excel Workbook (xlsx) file, will be imported (populated) into the spreadsheet ("Data Import Export" screen).

    1. Click on the Import icon. 

    1. You may add new stock items, edit the stock items, change the quantities, selling prices, whatever field, is necessary. 
    2. Once you are done, check the following options: 
      1. Import check 
      2. Profile  
      3. Import type - 
    1. Click on the Import icon. 

    Optional - Show data before import setting 

    The "ImportCheck" screen (if the Import the "Show data before import" option was selected (ticked) on the "Data Import Export" spreadsheet screen) is displayed:

    Click on the All or OK buttons to start the import process.  

    1. Once the import process is finalised, Column A will be in a green background colour.
    2. Close the "Data Import Export" spreadsheet screen.  
    3. You may to the Default ribbon and select Stock items to edit the stock items, if necessary.  
    4. Go to the Reports ribbon and select Reports → Stock items → Stock item listing to check the stock items.
