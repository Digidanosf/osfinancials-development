    Report designer - Customise layout files

    In osFinancials, you have the option to further customise document layout files using the Report designer feature available on the Setup ribbon. This functionality provides users with additional flexibility and control over the appearance and content of their financial documents.

    Here's how you can customise document layout files using the Report designer:

    1. Select Existing Layout or Your Own Layout File: In osFinancials, the Report designer allows you to choose from existing layout files or create your own layout from scratch. You can select a pre-defined layout file that closely matches your requirements and then make modifications as needed. Alternatively, you have the freedom to start with a blank layout file and build it according to your specific preferences.
    2. Customize Layout Elements: Using the Report designer, you can customize various elements of the document layout. This includes adjusting the placement and formatting of fields, headers, footers, company logo, transaction details, and any other relevant content.
    3. Add or Remove Fields: You can add or remove fields as necessary to display the information you want on the document. This allows you to include or exclude specific details that are relevant to your business and industry.
    4. Formatting and Styling: The Report designer provides options to format and style the document layout. You can choose font styles, colors, borders, and other visual elements to match your branding and create a professional look.
    5. Preview and Test: As you make changes in the Report designer, you can preview how the document will look before finalizing the layout. This allows you to test the design and ensure that it meets your expectations.
    6. Save and Apply Layout: Once you are satisfied with the customizations, you can save the layout and apply it to the respective document type. The modified layout will then be used when generating documents of that type in osFinancials.

    By providing the option to customize document layout files using the Report designer, osFinancials empowers businesses to create personalized and professional financial documents that align with their unique needs and preferences. This feature adds another layer of adaptability and customization to the accounting software, enhancing the user experience and ensuring that documents reflect the company's branding and style.
