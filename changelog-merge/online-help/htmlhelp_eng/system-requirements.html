    System requirements

    The system requirements for osFinancials version 5 are as follows:

    1. Operating System: A 32-bit or 64-bit Microsoft Windows operating system is required. Supported versions include Windows 98, Windows 2000, Windows ME, Windows NT, Windows XP, Windows Vista, Windows 7, Windows 8, Windows 10, or Windows 11.
    2. RAM: For a single-user setup, at least 256 MB of RAM is required. For a multi-user setup, a minimum of 512 MB of RAM is recommended.
    3. Hard Disk Space: Approximately 1 GB of available hard disk space is needed to install and run osFinancials.
    4. USB Port and Drives: A USB port is required, and available USB drives are needed to save and store backups.
    5. Screen Resolution: An SVGA/LCD/LED screen with a resolution of 1440 x 900 or higher is recommended.

    Screen Resolution Adjustments: If your device has a screen resolution of 1366 x 768, you may need to adjust your Screen resolution when working in forms or screens such as Documents, Debtors, Creditors, Stock, etc. using the following options:   

    • Collapse the ribbons (automatically hide the ribbons) 
    • Set the "Style values for ribbon" to Tablet mode.

    This adjustment ensures proper display and functionality.
    1. System Number Format: Configuring the System number format setting to print reports like “Budget vs actual” and “This year vs last year”.
    2. Spreadsheet Software: Microsoft Excel or other spreadsheet programs such as LibreOffice, OpenOffice or UltraOffice should be installed to save reports or export data from the Ledger analyser and other options.
    3. Recommended Device: While osFinancials5 may run on older devices, it is recommended to use a better device for more secure and efficient operation. Modern devices are significantly more powerful than older ones, making them more suitable for running osFinancials smoothly.

    Please note that the information provided here is specific to osFinancials version 5. Always refer to the official documentation and system requirements provided by the osFinancials team to ensure accurate and up-to-date information. Additionally, consider using the most recent and supported operating system and hardware for optimal performance and security.
