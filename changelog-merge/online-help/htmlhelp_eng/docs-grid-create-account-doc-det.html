    Context menu - Create account from document details

    This documents context menu allows you to create a new debtor (customer / client) account for unposted sales documents (i.e. Invoices, Credit notes or Quotes) or an new creditor (supplier / vendor) account for unposted purchase documents (i.e. Purchases, Supplier returns or Orders) using the document details. 

    This may be necessary if a Document is imported and the account does not exist. 

    Create a new debtor (customer / client) account using the document details

    To create a new debtor (customer / client) account using the document details:

    1. Select the Invoices, Credit notes or Quotes document type.
    2. Select the document to create an account for.
    3. Right-click and select Documents â†’ Create account from document details option on the context menu. The "Debtor accounts" edit form with the details on the document header is displayed:

    1. Enter a "Debtor code" and enter or edit the fields, as necessary. 

    Debtor code - Enter a prefix followed by 5-digits. In this example, the Debtor code is prefixed with "1".  Creditor codes are prefixed with "2" in this documentation. 

    This is important to distinguish accounts for debtors (customers / clients) from accounts for creditors (suppliers / vendors). In some features and on some screens and reports this makes it easier to identify the account type.  
    1. Click on the OK or Save button. A confirmation message will be displayed.

    Create this account?"

    1. Click on the Yes button to continue. 

    Once the New account is saved the document will be listed in the Documents tab of the new debtor (customer / client) account.
