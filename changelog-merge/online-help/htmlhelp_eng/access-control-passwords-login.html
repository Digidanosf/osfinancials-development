    Enter passwords (Login)

    If you need to Open a Set of Books or Delete a Set of Books, the "Login" screen to allow the user to enter a valid username and password password for the Set of Books: 

    Enter a valid password and click on the OK button.

    The password is an eight-character alphanumeric and special character field. The password is also Case sensitive. Remember if you use Capital alpha characters. Once a user password is set, you will not be able to Open a Set of Books or to Delete the Set of Books, if you do not enter the password absolutely correctly.

    Should you enter an incorrect username and password, an error message is displayed:                                                             

    Invalid password!

    Click on the OK button and re-enter the correct password. Check that your Caps Lock key is not accidentally switched on.

    After the third attempt when an invalid username and/or password is entered, the Set Books (open) screen will be automatically closed.

    To use (enable) passwords, you need to create at least two (2) users with passwords.

    If users have been created and passwords are set for the users, a login screen for Point-of Sale Invoicing access will be displayed on which the Cashier or Salesperson needs to enter a valid username and password.
