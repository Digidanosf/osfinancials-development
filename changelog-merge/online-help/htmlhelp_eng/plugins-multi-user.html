    Say hello to seamless collaboration and goodbye to constraints with the Multi-User Plugin for osFinancials/TurboCASH.

    The Multi-User Plugin for osFinancials5 offers the capability to configure and set up a network server to accommodate multiple users. However, to unlock the full functionality for multi-user access, purchasing a multi-user licence and registering osFinancials as a multi-user product is necessary.

    Documentation is provided for installing and configuring multi-user setups on both Windows and Linux servers, which can assist users in setting up their network environment effectively.

    It's important to note that the multi-user functionality is limited in this edition, indicating that additional licensing is required to fully utilize it. Therefore, acquiring a multi-user licence is essential for organizations or teams requiring simultaneous access to osFinancials5.
