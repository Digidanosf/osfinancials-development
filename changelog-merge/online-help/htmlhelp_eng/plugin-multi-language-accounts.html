    Multi-language accounts

    Multi-language Set of Books plugin

    Multi-language Set of Books plugin - Manual - Shop - Licence : Once-off -

    This plugin makes it possible to translate the ledger and group names in multiple languages. It is already possible to choose different languages for the interface (the buttons). But this plugin makes it possible to do the accounting in different languages. Per user you can select a language, so the user can account in his/her own language. The set of books can be viewed in different languages, therefore it is possible an employee enters information into an English Set of Books and a bookkeeper, auditor, etc. can see the set of books in a set language, e.g. Afrikaans, Dutch, etc.

    The new multi-language ledgers, groups, types and batches is probably best feature implemented in osFinancials.

    This makes it possible for an international company to work in different languages on the same set of books.

    This helps to show local investors in their language the balances and turnover of the company. 

    The multi-language feature allows different users to use the same set of books in the user language interface. The translation of ledger accounts need to be purchased from the shop. 

    By default, each Set of Books will run in the language associated with the Set of Books when opening a Set of Books. You may, if necessary (i.e. incorrect language is selected) or when you need to work in a different language, use Start → Switch language to change the language.

    Any language may be customised / edited for your specific requirements on the Setup ribbon, select Tools → Customise language.

    In addition to this, you may activate the Multi-Language feature (using any of the available languages in the Start → Switch language) in a Set of Books to:

    • Create a Multi-Language User Interface. This makes it possible for a multi-lingual company to work in different languages on the same set of books.
    • Create a Multi-Language Accounts (Chart of accounts) - This helps to show local investors in their language the balances and turnover of the company.

    System Language will list all the available languages in the Start → Switch Language menu.

    Any language may be customised / edited for your specific requirements on the Setup → Tools → Customise Language menu.

    The available languages is stored in the ... \ bin \ languages folder of your osFinancials5 installation directory. The default language is selected "System lang" field on the Standard tab of Setup → Access control . This is the language in which osFinacials5 will open the Set of Books.
