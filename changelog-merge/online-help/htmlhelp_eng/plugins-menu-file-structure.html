  1. Plugins

    Plugins menu (Setup ribbon)

    Plugins menu structure

    Plugins are included in the osFinancials5 installation. By default, some Plugins are not available from the Plugins menu on the Setup ribbon. 

    The following plugins in the osFinancials5 subscription, are activated (updated in the "Plugins load list") and will be available on the Plugins menu on the Setup ribbon:

    1. Document plugins → Document E-mail Pro 
    2. Generic plugins → Multimedia 

    Access to the Plugins menu

    On the Setup ribbon, select Plugins. 

    When all Plugins are activated from the Tools → Activate plugins option on the Setup ribbon, the Plugins menu is structured and grouped into seven categories: Document plugins, External system links, Financial tools, Generic plugins, Point-of-Sale, Stock plugins, and Tools.

    Any other Plugins, except the Bank import plugin, needs to be activated to be listed on Tools → Activate plugins menu on the Setup ribbon. This will update the "Plugins load list".

    Some of the Plugins has a nag screen, some plugins are limited to a number of records, etc. Plugins need to purchased from the osFinancials online shop and need to be registered (on the Setup ribbon, select Tools → Activate plugins), before you can use them.

    Some plugins need to be activated for each Set of Books. If you have purchased a plugin, you may use it in any Set of Books.

    Plugins included in the osFinancials5 subscription

    Bank import 

    Bank Import Plugin Licence

    The Bank Import Plugin is included with the osFinancials5 Business class subscription. With a registered osFinancials5 Business class subscription, you gain access to import and process unlimited statements for each of your bank accounts.

    Upon purchasing the osFinancials5 Business class subscription, you will receive an email containing registration (activation) codes and detailed instructions.

    For unregistered versions of osFinancials5, a message may appear: 

    "This demo plugin allows %s records to be entered. If you need more, please buy the product at https://www.osfinancials.org/en/webshop".

    We encourage users to support osFinancials by purchasing the osFinancials5 Business class subscription. This not only unlocks the full functionality of the Bank Import Plugin but also supports ongoing development and improvements.

    For users of osFinancials 4, the Bank Import Licence is available for purchase from our osFinancials webshop.

    The Bank Import plugin included with the osFinancials5 Subscription is a handy tool for managing and processing the transactions in your bank and credit card accounts. You can download electronic bank statement files directly from your online banking services at your bank or financial institution. After that, you can use the Bank Import plugin to import these bank statements into your bank accounts. This process saves you a lot of time and effort as you do not have to manually enter and process the transactions of your bank accounts and credit card accounts in separate Payment journals and Receipt Jjurnals. 

    Additionally, the Bank Import Plugin has the ability to use bank references to automatically assign transactions to the appropriate accounts, This automation not only saves time, but also reduces the risk of human error associated with manual data entry. Overall, using this Bank Import plugin is a smart choice for businesses or individuals who want to optimize their financial workflow within osFinancials.

    Budget Plugin

    Budget Plugin - Manual - Shop - Licence : Included in osFinancials5 Subscription. 

    Documentation : Help topic. 

    Effortlessly manage budgets for accounts, cost centres, and projects with the Budget Plugin, now included as a standard feature in the osFinancials5 Subscription. Access the Budgets option on the Reports ribbon for seamless integration.

    Key Features:

      • Flexible Budget Entry: Easily input budgets for accounts, cost centres, and projects directly within the osFinancials5 interface. Copy budget figures from the previous year of actual account balances.
      • Comprehensive Budgeting: Cover all aspects of financial planning by setting budgets for various entities, including accounts, cost centers, and projects.
      • Integrated Solution: Seamlessly integrate budget management into your overall financial workflow within osFinancials5.
      • Advanced Filtering and Pivot Options: Utilize advanced filtering and pivot functionalities to analyse budget data effectively and gain deeper insights into your financial performance.
      • Streamlined Processes: Simplify budgeting tasks and ensure accuracy by leveraging the intuitive interface and advanced features of the Budget Plugin.
      • Comprehensive Reporting: Generate budget listing reports (Trial balance, Income statement, Balance sheet, or Standard column balances report types) and budget performance reports (Budget vs Actual to compare the budget figures versus the actual posted transactions). Export reports to spreadsheets and utilize powerful pivot features for in-depth analysis. Print and export budgets in the T-Account viewer.

    Enhance your financial planning capabilities and streamline your budgeting processes with the Budget Plugin, empowering you to make informed decisions and drive financial success within your organisation.
