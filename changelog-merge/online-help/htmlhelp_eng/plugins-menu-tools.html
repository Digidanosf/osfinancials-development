    Reportman

    Reportman - Manual - Licence : Free. 

    Accessible from the Report designer icon on the Setup ribbon

    Translations : Translations outstanding 

    Unlock the power of report generation and customization with our Reportman Plugin, provided free of charge. Accessible from the Report designer icon on the Setup ribbon, this plugin empowers users to design and print reports and document layout files effortlessly. All reports and document layout files included in the osFinancials installation are designed with the powerful Reportman Plugin.

    Compatibility with Database Types:

    The Reportman plugin seamlessly integrates with osFinancials, allowing users to edit and write reports and document layout files across different database types. Whether your Set of Books in osFinancials relies on Firebird or MSSQL database types, the Reportman plugin ensures smooth functionality and compatibility.

    Key Features:

    • Powerful report generator and designer
    • Customize existing reports or create your own
    • Design document layout files for various sales and purchase documents
    • Includes default layout files and reports in the osFinancials installation

    Best Practices:

    • Always make a copy of a report or layout file before editing. You can always fall back on the original report or layout if things go wrong.
    • Rename your layouts or reports to prevent overwriting during reinstallation or during installation of osFinancials updates.
    • Add your customised reports to the User reports menu

    Documentation and Resources:

    • osFinancials Help and Online documentation: Comprehensive detailed information provided in the "Working with Reportman" section of the osFinancials5 Help file and on the osFinancials website in English and Afrikaans.
    • Official offline documentation: Navigate to the "...\plug_ins\reportman\doc" folder within your osFinancials5 installation directory. Open the index.html file located at ".../plug_ins/reportman/doc/index.html" in your web browser to access this resource.
    • Official online documentation: For additional resources and information regarding the Report Manager, refer to the official online documentation available at https://reportman.sourceforge.io/. This online documentation offers comprehensive guidance and support for maximizing the functionality of the Report Manager.

    Harness the capabilities of the Reportman Plugin to create tailored reports and document layouts while maintaining flexibility and control over your printing needs.

    SQLBrowser

    SQLBrowser - Manual - Shop - Licence : Once-off -

    Documentation : Help documentation  

    Translations : Outstanding  

    The SQL Browser plugin in osFinancials empowers users with a robust tool for interacting with their data using Structured Query Language (SQL). Tailored for users familiar with SQL, this feature enables the execution of custom queries, effortless grouping, filtering, and sorting of results. 

    You can save their SQL Script files for future use and export data to various formats including XML (Extensible Markup Language), HTML (Hypertext Markup Language), Microsoft Excel, and Text File formats. Additionally, users can seamlessly export data to "JSON"-files and create a new table from a "JSON"-file, expanding the flexibility and capabilities of data management within osFinancials.

    Compatibility with Database Types

    The SQL Browser plugin seamlessly integrates with any Set of Books utilizing either Firebird or MSSQL database types. Whether your organisation relies on Firebird or MSSQL for database management, the SQL Browser provides a unified platform for accessing and manipulating data.

    This flexibility allows users to leverage the SQL Browser's powerful features across different database environments, ensuring consistent functionality and ease of use regardless of the underlying database technology.
