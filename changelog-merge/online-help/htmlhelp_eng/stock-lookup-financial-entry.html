    Item lookup - Financial entry

    If you have added a Financial entry stock item type, you may select a general ledger account, debtor (customer / client) account or a creditor (supplier / vendor) account from the Accounts lookup when processing documents.  

    To select the Financial entry item in documents:

    1. On the "Document entry" screen, select the "Financial entry" stock type code.

    1. On the "Accounts" lookup, select a general ledger account, debtor (customer / client) account or a creditor (supplier / vendor) account.

    Disabled (blocked) accounts will not be available in the "Accounts" lookup. 

    • Ledger accounts - Status set to disabled.
    • Account disabled selected (ticked) - Accounting information tab of debtor (customer / client) accounts. 
    • Account disabled selected (ticked) - Accounting information tab of creditor (supplier / vendor) accounts. 

    Do not select Main accounts. For example, an error message will be displayed if an account is selected which contains sub-accounts and the document is posted:

    You cant post a account With subaccounts! 
    IN000023        Motor vehicle expenses                210-000

    You need to select a valid sub-account, for example:

    • G210010 - Fuel and oil
    • G210020 - Maintenance
    • G210030 - Repairs
    1. Once an account is selected, it will display on the Document lines as follows:

    1. The selected account code will be displayed in the "Item no." column and the account description will be added in the "Description" column. 

    You may enter comments (F9: Comment) and/or remarks for additional information regarding this item.

    If comments were added, these will also be available in the Search - Investigator. 

    Remarks will not be searchable in the Search - Investigator. 
    1. Once a document containing a financial entry stock item type, is posted to the ledger, you may search for the stock item description in the Search - Investigator.
