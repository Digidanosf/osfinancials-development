    FIXED - Reports layout long descriptions and data - Layout files-Labels-Statements

    Company data " SCompanyName {chartype}(64) "

    Company name supports up to 64 characters in Setup → Company info - this will print ok on most reports. In the case of Creating a Set of Books from scratch, using the "Customise your own Set of Books (Advanced)?" option - for 3-Digits to 8-Digit main accounts in the Chart of Accounts, the Company data " SCompanyName {chartype}(64) " field in the SYSVARS table allows for 64-characters to be entered. Only about 42-characters will print on the debtor statements, creditor remittance advices, layout files. More than 42-characters may not print or overlap with the address details. 

    The following reports and layout files is adjusted:

    The combined expressions will only allow to set the FontStyle to Bold or Regular
