    Report designer - Writing SQL

    To write SQL is actually very simple, for example, “select data from table”,  where data is a valid field name (e.g. ScompanyName Company Name) in the table and table is a valid table name (e.g. Sysvars). 

    To call the company name and Fax number (as entered in the Company info (Setup ribbon) screen (it is stored in the sysvars table)), you may simply write select SCompanyName CompanyName, SFaxNumber Fax from Sysvars.  

    Select YOUR DATA FIELDS  from sysvars
