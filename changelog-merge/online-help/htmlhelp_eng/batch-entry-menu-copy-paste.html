    Batch entry - Context menu - Copy

    Copy the "Contra account, Date, Reference" or "Description" from one transaction in a batch to another transaction.

    To copy and paste Contra account, Date, Reference or Description from a transaction in a batch to another transaction:

    1. Select a transaction to copy from.
    2. Right-click and select the Copy → Copy from / init  → Contra account, Date, Reference or Description options on the context menu. 
    3. Select the transaction to paste the copied field to.
    4. Right-click and select the Copy → Paste to / perform  → Contra account, Date, Reference or Description options on the context menu. The selected field's content will be pasted or replaced in the selected transaction.
