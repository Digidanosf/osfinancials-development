    Copy / Paste

    You may copy the specific data of a selected stock item and paste (replace) the data of selected stock item(s). 

    To copy and paste data from a stock item to other stock item(s): 

    1. On the Default ribbon, select Stock items.
    2. Select the stock item, from which you need to copy data.
    3. Right-click and select the "Copy from / init" (initiate) option on the context menu. 
    4. Select the stock item(s) to which you wish to paste the data.

    If you need to select multiple stock items, press the Ctrl or Shift key and click on each stock item, for which you need to paste (replace) the selected data.
    1. Right-click and select the "Paste to / init" (initiate) option on the context menu. 

    Stock item - This option is another option to the Copy from item button. This will launch a new Stock item edit form with the data of the selected source stock item. You need to enter a stock code and edit the copied stock item, where necessary. 

    Copy / Paste Supplier 1

    The preferred supplier 1 account as selected on the selected source stock item will be copied to (replace) the "Preferred supplier 1" field of the selected stock item(s). 

    When the "Paste to / init" (initiate) option on the context menu, is selected, A confirmation message will be displayed:

    Replace old data for &Supplier 1? 

    The Supplier stock code for preferred supplier 1 will not be copied (replaced).

    Copy / Paste Ledger and Tax

    The Tax class, Tax (Input tax and Output tax) and Ledger (Sales account, Cost of sales and Stock control) accounts as set on the Ledger tab of the selected source stock item will be copied to (replace) the Tax class and Input tax Output tax and Ledger accounts of the selected stock item(s). 

    When the "Paste to / init" (initiate) option on the context menu, is selected, a confirmation message is displayed:

    Replace old data for &Ledger and tax?


    Copy / Paste Multi-group

    The reporting groups as set on the Multi-group tab of the selected source stock item will be copied to (replace) the groups of the selected stock item(s). 

    When the "Paste to / init" (initiate) option on the context menu, is selected, a confirmation message is displayed:

    Replace old data for M&ulti-group copy?
