    Spreadsheet reports menu

    Spreadsheet reports osFinancials and the power of Spreadsheets 

     - Spreadsheet sales (Default ribbon). 

     - Spreadsheet reports (Reports ribbon).

    See - Video on Spreadsheet reports.

    These spreadsheet reports are stored in the “...\plug_ins\reports\excelreports” folder of your installation directory. 

    Selected reports will be opened in the powerful spreadsheet feature in osFinancials5. 

    When printing reports using the “Material” theme, the icons in the “Material” theme folder will be displayed in menus and the icons in the “Classic” theme folder will be displayed on the report parameters screens and in "Report man". 

    Spreadsheet reports menu - File structure and file names
