    Languages

    The Language groups allows you to add Languages. These languages can be selected in the Language field on Access control (Standard security tab) on the Setup ribbon. 

    It will add a Languages tab on the Stock items (Default ribbon).  

    Multi-Language accounts - Manual - Shop - Licence : Once-off -

    This plugin makes it possible to translate the ledger and group names in multiple languages. It is already possible to choose different languages for the interface (the buttons). But this plugin makes it possible to do the accounting in different languages. Per user you can select a language, so the user can account in his/her own language. The set of books can be viewed in different languages, therefore it is possible an employee enters information into an English Set of Books and a bookkeeper, auditor, etc. can see the set of books in a set language, e.g. Afrikaans, Dutch, etc.

    To add / edit /delete Language groups: 

    1. On the Setup ribbon, select Setup â†’ Groups. The "Groups (Account group 1)" default screen will be displayed.
    2. Select "Languages". 

    If you have created a Set of Books from scratch (using the "Create Set of Books (Advanced)" option for Firebird database types or the "Create Set of Books (Advanced)"  option for MSSQL database types), the Languages group will not be available.   

    You need to manually add the Project group to the TYPES table 

    See topics in this Help file : - 

    • Steps after creating the Set of Books (Advanced) - Add missing Groups and Paper types in the Types table.
    • Reportman - References - Groups.

    You may set different sequences for each reporting group in which the languages are listed on reports, etc.  This is done by selecting the groups on the list and use the Move up or Move down buttons.
    1. If you wish to rename the Reporting group name - Languages, over-type the description with your new description. This will change the description of the button for Languages.
    2. To add a new Group to the root directory or list, select the "REPORTING GROUP ROOT" on the list and click on the New button.
    3. Enter the name of your next Language and click on "REPORTING GROUP ROOT" on the list. 
    4. Repeat the process until you have entered all the Languages, etc. 
    5. Click on the OK button to save your changes and close (exit) this screen.

    You may click on the Print button to print a list of your languages.
