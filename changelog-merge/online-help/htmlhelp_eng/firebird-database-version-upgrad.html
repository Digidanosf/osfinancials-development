    Firebird database upgrade

    Before you upgrade (Backup and Restore) the Database, use the make a copy or backup of the Set of Books to be upgraded.

    In FlameRobin, after you have added a database with the Register existing database option, you may check the Firebird version of the database.

    To do this connect to the database and select Database Properties from the Database menu (or from the context menu). 

    Firebird database ODS version 10.1 = Firebird version 1.5 


    If you wish to upgrade the database (Set of Books) from Firebird1.5 (ODS Version 10.1) to Firebird2.1 (ODS Version 11.1), you need to do the following:

    1. Disconnect the database in FlameRobin.
    2. Right-click and select the Backup database... option. 

    1. Backup the Database 
    2. Enter a "Backup file" name. 

    1. Restore the Database - Use the same "Backup file" name. 

    1. Once the Database is restored, connect to the Database and check the Database version. 

    It should display Firebird database ODS version 11.1 = Firebird version 2.1
