    Export - Stock items (Setup ribbon)

    This feature is additional to the powerful Report Export facility in which all reports for Stock items in osFinancials5 may be exported, and saved in any available format on your system, such as Word Processing (Text), Spreadsheets, etc. Using this facility, you may also export the data of your Stock items in an open Set of Books to XML Database format, or any available format on your system. 

    You may then open the other applications to which the exported data is saved, and work in the applications, such as OpenOffice.org Calc, Lotus 123, Microsoft Excel, etc. You may edit the data for your Stock items in a Lotus123 (*WK1) file format and import it in Stock items (Default ribbon). 

    All the information regarding each stock item as entered in the Stock items (Default ribbon) (Stock code, Barcode, Description, Re-order levels, Apply invoice discount, Selling prices, Unit Description, Cost of sales account, Sales account, Input Tax and Output Tax accounts) will be displayed. In the same export file, the latest updated information regarding any adjustments and transactions (Quantity on hand and Average cost) would also be included into the selected export file format. 

    You may edit the data for your Stock items in a Lotus123 (*WK1) file format and import it in the Stock items (Default ribbon). 

    To export Stock items: 

    1. On the Setup ribbon, select Export. 
    2. Click on the Stock button, if not selected. 
    3. Select and/or enter the following: 
      1. Export to - The installation directory in which osFinancials5 is installed, will be displayed. You may click on the Browse icon to select a folder on the "Browse for Folder" screen. You may then select any available folder, or drive, on your system into which you wish to export the data.
      2. File type - Select to export the file as "Tab delimited" (*.txt) file format (recommended).

    "Import file format not implemented!" information message is displayed for the following file types:

    • TXF TurboCASH Exchange File
    • SXC OpenOffice Spreadsheet

    The XML Database (*XML) file format export can only be imported in Import (Setup ribbon). It cannot be imported in Stock items (Default ribbon).
      1. Export files name - Enter the File name. You need to select this file at a later stage when you wish to import the file, or analyse the data in a spreadsheet.
      2. Log file - Enter the Log file name.
      3. Range - From / To - Select the stock items to include in the export file. 
      4. Increment Export - Leave this option blank.
    1. Click on the Export button to create the export file, and save it in the specified directory.

    The information for the Stock items, which will be exported, is: 

    • Stock code 
    • Stock item type 
    • Barcode 
    • Description 
    • Quantity on hand 
    • Reorder quantity 
    • Apply invoice discount 
    • Selling price 1 amount 
    • Selling price 2 amount 
    • Selling price 3 amount 
    • Unit average cost amount 
    • Unit cost amount 
    • Unit description 
    • Stock group 1 
    • Stock group 2 
    • Cost of sales account 
    • Sales account code 
    • Stock account 
    • Input Tax account 
    • Output Tax account
