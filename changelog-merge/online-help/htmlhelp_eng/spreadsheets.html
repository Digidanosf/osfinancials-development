    Excel Export / Import - 

    See video on Improved Export/Import - 

    The Excel Export / Import spreadsheet feature allows to easily export data into spreadsheets, edit the data and import it. 

    Excel Export / Import is available in the following options:

    Setup ribbon: 

    • Setup → Accounts 
    • Setup → Groups 
    • Import Accounts, Debtors/Creditors, Stock 

    Default ribbon 

    • Batch entry (all batches) 
    • Edit → Repeating transactions
    • Stock (Import / Export tab)
