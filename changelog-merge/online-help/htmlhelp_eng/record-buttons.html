    Record buttons

    There are ten (10) record buttons are used to manage records some of the following screens:

    • Setup â†’ Accounts (Setup ribbon)
    • Stock items (Multi-group) tab (Default ribbon)
    • Setup â†’ Stock information - (Tax classes) tab (Setup ribbon)

    These are:

    1. First record - Go to the first record.
    2. Prior record - Go to the previous record.
    3. Next record - Go to the next record.
    4. Last record - Go to the last record.
    5. Insert record (add) a new record.
    6. Delete record - (permanently remove) a record. Once you click on this button, a confirmation message  will be displayed:

    "Delete record?"

    Make absolutely sure that the correct record before you click on the OK button. If it is not the correct record, click on the Cancel button and select the correct record.
    1. Edit record - Edit a selected record. This will activate the Post edit or Cancel edit buttons.
    2. Post edit - Saves (accept) the changes to a record. 
    3. Cancel edit - Cancels the changes to a record.
    4. Refresh data - This will refresh the latest changes.
