    FIXED - Documents - Debtor / Creditor Lookup

    Create new Document - Select Debtor / Creditor 

    When creating a New sales document (e.g. Invoice, Credit note or Quote), the Debtor accounts screen list the last line. You need to use the scroll bar to view the search filter options to find an account.

    Similarly when created a New purchase document (i.e. Purchase, Supplier return or Order), the Creditor accounts screen list the last line. You need to use the scroll bar to view the search filter options to find an account.

    To use the filter options you need to scroll up

    Deleting the HKEY_CURRENT_USER\SOFTWARE\osfinancials

    To Delete the HKEY_CURRENT_USER\SOFTWARE\osfinancials in Windows Registry:

    1. Close osFinancials.
    2. Right-click on the Start icon of your Windows operating system and select "Run".

    1. Click OK to open the "Registry editor". Your system may ask for permissions and click Yes.

    1. Navigate to the "HKEY_CURRENT_USER\SOFTWARE\osfinancials" key.
    2. On "osfinancials" key right-click and select Delete on the context menu.
    3. A "Confirm Key Delete" message will be displayed:

    Are you sure you want to permanently delete this key and all of its subkeys?

    1. Make absolutely sure that the correct entry is selected and click Yes.
    2. Close the "Registry Editor" and launch osFinancials. It will create a new key and subkeys. The Debtor accounts / Creditor accounts lookup will display correctly when when creating new documents.
