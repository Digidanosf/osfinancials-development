    Spreadsheet Export / Import

    Spreadsheet Export / Import - 

    See video on Improved Export/Import - 

    The Spreadsheet Export / Import spreadsheet feature allows to easily export data into spreadsheets, edit the data and import it. 

    Excel Export / Import is available in the following options:

    Setup ribbon: 

    • Setup → Accounts 
    • Setup → Groups 
    • Import Accounts, Import Debtors/Creditors, Import Stock 

    Default ribbon 

    • Batch entry (all batches) 
    • Edit → Repeating transactions
    • Stock (Import / Export tab)
    • Documents - Context menu - Documents → Spreadsheet import

    Export - Reports ribbon

    Ledger analyser - Financials

    There are six (6) report types, which may be selected (i.e. Trial balance, Income statement, Balance sheet, Standard column balances, Budget vs actual and This year vs last year) which may be printed and exported to spreadsheets.

    Column balances

    In addition to this, there are 6 report types which may be exported to spreadsheets (i.e. Column balances, Column balances Debit/Credit, Column balances debtors, Column balances creditors, Column balances debtors Debit/Credit, and Column balances creditors Debit/Credit.)

    Budgets

    Budget figures as entered or processed in the Budgets option on the Reports ribbon; or in Setup → Accounts (Setup ribbon). may be printed and exported.  This option, if selected, will list only the budget figures (Budget listing) You may select the "Budget" option on the Trial balance, Income statement, Balance sheet and Standard column balances report types. These budget figures may be printed or exported. 

    The Budget vs actual is the budget performance report. It will list and compare the budget figures versus the actual posted (updated) transactions. The Budget vs actual report may also be printed or exported. 

    Transactions

    T-Account viewer - You may select a specific account, group, etc. to view the transactions in the T-Account viewer. 

    On the report types in the Ledger analyser, you may right-click on a selected account, and select the “Show details” option on the context menu. You may print these transactions and exported these transactions to spreadsheets.

    Open item link

    The lists on the Open item link - Totals tab, Transactions tab and selected accounts may be exported and opened in spreadsheets.
