    Backup / Restore Sets of Books

    What is a Backup file in osFinancials?

    A backup file is a copy of data in a Set of Books in osFinancials that is stored separately in case the original data is lost or becomes inaccessible. The backup file is essentially a snapshot of the original data at a particular point in time, and it can be used to restore the data in a Set of Books in case of data loss, corruption, or system failure.

    They are typically stored on an external device or in a remote location to protect against disasters like fire, theft, or hardware failure that could damage or destroy the original data.

    Backup files can be created manually or automatically on a scheduled basis. Manual backups require the user to initiate the backup process, while automatic backups are configured to run automatically at regular intervals.

    In summary, backup files are essential for ensuring data integrity and availability in case of data loss or system failure. They provide a way to recover data and restore system functionality quickly and efficiently.

    Options to store backup files

    Backup files can be stored in a variety of locations, depending on your needs and preferences. Here are some common options:

    1. External hard drive: You can store backup files on an external hard drive that is connected to your computer or server. This provides a physical backup that is easily accessible and can be quickly restored if needed.
    2. Cloud storage: You can store backup files in cloud storage services such as Google Drive, Dropbox, or OneDrive. This provides an off-site backup that is accessible from anywhere with an internet connection and can protect against physical disasters like fire or theft.
    3. Network-attached storage (NAS): You can store backup files on a NAS device that is connected to your network. This provides a centralized backup solution that can be easily accessed by multiple devices.
    4. Tape backup: You can store backup files on magnetic tapes that are stored in a secure location. This is a traditional backup method that is still used in some organizations because of its durability and reliability.
    5. USB drive: You can store backup files on a USB drive that is stored in a secure location. This is a portable backup solution that can be easily transported and used to restore data on different devices.
    6. Optical media: Optical media such as CDs, DVDs, and Blu-ray discs can be used to store backup files.

    Overall, the choice of storage location for backup files depends on your needs for accessibility, security, and reliability. It is recommended to store backup files in at least two separate locations to ensure redundancy and protect against data loss.

    Options to Backup or Copy a Set of Books in osFinancials

    In addition to the Backup/Restore feature, you can create exact copies of existing Sets of Books or templates that are installed in your osFinancials5/books directory.

    The following options are available in osFinancials for backing up or copying a Set of Books:

    1. Backup / Restore: This option is available on the Start ribbon. It allows you to create backups of your Set of Books and restore them when needed. You can select the backup option to create a backup file and choose the restore option to restore a previously created backup.
    2. Save as: This option is also available on the Start ribbon. It enables you to create a copy of an existing Set of Books with a new name and/or a new location. This is useful when you want to create a duplicate or backup of a Set of Books.
    3. Create Set of Books: This option is found in the "Copy an existing Set of Books?" section of the "Creation wizard" on the Start ribbon. It allows you to create a new Set of Books based on an existing one. You can select an existing Set of Books and provide a new name and/or location for the copied Set of Books.
    4. Tools → Global processes → Copy from a Set of Books: This menu-option is available on the Setup ribbon. It provides a feature for copying a Set of Books from an existing one. You can select the source Set of Books and specify the name and/or location for the copied Set of Books.

    These options allow you to create backups or copies of Sets of Books in osFinancials, either by directly saving them as new files or by using existing Sets of Books as templates. Each option provides flexibility in terms of naming, location, and source for the copied Set of Books.
