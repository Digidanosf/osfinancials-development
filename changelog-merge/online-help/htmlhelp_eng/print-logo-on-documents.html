    Company Logo in Layout Files

    You can easily include your company logo on various documents in osFinancials, enhancing your branding and ensuring a professional look for your financial documentation.

    If any of the existing layout files that come with osFinancials installations or updates are customized, there's a risk of losing your changes. When working with layout files other than your own, it's crucial to create a copy of the layout file or report and rename it before making any modifications.

    osFinancials offers two options to add logos to document layout files:

    1. Load Company logo:  Logo Expressions included in the document layout files installed with osFinancials i.e. Layout file, Document layout, Document layout (15 code), and their NO-TAX equivalents  (NO-TAX Layout file, NO-TAX Document layout, NO-TAX Document layout (15 code)layout files, Delivery note layout files as well as the Tutorial-Invoice and NO-TAX Invoice-English layout files and on Debtor statements as well as on Creditor remittance advises . When you load the Company logo the logo will by default be displayed and printed in the place of the Company name, Address, Telephone, Fax, E-mail address and Website. 
    2. Insert a logo image in to your custom layout files: When you design or customise your own layout file, you can insert or add the Image object into the place of your layout file.

    Options for Including Company Logo in Layout Files

    Options for Including Company Logo in Layout Files:

    1. Load Company Logo Option:
      1. Description: osFinancials provides a straightforward way to include logos in document layout files.
      2. Usage: This option applies to document layout files that come pre-installed with osFinancials, including standard layouts and their NO-TAX equivalents, as well as delivery note layout files, Tutorial-Invoice, and NO-TAX Invoice-English layouts. It also extends to Debtor statements and Creditor remittance advises.
      3. Functionality: By loading the Company logo, it will automatically replace designated elements like Company name, Address, Telephone, Fax, E-mail address, and Website. The logo is displayed and printed in these locations.
      4. Implementation: Access the "Company info" options in the Setup menu to load the logo. This logo expression is integrated within the layout files.
    2. Insert Logo Image in Custom Layout Files:
      1. Description: For enhanced customization, osFinancials offers the flexibility of inserting a logo image directly into custom-designed or modified layout files.
      2. Usage: This option is ideal when you're creating your own layout or making changes to existing ones.
      3. Functionality: Design your layout to include an Image object in the desired location. This object will hold the logo image.
      4. Implementation: Access the Report designer (Reportman) tool within the Setup menu. Drag and drop the Image object into your layout and configure its settings as needed.

    Example : Company logo as Image 

    In this example a company logo may also be added as an image in the layout file. In this example, you do not need to load the Company logo in  Setup → Company info (Setup ribbon).

    By understanding and utilizing these two options, you can seamlessly incorporate your company logo into various layout files within osFinancials. Whether you're using pre-installed layouts or creating your own, these choices ensure that your branding is effectively represented on your documents.

    Load Company Logo

    To display and print the logo, you need to load the logo in the “Company info - Options” screen. 

    To load the Company logo:

    1. On the Setup ribbon, select Company info.
    2. Click on the Options tab.

    1. Click on the Load from file button (or right-click in the Logo area and select the Load from file option). 
    2. On the "osFinancials - Load Picture" screen, select a valid file format (i.e. jpg, png, gif, bmp).
    3. Click OK to save.
    4. The selected Company logo will print on Document layout files and Debtor statements as well as on Creditor remittance advises.

    If you do not wish to print the logo, repeat the process but select the Clear option.

    See Reportman - Company logo. You add your own document layout files and add resize and move the image on your layout file and change the settings (i.e. crop, stretch, display full size, tile or proportional tile) on the Image tab - Draw Style field to display the image or logo (in the Report designer (accessed from the Setup ribbon).
