    Print single item - Yagoda barcode

    Yagoda barcode - Stock grid reports

    The "Yagoda barcode" report prints Yagoda barcodes for all stock items or a selected stock item.

    You may select the Yagoda barcode report on the Reports → Stock → Print labels menu (Reports ribbon) to print Yagoda barcodes.  You may select a specific stock item or a range of stock items; or all stock items linked to a specific Stock reporting group 1.   

    Yagoda barcode options

    Print single item (context menu) 

    Prints directly to default printer and not to the Reportman preview screen. 

    In this example, the system's default printer is set to "Microsoft print to PDF"  where you need to enter a file name to save the file print as "PDF Document (*pdf)" file type on the " Save Print Output As" screen.
