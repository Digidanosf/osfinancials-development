    Financial tools 

    Activa Plugin 

    Activa Plugin - Manual - Shop - Licence : Once-off -

    Documentation : Help documentation 

    Maximize your accounting efficiency with our Activa Plugin. Designed for managing fixed assets, this plugin automates the depreciation process in your accounts, ensuring accurate and timely calculations.

    Key Features:

      • Streamlined depreciation management for fixed assets
      • Periodic notifications for new asset purchases and depreciation processing
      • Asset registration for automated depreciation allowances
      • Simplified management of fixed asset depreciation within your Set of Books

    Optimize your accounting practices and ensure compliance with depreciation regulations with the Activa Plugin.This plugin will help you with the depreciation of fixed assets in your accounts. Periodically you will receive a notification screen, to list the new purchases of assets as well as those assets, you need to calculate and process depreciation. 

    Charter

    Charter Plugin - Manual - Shop - Licence : Once-off 

    Documentation : Help documentation  

    Translations : Outstanding 

     Unlock powerful visualization capabilities with our Charter plugin. Seamlessly generate insightful charts and graphs directly from your Set of Books data, empowering you to make informed decisions and communicate key insights effectively. Save charts as image files for integration into documents, presentations, spreadsheets, and more.

    The Charter Plugin enhances your experience by adding a Charter tab on the Debtor accounts, Creditor accounts, and Stock items. This allows you to build and customise charts effortlessly. Within the Debtors grid, Creditors grid, and Stock item grid, simply right-click to launch the "Charter" screen from the Plugin action → Charter context menu.

    In the Documents grid, you can also access the "Charter" screen by right-clicking and selecting the Plugin action → Charter context menu. This feature enables you to launch the "Doc value" or "Debtors" charts. Additionally, in document entry screens for both posted and unposted sales documents such as invoices, credit notes, quotes, purchases, supplier returns, and orders, an icon will be available to launch the "Charter" screen.

    Furthermore, the "Charter" plugin seamlessly integrates with other plugins supporting the "Charter" functionality.

    DrillDown 

    DrillDown plugin - Manual - Shop - Licence : Once-off -

    Documentation : Help documentation 

    Translations : Outstanding 

    The "TC DrillDown" plugin for osFinancials streamlines data analysis and reporting. With robust features, it offers quick access to transaction details, customer invoices, and sales data through customizable views. Its intuitive interface includes powerful grid search and filter options, along with visual charts for sales analysis. Additionally, users can export data to XML, HTML, Excel, or Text files, facilitating further analysis or integration with other tools. This makes it a valuable tool for efficient financial management and decision-making.

    Remittance

    Remittance - Shop - Licence : Once-off - 

    Documentation : osFinancials website - Manual 

    Documentation : Help documentation 

    Translations : Outstanding 

    The Remittance Plugin in osFinancials facilitates the streamlined management of outstanding items. With this plugin, you can efficiently send reminder emails to all debtors regarding their outstanding amounts and totals. Additionally, the plugin enables you to review unpaid invoices of your customers and send them reminders with detailed open amounts. 

    You have the flexibility to customize the layout of these reminders according to your preferences or specific requirements, and the option to utilize multiple layouts for different purposes. Simplify your accounts receivable process and enhance communication with debtors using the Remittance plugin.

    XML Audit file Plugin

    XML Audit file Plugin - Shop - Licence : Supported language: Netherlands

    The XML Auditfile Plugin, tailored for the Dutch market, facilitates the seamless export and import of ledger, debtor, creditor, and transaction data in XML format. This standardized format ensures that transaction data remains accessible by any system capable of reading XML, promoting interoperability across various platforms and applications.

    Recognized by the Dutch government, the XML Auditfile serves as a valid backup for electronic transaction data, meeting regulatory requirements and enhancing data security practices. 

    Furthermore, the widespread adoption of this format has led to the emergence of complementary products in the Dutch market. Software solutions designed to generate comprehensive yearly overviews can now import data seamlessly, while other financial software offerings provide export and import capabilities, allowing users the flexibility to access and analyse data in their preferred programs. Experience enhanced data management and interoperability with the XML Auditfile Plugin.
