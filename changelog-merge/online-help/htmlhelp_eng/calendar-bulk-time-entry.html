    Bulk time entry

    The bulk time entry feature in the Calendar of osFinancials allows you to efficiently add multiple time entries for a salespersons and a specific debtor (customer/client) account. This feature is particularly useful when scheduling recurring appointments or events.

    To use bulk time entry, you can select the desired dates and set the time for a salesperson's hours event on the Calendar. For example, if a debtor (customer/client) has booked a training course or workshop for a week, you can easily schedule appointments for each day of that week. These bulk time entries will create multiple appointments on the calendar as hours-type events for the selected dates and time slots.

    Once the bulk time entries are created, you have the option to generate quotes for these events. Quotes can be customised and edited as needed before sending them to the debtor (customer/client). This allows for clear communication and documentation of the scheduled appointments or services.

    After the appointments are completed, you have the flexibility to convert the quotes to invoices. This streamlined process eliminates the need for manual data entry and ensures accuracy in billing and invoicing. By converting the quotes to invoices, you can efficiently generate invoices based on the scheduled appointments and bill the debtor (customer/client) for the services provided.

    Overall, the bulk time entry feature in osFinancials' Calendar provides a convenient way to schedule and manage multiple appointments for salespersons and debtors (customers/clients). It enables the creation of quotes, smooth transition to invoices, and simplifies the workflow for tracking and billing services.

    See video on Bulk time entries 

    To process bulk entries: 

    1. On the Default ribbon, select Calendar. 
    2. Click on the Bulk time entry button.
    3. Select the dates / days and a time slot to schedule the hours.
    4. "Week" is the default option - it displays Monday to Sunday. If the "Week" option is not selected it will list only the selected date.
    5. Select the Salesperson and enter the start dates and number of hours. 
    6. Select a stock item (usually sales no stock type)
    7. Select the debtor (customer / client) account. 

    1. Click on the Create entries button. A confirmation message will be displayed.

    "Entries created!" 

     

    1. Once the entries are created, these entries will be listed in the calendar.  

    1. You may edit these events, set reminders and add notes. 
    2. On the top of the screen, click on the Create invoices for hours button. 

    The "Hours between date" options screen is displayed: 

    1. Select the dates (from date and to date) and a Stock item. 
    2. Click OK. An information message will be displayed. 

    "Quote(s) generated!" 

    A quote is generated for the entries on the Bulk time entries for the selected dates and number of hours. 

    1. You may edit the quote 

    1. Check and edit the quote, if necessary. You may wish to:
      1. Enter a reference.
      2. Enter a message.
      3. Add comments and remarks. 
      4. Change the number of hours and selling prices.
      5. Add discount, etc.
    1. Click OK to save the quote. 
    2. Once these appointments bookings are done, you may print the quote and forward it to the debtor (customer / client). The quote needs to be converted to an invoice.  

    The bulk time entries will be printed in the "Calendar - Hours" report (Reports → Calendar (Reports ribbon).

    These bulk time entries will be listed in the Appointment tab of the selected debtor (customer / client) account.

    If you double-click on a selected appointment, the Bulk time entry screen will be launched. You may select the dates edit reschedule, add notes, etc. and set reminders, etc.
