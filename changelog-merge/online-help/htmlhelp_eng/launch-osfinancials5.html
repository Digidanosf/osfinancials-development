  1. Getting started

    Launch osFinancials5 for the first time

    In osFinancials5, the interface language will be displayed based on the language associated with the Set of Books for the country. Each Set of Books can have a specific language linked to it, and when you open that Set of Books in osFinancials5, the interface language will be displayed in the language associated with that particular Set of Books by default.

     Default languages have already been pre-set for each Set of Books. (Setup â†’ Access control on the Setup ribbon). 

    When opening a Set of Books, it will automatically load the language file. All screens, reports and layout files are translatable via the language files.

    Some languages may not display correctly if your system locale in the Regional and Language Settings option in the Control Panel of your system may not support the language and system locale (unicode).

    English is the default (driver language file â€“ used as the basis for translation, and to Customise language files). For example, the American and the English files contain specific label, text, currency and tax terminology for each country.

    Should you select the incorrect language at this stage, you may change the language in Switch language on the Start ribbon).

    Once osFinancials5 is launched, the "Set of Books" screen is displayed. To display the interface of osFinancials5 in the default "English" language, select the "HANDYMAN" Set of Books and click on the Open button.

    Add Sets of Books

    By default, osFinancials5 installs with the "books" directory, which includes the "HANDYMAN" Set of Books that opens in English. 

    The Books are by default installed in the following directories:

    • books - The Sets of Books in this directory, is installed when installing osFinancials5, is HANDYMAN which will open in the English language. 

    If your language or country's Set of Books is not available in the "books" directory, you can download and customize any of the available Firebird Sets of Books and linked languages for free. 
    osFinancials5 supports 79 different languages and countries/tax regimes languages linked to 87 Sets of Books (databases), which you can download and customize to meet your requirements.
