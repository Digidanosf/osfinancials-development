# Manage Sets of Books

***

# Options to manage Sets of Books in osFinancials5/TurboCASH5&nbsp;

There are several options on the **Start** ribbon to manage Sets of Books (databases) in osFinancials5 / TurboCASH5, including:

1. [**Open Set of Books**](<open-set-of-books.md>): Clicking on the **Open** icon on the **Start** ribbon allows you to select and access a specific Set of Books from the list of valid Sets of Books. The list includes Sets of Books located in the "books" folder of the osFinancials5/TurboCASH5 installation directory and any Sets of Books stored in other locations that have been added to the list.
1. [**Close Set of Books**](<close-set-of-books.md>): The Close Set of Books option on the **Start** ribbon allows you to close the currently active/opened Set of Books.
1. [**Create Set of Books**](<create-set-of-books.md>): The "*Creation wizard*" provides three options to copy and create a Set of Books:
- &nbsp;

  - [**Copy an existing Set of Books?**](<copy-set-of-books.md>): This option enables you to create a new Set of Books by making an exact copy of an existing one, including all the data it contains. You can select the Set of Books you want to copy and specify the directory for the new Set of Books.
  - [**Let osFinancials help you to create a Set of Books?**](<create-books-default-download.md>): / [**Let TurboCASH help you to create a Set of Books?**](<create-books-default-download.md>):&nbsp; This option guides you through the process of setting up a new Set of Books using an existing Set of Books as a template. You can also download pre-existing Sets of Books as templates for your country or industry.

| ![Image](<lib/admon-note.png>) | *After downloading a Set of Books (template) using the "[Let osFinancials help you to create a Set of Books?*](<create-books-default-download.md>)*" option, the "Save as" screen will be launched, allowing you to save the template in a directory of your choosing.*&nbsp; *You can also view a list of available [Sets of Books available for a download*](<sets-of-books-download-templates.md>) *to use as templates for your new Set of Books.* *After downloading a Set of Books, the Set of Books (template) will automatically open in the language associated with the country and language of the Set of Books.*&nbsp; |
| --- | --- |


* &nbsp;
  * [**Create a Set of Books (Advanced)?**](<create-books-advanced.md>): This option is for advanced users who want to create a new Set of Books from scratch. It allows you to choose the database type (Firebird or MSSQL) and set a maximum of 8-digit account codes, providing more flexibility in creating the Set of Books structure.

| ![Image](<lib/admon-note.png>) | *When creating, or copying a Set of Books, the default number of characters for the account codes is pre-determined by the selected Set of Books. Most Sets of Books currently use 3-digit account codes as their default setting.*&nbsp; *However, if you choose the "[**Create a Set of Books (Advanced)?***](<create-books-advanced.md>)*" option, you can select a maximum of 8-digit account codes, providing more flexibility in creating a Set of Books that suits your needs.* |
| --- | --- |


4. [**Save as**](<books-save-as.md>): This option allows you to save a Set of Books to a different directory or folder on your system. It creates a copy of the Set of Books with a new name and location, without affecting the original data.
4. [**Delete Set of Books**](<delete-set-of-books.md>): Deleting a Set of Books permanently removes all data within it. It is important to have backups, print necessary reports, and retain supporting documents before deleting a Set of Books, as the information may be required for tax returns or other legal purposes.

These options provide flexibility in managing Sets of Books in osFinancials5/TurboCASH5, including creating new ones, copying existing ones, saving copies, and deleting Sets of Books.

# Options to Backup or Copy a Set of Books in osFinancials5/TurboCASH5

In addition to the [**Backup/Restore**](<backup-restore-set-of-books-opti.md>) feature, you can create exact copies of existing Sets of Books or templates that are installed in your osFinancials5/TurboCASH5 .../books directory.

The following options are available in osFinancials5/TurboCASH5 for backing up or copying a Set of Books:

1. [**Backup / Restore**](<backup-restore-set-of-books-opti.md>): This option is available on the **Start** ribbon. It allows you to create backups of your Set of Books and restore them when needed. You can select the backup option to create a backup file and choose the restore option to restore a previously created backup.
1. [**Save as**](<books-save-as.md>): This option is also available on the **Start** ribbon. It enables you to create a copy of an existing Set of Books with a new name and/or a new location. This is useful when you want to create a duplicate or backup of a Set of Books.
1. [**Create Set of Books**](<create-set-of-books.md>): This option is found in the "[*Copy an existing Set of Books?*](<copy-set-of-books.md>)" option of the "*Creation wizard*" on the **Start** ribbon. It allows you to create a new Set of Books based on an existing one. You can select an existing Set of Books and provide a new name and/or location for the copied Set of Books.
1. **Tools → Global processes** → [**Copy from a Set of Books**](<copy-from-set-of-books.md>): This menu-option is available on the **Setup** ribbon. It provides a feature for copying a Set of Books from an existing one. You can select the source Set of Books and specify the name and/or location for the copied Set of Books.

These options allow you to create backups or copies of Sets of Books in osFinancials5/TurboCASH5, either by directly saving them as new files or by using existing Sets of Books as templates. Each option provides flexibility in terms of naming, location, and source for the copied Set of Books.
