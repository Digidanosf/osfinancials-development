# Context menu - Confirm quotes and orders - Quotes

***

Once the debtor (customer / client) accepts the Quote, it can then be confirmed and converted to an Invoice. Once a Quote is confirmed and converted to an Invoice, you may need to edit the Invoice, before you update (post) the Invoice to the ledger.

| ![Image](<lib/admon-tip.png>) | *Quotes may also be confirmed in&nbsp; **Input → [Confirm***](<confirm-quotes.md>) *menu on the **Default** ribbon. (**Shift+F3**).*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *If the [**Document posting***](<doc-setting-post-delete-buttons.md>) *option is selected (ticked) on the **Documents** tab of **Access control** (**Setup** ribbon), the caption of the **Posting** button, will change to **Confirmation** button if Quotes and Orders are selected as the document type.* |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *The **Documents →** **Confirm quotes and orders** option on the context menu, is only available, if the Quotes or Orders document type is selected.* |
| --- | --- |


**To confirm Quotes(s):**

1. Select the "*Quote*" document type.
1. Select the Quote(s) to confirm.

| ![Image](<lib/admon-tip.png>) | *If you need to select multiple documents, press the **Ctrl** or **Shift** key and click on each Quote, you need to confirm.* |
| --- | --- |


![Image](<lib/documents-context-menu-confirm-quotes.png>)

3. Right-click and select **Documents →** **Confirm quotes and orders** option on the context menu. A confirmation message is displayed:

*&nbsp;"Do you wish to: (2) Quotes Confirmation" (where (2) is the number of documents selected).*

4. Click on the **Yes** button to continue. A confirmation message will be displayed.&nbsp;

*&nbsp;"Do you wish to delete this Quotes?"*&nbsp;

| ![Image](<lib/admon-important.png>) | *If you click on the **No** button, the selected Quotes will still be confirmed and converted to Invoices. These Quotes will still be available to edit.*&nbsp; *You may delete the Quotes manually in **Edit → [Documents***](<delete-documents.md>) *menu on the **Default** ribbon.* |
| --- | --- |


5. Click on the **Yes** button to continue. Once the process is finished, an information message is displayed:

*&nbsp;"Document(s) created." (This message will list the new Invoice number, reference, account code and account description of the debtor (customer / client) account)).*

6. Click on the **OK** button. Note - If the Orders document type was selected, this confirmation message will not be displayed.&nbsp;
6. The converted Quotes will be available will be available as unposted Invoices.&nbsp;

| ![Image](<lib/admon-note.png>) | *You need to check and edit the Messages in the Document header, if necessary.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *The source document, Quote number (e.g. "Ref:QU000004") will be added to the third Message line in the Document header&nbsp; of the Invoice.* |
| --- | --- |
