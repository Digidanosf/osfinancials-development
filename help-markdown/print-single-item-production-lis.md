# Production list - Stock grid reports&nbsp;

Production list report lists the transactions of all items (other stock item types) linked to the [BOM](<stock-item-new-bom.md>) and [BOM (Production)](<stock-item-new-bom-production.md>) stock item type on the BOM tab.&nbsp;

This report will only list the active stock items. Disabled (inactive) stock items will be excluded from this report.&nbsp;

# Print singe item (context menu)&nbsp;

The **Print singe item** (context menu option) will only print the selected Stock item, if the Stock item type is [BOM](<stock-item-new-bom.md>) and [BOM (Production)](<stock-item-new-bom-production.md>).

![Image](<lib/sidebar-stock-production-list-single-item.png>)

&nbsp;

# Report button&nbsp;

The report will print all active Stock items for the [BOM](<stock-item-new-bom.md>) and [BOM (Production)](<stock-item-new-bom-production.md>) stock item types.

![Image](<lib/sidebar-stock-production-list-print.png>)
