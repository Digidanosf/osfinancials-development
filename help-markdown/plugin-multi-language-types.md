**To add translations to types:**

1. On the **Setup** ribbon, select **Tools → Customise language**.
1. Click on the **Types** tab.    

![Image](<lib/multi-language-customize-language-types.png>)

3. Select the Language and enter the name and description for each language.
3. Repeat the process for all Types.
3. Click on the **Rest** tab.&nbsp;

![Image](<lib/multi-language-customize-language-rest.png>)

6. For the "*Description of Zero account*", select the Language and enter the name (description) for each language.
6. For the "*Groups empty account*", select the Language and enter the name (description) for each language.
6. Repeat the process for all Types.
6. Click on the **Close** button.&nbsp;

**After translation**&nbsp;

After translations added, the document types will be displayed in the Language set for the logged in user.
