# Debtor Context menu - Set Debtor group 1/2

***

You may change the Debtor group 1 and/or Debtor group 2&nbsp; for selected debtor (customer / client) account(s).&nbsp;

**To change the Debtor group 1 or 2:**&nbsp;

1. On the **Debtors** (Debtors list screen) (**Default** ribbon), select the debtor (customer / client) account(s).

| ![Image](<lib/admon-tip.png>) | *If you need to select multiple debtor (customer / client) account(s), press the **Ctrl** or **Shift** key and click on each debtor (customer / client) account, for which you need to change the debtor reporting group.* |
| --- | --- |


2. Right-click and select "*Debtor group 1*" or "*Debtor group 2*" on the context menu.&nbsp;
2. Select "*Empty*" or the available debtor group from the sub-menu. The "*Debtor group 1*" or "*Debtor group 2*" will change on reports.&nbsp;
