# Sets of Books - Installed in osFinancials5/TurboCASH5

The sets of books in the "*books*" directory are installed during the osFinancials5/TurboCASH5 installation. The specific sets of books depend on the country and language you have chosen during the installation process.

**Sets of Books (Templates) available for download**&nbsp;

osFinancials and TurboCASH users may download these Sets of Books (templates) free in a matter of a few seconds (using the option, [Let osFinancials / TurboCASH help you to create a Set of Books](<create-books-default-download.md>)).&nbsp; It takes only 3 clicks to download a Set of Books.&nbsp;

After downloading a Set of Books, the Set of Books (template) will automatically open in the language associated with the country and language.&nbsp;
