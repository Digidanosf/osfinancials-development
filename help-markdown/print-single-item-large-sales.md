# Price list - Large retail - Sales

The "*Price list - Large retail - Sales*" report will list quantity discount up to five (5) price levels of a selected stock item for a selected debtor (customer / client) account.

## Price list - Large retail - Sales options

![Image](<lib/stock-sidebar-price-list-options.png>)

## Report button on Sidebar&nbsp;

![Image](<lib/price-list-large-retail-sales-parm.png>)

Select the debtor (customer / client) account.

Enter the quantities (up to 5 levels).

![Image](<lib/price-list-large-retail-sales.png>)

&nbsp;

## Print single item on context menu&nbsp;

Right-click on a selected stock item (product).

Select the debtor (customer / client) account.

Enter the quantities (up to 5 levels).

![Image](<lib/price-list-large-retail-sales-parm.png>)

Prints directly to the Reportman preview screen.&nbsp;

![Image](<lib/price-list-large-retail-sales-single-item.png>)

&nbsp;

# Price agreement reports (Reports menu)

You may print lists of the price agreements.

## Print Price agreements for Debtors (Customers / Clients)

**To print Price agreement reports for debtors (customers / clients):**

1. On the **Reports** ribbon, select **Reports → Plugins → Price agreement**.                         &nbsp;
1. Select the "*Debtor - Price agreement*" report (default option).

![Image](<lib/price-agreement-report-debtors-options.png>)

3. Select the debtor (customer / client) account or creditor (supplier / vendor) account.
3. Enter up to 5 quantities.&nbsp;
3. Click on the **OK** button.&nbsp;

An example of the "*Debtor Price agreement*" report, is as follows:

![Image](<lib/price-agreement-report-debtors.png>)
