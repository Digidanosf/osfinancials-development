# Report designer menu structure

***

The "*Report manager designer*" is launched from the "*Report man*" screen.

The "*Report manager designer*"&nbsp; interface an menu structure provides all the options and tools to create and edit reports document layout files as well as reports and templates used for Plugins. &nbsp;

# Report menu&nbsp;

![Image](<lib/report-desingner-menu-report.png>)

&nbsp;

## Report → Add

![Image](<lib/report-desingner-menu-report-add.png>)

**Add** - You may select to add a Page header, Page footer, Group header and footer, Subreport or Detail to the structure of the report.&nbsp;

You may also click on the ![Image](<lib/reportman-sturcture-icon.png>) icon on the **Structure** tab.

![Image](<lib/report-designer-rstructure-tab-add-section.png>)

## Edit menu&nbsp;

![Image](<lib/report-desingner-menu-edit.png>)

To Delete or Cut more than one object, select more than one (1) object (using the **Shift** + mouse click).&nbsp;

## Edit → Select menu&nbsp;

![Image](<lib/report-desingner-menu-edit-select.png>)

**All** - This will select all objects on the report.

**All text** - This will select only the text objects and expressions, excluding simple drawings, shapes and images.

&nbsp;

## Edit → Move menu&nbsp;

![Image](<lib/report-desingner-menu-edit-move.png>)

This will move a selected object one space to the left, right, up or down. &nbsp;

To move more than one object, select more than one (1) object (using the **Shift** + mouse click).&nbsp;

To move all, first select **Edit → Select → All** option.

To move all the text objects and expressions, excluding simple drawings, shapes and images., first select **Edit → Select → All text** option.

## Edit → Align menu&nbsp;

![Image](<lib/report-desingner-menu-edit-align.png>)

To activate the align objects, you must select more than one (1) object (using the **Shift** + mouse click) or **Edit → Select → All / All Text** menu options.
