# Reportman - Save reports

***

Whenever you exit or close a report or a document layout file in the "*Report manager designer*" screen, a confirmation message will be displayed:

*"Save report?"*

Click on the **Yes** button.&nbsp;

| ![Image](<lib/admon-warning.png>) | *If you click on **No**, your changes will be discarded (lost).* |
