# Access control - Reports access tab

***

The **Reports access** tab in the **Access control** feature of osFinancials5/TurboCASH5 allows you to control a user's access to printing and viewing specific reports in a Set of Books.

**To set reports access options for an user:**&nbsp;

1. On the **Setup** ribbon, click on the **Setup → Access control** menu. Then, navigate to the **Reports access** tab.

![Image](<lib/access-control-tab-reports-access.png>)

2. **Selecting the User**: Ensure that the correct user is selected for whom you want to restrict or allow access to reports. If needed, click on the **User** icon to select the desired user.&nbsp;
2. **Configuring reports access**: To restrict the selected user from accessing specific reports, remove the tick in the respective field. To allow access to reports, place a tick in the field to allow access to the following reports:&nbsp;
1) &nbsp;

   1) **Creditor reports** - This option is not implemented, so it does not affect the user's access to creditor reports.
   1) **Debtor reports** - This option is not implemented, so it does not affect the user's access to debtor reports.&nbsp;
   1) **General ledger reports** - Allow access to print, view and export reports from the [Ledger analyser 1](<ledger-analyser.md>), [Ledger analyser 2](<ledger-analyser.md>), [T-Account viewer](<t-account-viewer.md>). All Ledger reports (except [Ledger analyser 1](<ledger-analyser.md>)) accessed from the **Reports** menu list on the **Reports** ribbon, are still available to print.&nbsp;
   1) **Stock reports** - This option is not implemented, so it does not affect the user's access to stock reports.
   1) **User reports** -&nbsp; This option is not implemented, so it does not affect the user's access to the User reports menu.

4. **Applying changes**: Click on the **Apply** button to save the changes for the selected user.
4. **Setting access levels for other users**: If necessary, select the next available user and configure their batches access and account restrictions in the same manner.
4. **Finalizing changes**: Once you have finished setting the access levels for all desired users, click on the **Close** button to exit the "*Access control*" screen.

&nbsp;By utilizing the **Reports access** tab in Welcome to osFinancials5 / Welcome to TurboCASH5, you can manage and control a user's access to specific reports, allowing you to tailor their reporting capabilities based on their responsibilities and requirements within the Set of Books.
