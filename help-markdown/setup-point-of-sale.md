# Point-of-Sale

***

Point-of-Sale can be set up only if you are selling stock items. Improved security and access control has been added for users to access or start a Point-of-Sale session or shift.&nbsp;

You may now link the salespersons (cashiers) added in **Setup → Salesperson** to users on the **Others** tab - **Setup → Access control**. This feature will add more security when users access or start a Point-of-Sale invoicing session.
