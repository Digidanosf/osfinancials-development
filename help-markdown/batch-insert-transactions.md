# Insert transactions in a batch

***

Sometimes you may have entered a number of transactions in a batch. You may wish to insert a transaction in a specific place in the batch.&nbsp;

**To insert a transaction in a batch:**&nbsp;

1. Select the line or row in which you wish to insert a blank line or row above the selected transaction.
1. Click on the **\^N:Insert** icon (**Ctrl + N**) / (**Insert**).&nbsp;
1. A new blank row above a selected transaction will be inserted.
1. Enter the transaction in the inserted row or line.
