# Import Accounts - Spreadsheet Export / Import

***

osFinancials5/TurboCASH5 supports the "*Excel Workbook (\*.xlsx)* file type Spreadsheet import and export feature#8202;*.* &nbsp;

* [Spreadsheet export](<accounts-export-excel.md>) - You may export ledger accounts and save it as a "*Excel Workbook (\*.xlsx) file type.*
* [Spreadsheet import](<accounts-import-excel.md>) - You may import ledger accounts from a saved "*Excel Workbook (\*.xlsx) file type.*

This feature may be used to import ledger accounts into:

* Into the same Set of Books.
* Into a different Set of Books.
* Into a new Set of Books - Created from scratch - [Create your own Set of Books (Advanced)](<create-books-advanced.md>) option for Firebird database types.
* Into a new Set of Books - Created from scratch - [Create your own Set of Books (Advanced)](<mssql-sql-database-create.md>) option for Microsoft SQL sever (MSSQL) database types.

| ![Image](<lib/admon-note.png>) | *Before you may import ledger accounts in a new Set of Books, you need to do the necessary setup and configuration.*&nbsp; *Once this is done, you may start to import ledger accounts from a saved "Excel Workbook (\*.xlsx) file type.* |
| --- | --- |


You may access the "*Data Import Export*" feature from the following options:&nbsp;

* &nbsp;**Setup → [Accounts**](<accounts-edit.md>) (**Setup** ribbon).

![Image](<lib/spreadsheet-export-accounts-option.png>)

* [**Import**](<stock-import-tab-delimited.md>) option on the **Setup** ribbon.

![Image](<lib/accounts-excel-export-option.png>)

Select **Accounts** as the "*Data type to import*". Click on the **Excel import export** button.&nbsp; The "*Data Import Export*" spreadsheet&nbsp; will displayed column headings for the Accounts (without any data).
