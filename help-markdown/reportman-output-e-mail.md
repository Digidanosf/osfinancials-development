# Reportman output - E-mail

***

This will generate a Portable Document Format (pdf) file. It will also launch a blank E-mail message (using your system's default e-mail program) and insert the report (or document) as an attachment.&nbsp;

| ![Image](<lib/admon-note.png>) | *The name of the report (or document) is automatically generated (for example, REPC526.pdf). It will also automatically be inserted in the "Subject" field and displayed in the title bar of the e-mail message.*&nbsp; *It is recommended that you add your own sensible subject to this in the "Subject" field. You still need to select a recipient (and other recipients, if necessary) and type a message before clicking on the **Send** button.*&nbsp; |
| --- | --- |


To use the E-mail option, you need to have a correctly configured e-mail account on your system. You may also need to configure the following in your Set of Books:

* **Setup → Company info** (**Setup** ribbon) - Add the e-mail address for the Set of Books.&nbsp;
* **Setup → Internet** (**Setup** ribbon) - Configure your Internet, e-mail accounts to send reports, lists and documents via the internet or to configure a network server.
* **Debtors** (**Default** ribbon) - Correct e-mail addresses added for your debtor (customer / client) accounts.
* **Creditors** (**Default** ribbon) - Correct e-mail addresses added for your creditor (supplier / vendor) accounts.

| ![Image](<lib/admon-warning.png>) | *The following message will be displayed if you don't have a default E-mail App installed and associated on your operating system:* &nbsp; *"Email*&nbsp; *There is no email program associated to perform the requested action. Please install an email program or, if one is already installed, create an association in the Default Programs control panel."* |
