# Add Multi-language groups

***

By creating / adding languages, languages may be translated on various screens.&nbsp;

**To add / activate Multi-language feature:**

1. Go to **Setup → Groups**.
1. Click on the **Languages** tab.

![Image](<lib/multi-language-setup-groups-languages.png>)

3. Click on the **REPORTING GROUP ROOT** and click on the **New** button.
3. Enter the Name and Description for the Language (e.g. English, Afrikaans, etc.).
3. Repeat the process to add all required languages.
3. Select the **Language** and right-click and select the Languages on the context menu.

![Image](<lib/multi-language-groups-context-menu-languagea.png>)

7. Enter the Name and Description for each language.

![Image](<lib/multi-language-groups-add-descriptiions.png>)

8. Click on the **OK** button.
8. Repeat the process for all Languages.
8. Click on the **OK** button of the Groups screen.&nbsp;
