# View graphs

You may filter and view the documents in a graph.

To view the documents in graph, you may use the following options for the data to be included in the graph:

* [Document types](<document-types.md>) - Select the document type (i.e. invoices, credit notes, quotes purchases, supplier returns or orders).&nbsp;
* [Document status](<document-status.md>) - Select All, Posted or Unposted and Confirmed (Quotes and Orders).&nbsp;
* [Filter options](<documents-filter-search-options.md>) to select the documents to include or exclude from the graph , etc.&nbsp;

To view the documents in a graph, click on the **Graph** button.

To close the graph, click on the **Graph** button again.

# Customize Chart

## Customize Chart : Types

The default chart diagram is the “*Column diagram*”. You may select the one of the following diagrams to view the graph.&nbsp;

![Image](<lib/docsuments-chart-customize-chart-type.png>)

&nbsp;

## Customize Chart : Series

By default, only the Amount inclusive and Due amount and Tax amount is displayed.&nbsp;

You may remove the tick of the series or add a tick to a series. For example, if you wish to see the amount exclusive, tick that option and if you do not wish to view the tax amount, remove the tick.

![Image](<lib/documents-chart-customize-series.png>)

**Sort by:** select the options in the series to change the sequence of the documents in the graph is displayed.

## Customize Chart : Data Groups&nbsp;

The Data Groups will list all the columns headings (field names) of the Document selection screen.

## Customize Chart : Options&nbsp;

You may change the default values for the “*Legend, Title, Toolbox*” and “*Other (Value Hints)*“ according to your requirements. &nbsp;

&nbsp;![Image](<lib/budget-pivot-ledger-analyser-custom-options.png>)

# Customize Chart : Data groups&nbsp;

The data groups list all the available columns on the Documents grid, which is not already set as "*Data Levels*".&nbsp; You may add or remove any of the available Data Groups on the "*Data Levels*".

**To add a "*Data Group*" to the "*Data Levels*":**

1. Click on the **Customize Chart** button.
1. On the "*Customisation*" screen, click on the **Data Groups** tab.
1. Select the available data group form the list you wish to add to the "*Data Levels*".
1. Drag and drop the data group to the required place on the "*Data Levels*". &nbsp;

![Image](<lib/documents-graph-filter-custom-data.png>)

&nbsp;

**To remove a "*Data Group*" from the "*Data Levels*":**

1. Click on the **Customize Chart** button.
1. On the "*Customisation*" screen, click on the **Data Groups** tab.
1. Select the available data group on the "*Data Levels*". &nbsp;
1. Drag the data group to any place below the "*Data Levels*". When the mouse pointer change to a big **X**, drop it. The data field will be added to its original place on the list in the "*Data Groups*" tab of the "*Customisation*" screen.&nbsp;

![Image](<lib/documents-graph-filter-custom-data-ungroup.png>)
