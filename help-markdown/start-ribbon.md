# Start ribbon

***

The [**Start**](<start-ribbon.md>) ribbon is used to manage your sets of books, backups, registration and language.

![Image](<lib/ribbon-start.png>)

The following nine options are available:

1. [**Open**](<open-set-of-books.md>) - This option allows you to open an existing Set of Books, which is essentially a collection of financial data and records that you have previously created / saved.&nbsp;
1. [**Backup / Restore**](<backup-restore-set-of-books-opti.md>) - With this option, you can create a backup file of your current Set of Books, which is useful for data recovery in case of loss or corruption. You can also restore a Set of Books from a valid backup file.&nbsp;
1. [**Close Set of Books**](<close-set-of-books.md>) - This option allows you to close the currently opened Set of Books. Once the Set of Books is closed, you will not be able to access any of its data or records until you reopen it.
1. [**Create Set of Books**](<create-set-of-books.md>) - You may use the different options available when creating a new Set of Books in osFinancials:
1) &nbsp;

   1) [**Copy an existing Set of Books**](<copy-set-of-books.md>) - With this option, you can create a new Set of Books by making an exact copy of an existing one. This includes all of the data contained within the Set of Books. You simply select the Set of Books you want to copy and the system will create a new one in a directory of your choosing.
   1) [**Let TurboCASH5-4 help you to create a Set of Books**](<create-books-default-download.md>) - This option allows you to create a new Set of Books by selecting a selected Set of Books as a template. The program will then guide you through the process of setting up your new Set of Books. You can also download pre-existing Sets of Books for your specific country or industry as templates to use as a starting point. See a list of the [Sets of Books available for a download](<sets-of-books-download-templates.md>).&nbsp;
   1) [**Create a Set of Books (Advanced)**](<create-books-advanced.md>) - This option is intended for more advanced users who want to create a new Set of Books from scratch. In addition to the Firebird database type, you can also select to create a Set of Books using the MSSQL database type. This option provides more flexibility and control over the structure and content of your new Set of Books.

5. [**Delete Set of Books**](<delete-set-of-books.md>) - If you no longer need a particular Set of Books, you can use this option to permanently delete it.
5. [**Save as**](<books-save-as.md>) - This option is used to save a copy of the currently opened Set of Books with a different name.
5. [**Switch language**](<switch-language.md>) - With this option, you can change the language used in the osFinancials5/TurboCASH5 program.
5. [**Online registration**](<register-osfinancials5.md>) - This option will take you to the "*Register your osFinancials / Register your TurboCASH"* screen where you can register your copy of osFinancials5/TurboCASH5 online.
5. **Help** -&nbsp;
1) &nbsp;

   1) **Help** - This option will launch the context-sensitive Help file and display support information respectively. You can use the Help file to get more information on any particular feature or option in osFinancials5/TurboCASH5. You may press **F1** to launch the help file from most screens and fields to launch the topic relating to that screen or field. This help system is designed to assist users in understanding and navigating the osFinancials accounting software, as well as to provide guidance on accounting principles and practices.&nbsp;

   2. [**Webshop**](<webshop.md>) - This option will take you to the [osFinancials webshop](<https://www.osfinancials.org/en/webshop> "target=\"\_blank\"") where you can purchase additional features or support services.
