# Translate Multi-language chart of accounts

***

The names or descriptions for Groups, as entered / edited in **Setup → Groups** for Account Group 1 / 2 tabs, are by default displayed on screens, lookups, documents, reports, etc.

| ![Image](<lib/admon-note.png>) | *To use multi-language Chart of accounts, Language groups need to be created / added in **Setup → Groups** - **Languages** tab.* *Once these Language groups is created, the Account group 1 (and / or Account group 2) needs to be translated.* |
| --- | --- |


# Translate Chart of accounts

**To translate the Chart of accounts into Multi-languages:**

1. On the **Setup** ribbon, select **Setup → Accounts**.
1. Select an account (e.g. Sales, etc.) on the list and click on the **Translations** tab.&nbsp;

 ![Image](<lib/multi-language-accounts.png>)

3. Enter the description (account name ) of the account in the other language (e.g. Afrikaans).&nbsp;

| ![Image](<lib/admon-important.png>) | *If you select the next account, the changes to translated language(s) will not be saved.*&nbsp; *To activate the **Save** button, click on any field, e.g, "Description" and add a space and remove the space.*&nbsp; |
| --- | --- |


4. Click on the **Save** button.
4. Repeat the process to translate all account descriptions.
4. Click on the **Close** button (on the titlebar).&nbsp;

| ![Image](<lib/admon-note.png>) | *Once the translations of the multi-language Chart of accounts and Account groups is finished you may print the Chart of accounts in the user to which a different language is assigned.*&nbsp; *See sample of the [Chart of accounts printed in English and in Afrikaans*](<plugin-multi-language-listing.md>) *for the same Set of Books.* |
| --- | --- |
