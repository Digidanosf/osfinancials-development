# User reports menu&nbsp;

These [user reports](<user-reports-menu.md>) are stored in the “*...\\plug\_ins\\reports\\userreports*” folder of your installation directory.&nbsp;

When printing reports using the “*Material*” theme, the icons in the “*Material*” theme folder will be displayed in menus and the icons in the “*Classic*” theme folder will be displayed on the report parameters screens and in "*Report man*". For your convenience, a list of the file names and structure of the reports in the **User reports** menu (**Reports** ribbon) is as follows:

![Image](<lib/menu-reports-user.png>)

**To access the [Report designer**](<report-designer.md>)**:**

1. On the **Setup** ribbon, select [**Report designer**](<report-designer.md>).&nbsp;
1. The [**Report designer**](<report-designer.md>) will list all "*userreports*" files and report folders as follows:

![Image](<lib/report-man-userreports.png>)
