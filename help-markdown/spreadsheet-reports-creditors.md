# Spreadsheet - Creditors

***

The Spreadsheet report - Creditors will list all the creditor (supplier / vendor) accounts in a spreadsheet.&nbsp;

The data in this spreadsheet is the same as the Creditor listing - "[*List postal addresses*](<creditors-list-postal-address.md>)" report (**Reports → Creditors →**&nbsp; **Listing** (**Reports** ribbon)).

**To import the creditor (supplier / vendor) accounts:**&nbsp;

On the **Reports** ribbon, select **Spreadsheet reports → Creditors**.
