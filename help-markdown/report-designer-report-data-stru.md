# Report designer - Report Structure and Data

***

The left pane consists of the following two (2) tabs:

1. **Structure**&nbsp; – On this tab you need to select the report section (i.e. subreport, headers, footers, detail, etc.) to which you may add / edit objects (i.e. images, shapes,&nbsp; labels, expressions, barcodes, charts and variables).&nbsp;
1. **Data** – On this tab you need to select the dataset. You may easily add / edit objects (i.e. images, shapes,&nbsp; labels, expressions, barcodes, charts and variables) to include in your report or document layout file.&nbsp;

The right pane consists of the report document in the designer mode.&nbsp;
