# Customizing document type terminology - Order

***

The language associated with each Set of Books determines the specific labels used to cater to different countries or tax regulations. In most languages, the term *'Order'* or its equivalent will be displayed. For instance, en-UK and en-South Africa use terminology related to *'Order'*, while in South Africa, Afrikaans refers to *''Bestelling'* or *'Bestellings'*.&nbsp;

These order-related labels are visible within the osFinancials5/TurboCASH5 interface when you access a Set of Books. They are also present in reports and document layout files generated by the system.&nbsp;

To adapt the order document type terminology to suit your country, jurisdiction, and industry, you can customize and modify the order-related labels via the "*Customize Languages*" screen. This screen is accessible through the **Tools → [Customise Language**](<customise-language.md>) menu on the **Setup** ribbon.&nbsp;

You may also find it helpful to utilize the search options available. This enables you to search for a specific label either in the currently active language file you're editing or in the default "*English*" language file. This assists you in identifying and modifying specific text to better suit your requirements.
