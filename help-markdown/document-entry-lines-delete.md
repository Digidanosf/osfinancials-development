# Delete transactions / comments in documents

***

Sometimes you may have entered a number of transactions and/or comments in a document, and you wish to delete a transaction or a comment in the Document entry screen.

**To delete a transaction or a comment in a document:**

1. Select the row or line for the transaction or comment which you wish to delete.

| ![Image](<lib/admon-warning.png>) | *Once again, make sure absolutely sure that the correct stock item, transaction, or comment is selected.* |
| --- | --- |


2. Click on the **\^D:Delete** icon; or (**Ctrl** + **D**). A confirmation message is displayed:&nbsp;

*"Do you wish to delete this Line?"*

3. Click on the **Yes** button. The selected transaction, or comment, will be deleted or removed.
