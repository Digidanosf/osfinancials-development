## Business Class Subscription Features

A commercial license or active annual subscription upgrades the software from the Evaluation tier and unlocks unrestricted accounting capabilities:

* **Unlimited Processing:** Removes the strict evaluation limit, allowing you to process more than 500 transactions or documents.
* **Integrated Plug-ins:** The standard Business Class annual subscription bundle automatically includes full operational licenses for both the **Bank Import** and **Document E-mail Pro** modules.

## Core System Plug-ins Overview

1. **Bank Import Plug-in**
- &nbsp;

  - This utility eliminates manual bookkeeping entry errors by allowing you to import digital bank statement files directly into your selected Bank Account (Cash Book).
  - **Automated Reconciliation:** The engine reads bank-provided transaction reference strings to automatically match and allocate deposits, receipts, charges, and payment entries directly to their correct ledger accounts.
  - **Resources:** Review the *Help file - Bank import plugin* or the website documentation for *BankImportPlugin* and *Bank Reconciliations Plugin*.

2. **Document E-mail Pro Plug-in**
- &nbsp;

  - Allows you to email transactional layouts (such as Invoices, Orders, or Quotes) directly from the application interface.
  - **Custom Routing Layouts:** Each transaction category can use an independent email body layout (e.g., an invoice can use a completely different email notification style than a prospective quote).
  - **Batch Processing:** You can send multiple documents concurrently to different clients, provided the target document type and the baseline email layout remain uniform across the execution run.
  - **Resources:** Review the *Help file - Document email pro plugin* and *osFinancials website - Document email pro plugin*.

3. **Multimedia Plug-in Pack**
- &nbsp;

  - Extends core database tables by allowing you to attach, catalog, and reference digital files directly within specific master data profiles.
  - **Data Linking:** Attach product imagery directly to **Stock/Inventory items**, or link signed scanned contracts and PDF agreements straight to **Debtor (Customer)** or **Creditor (Supplier)** master cards.
  - **E-mail Archive Tracking:** When paired with Document E-mail Pro, copies of sent client emails are automatically cataloged under the respective customer file.
  - **Outlook Synchronization:** The integrated Outlook Link utility can automatically scan your mail folders for matching client email addresses, copying correspondence histories directly into the master data cards.
  - **Resources:** Review the *Help file - Multimedia plugin* and *osFinancials website - Multimedia plugin*.

4. **Multi-Workstation Plug-in**

Enables concurrent, multi-user database operations over a local network infrastructure.

* &nbsp;
  * **Scalable Client Access:** Supports an unlimited number of simultaneous database users, restricted only by your server hardware capacity.
  * **Infrastructure Requirement:** Requires a configured local area network (LAN) environment with an accessible shared network directory folder.
  * **Resources:** Review the website documentation for *Multi User Installation*, *multi user setup - Windows server*, and *Multi Werkplek (Linux server) Plugin* (Dutch language asset).

## Evaluation Limitations

Every time an unregistered or un-activated version of the application launches, the system displays the following warning notification:

*This software is limited to 500 transactions or 500 documents. Please register and buy a licence for osFinancials5*\
*This copy of osFinancials5 is not registered\! Do you wish to register it now?*

An evaluation installation strictly caps processing thresholds at **500 ledger transactions or 500 total documents**. Once this capacity limit is reached, the database engine locks down, preventing any further transaction posting, batch validation, or document generation. Entering a valid serial unlocking code immediately removes this restriction without data loss.

# Offline Registration (No Internet Option)

If the your system lacks an active internet connection, you can complete the licensing process manually using an authorization file provided by the development team.

## Step-by-Step: Offline File Activation Protocol

1. On the main registration interface, click the **No Internet** option button. This launches a standard Windows **Open** file dialog window, filtering specifically for the unlock.du data file format.
1. Navigate to the directory location where you saved your license asset (typically your local ...\\bin\\Plugins\\ directory within the main osFinancials/TurbpCASH installation root folder). Highlight the **unlock.du** file and click **Open**.
1. The engine will process the file data and instantly launch the first verification input screen: **Serial**. Type or paste your valid baseline master hardware serial number into the text field and click **OK**.
1. The software immediately triggers the second validation screen: **Register**. Inside the *Register to* entry field, input your exact, case-sensitive registered username credentials and click **OK**.
1. The application launches the third and final modal screen: **Enter unlock code**. Input your distinct cryptographic unlocking hash string into the provided field and click **OK**.

## Error Handler: Verification Failure

If any of the alphanumeric keys, username strings, or unlock code sequences contain a typo, or if the individual credentials do not match the encrypted values embedded inside your unlock.du data file matrix, the validation loop will fail:

**System Warning Notification:**

The software will immediately halt the activation routine and display an inline error alert:&nbsp;

*“Sorry wrong code.”*

Click **OK** on this warning prompt to reset the forms. You must re-verify your registration paperwork characters and start the offline input sequence again from Step 3.
