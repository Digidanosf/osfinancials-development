# Spreadsheet reports – Transactions per period&nbsp;

If a Spreadsheet report Sheet tab in the base.xlsx ("Excel 2007 -365") spreadsheet file is named in the lowercase (e.g. trn\_1296) instead of the uppercase (e.g. TRN\_1296) the label will not translate and populate with the data.&nbsp;

An additional sheet with the translated label "*Data*" is added. The osFinancials5 code now supports the uppercase / lowercase.&nbsp;
