# Stock movement - Stock grid reports

The "*Stock movement*" report will list the Stock code, Stock description, Quantities and Average cost for a selected stock item.

# Stock movement options

![Image](<lib/sidebar-stock-movement-options.png>)

# Stock movement (context menu)

Prints directly to default printer and not to the Reportman preview screen.&nbsp;

In this example, the system's default printer is set to "*Microsoft print to PDF*"&nbsp; where you need to enter a file name to save the file print as "*PDF Document (\*pdf)*" file type on the " *Save Print Output As*" screen.

![Image](<lib/single-item-stock-movement-drill.png>)

&nbsp;

# Report button&nbsp;

Irrespective of the stock item selected on the stock grid, it always prints the first available stock item on the list.&nbsp;

In this example, the first item on the stock grid (i.e. "*Compressor*" is printed while the "*Drill press*" is selected.

![Image](<lib/sidebar-stock-movement-report-button.png>)
