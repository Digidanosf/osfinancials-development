# Spreadsheet Export / Import - Debtor / Creditor accounts

***

osFinancials5/TurboCASH5 supports the "*Excel Workbook (\*.xlsx) file type* Spreadsheet export and import feature#8202;*.* &nbsp;

* [Spreadsheet export](<debtor-creditor-export-excel.md>) - You may export debtor (customer / client) accounts and creditor (supplier / vendor) accounts and save it as a "*Excel Workbook (\*.xlsx) file type.*
* [Spreadsheet import](<debtor-creditor-import-excel.md>) - You may import debtor (customer / client) accounts and creditor (supplier / vendor) accounts from a saved "*Excel Workbook (\*.xlsx) file type.*

This feature may be used to import debtor (customer / client) accounts and creditor (supplier / vendor) accounts into:

* Into the same Set of Books.
* Into a different Set of Books.&nbsp;

  * Created from scratch - [Create your own Set of Books (Advanced)](<create-books-advanced.md>) option for Firebird database types
  * Created from scratch - [Create your own Set of Books (Advanced)](<mssql-sql-database-create.md>) option for Microsoft SQL sever (MSSQL) database types.

**After import**

| ![Image](<lib/admon-tip.png>) | *To check the imported account details, you may print the necessary [listing reports*](<debtors-listing-reports.md>) *for debtor (customer / client) accounts and [listing reports*](<creditors-listing-reports.md>) *for creditor (supplier / vendor) accounts from the Set of Books in which you created the export file.*&nbsp; |
| --- | --- |


You may go to [**Debtors**](<debtor-accounts.md>) (**Default** ribbon)&nbsp; to view and/or edit the imported debtor (customer / client) accounts.&nbsp;

You may go to [**Creditors**](<creditor-accounts.md>) (**Default** ribbon)&nbsp; to view and/or edit the imported creditor (supplier / vendor) accounts. &nbsp;
