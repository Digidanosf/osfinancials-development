# Plugins (Reports menu)

***

Reports for the following two (2) plugins are available:

| ![Image](<lib/icon-plug-in-price-agree.png>) | ***Price agreement plugin - [Manual***](<https://www.osfinancials.org/en/plugin-manuals/513-price-agreement-plugin> "target=\"\_blank\"") ***- [Shop***](<https://www.osfinancials.org/en/webshop/other/price-agreement-plugin> "target=\"\_blank\"")&nbsp; *With this plugin you can have quantity and discount for up to four (4) levels. These prices will overrule the default selling price 1 till 3 and can be set per debtor (customer / client) and/or creditor (supplier / vendor).* *You can also set a special price for specific debtor (customer / client) and/or creditor (supplier / vendor) per product, which overrule the default product price.* *These also a special field available in which you can add the suppliers product number. Also possible for multiple suppliers.* *This field can also be used for information text why this debtor (customer / client) and/or creditor (supplier / vendor) has a special price.* *This option allows you to print the Price agreements for debtor (customer / client) and creditor (supplier / vendor) accounts.* |
| --- | --- |


| ![Image](<lib/icon-plug-in-serial-latch-loc.png>) | ***Serials / location / batch - [Manual***](<https://www.osfinancials.org/en/plugin-manuals/540-locations-batches-and-serials-plugin-en> "target=\"\_blank\"") ***- [Shop***](<https://www.osfinancials.org/en/webshop/other/serial-batch-location> "target=\"\_blank\"")&nbsp; *With this plugin, you can track your stock items in locations, bins and serial numbers or codes where the stock items are kept. Some stock items may also have expiry dates, etc.* |
