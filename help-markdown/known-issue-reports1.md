# Debtor list details&nbsp;

Debtor / Creditor - Accounting information - Bank account no. - Field focus

By default the cursor focus after 4 blank spaces. To enter need to select the blank spaces to enter the bank account no.&nbsp;

A bit sticky but can enter up to 13 characters which will print on the report
