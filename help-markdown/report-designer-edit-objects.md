# Report designer - Edit / Move / Align objects

***

Objects may be selected and edited or moved. These icons will only be active when an object is selected.

1. **Editing Icons** (also available on the **Edit** menu)
1) &nbsp;

   1) ![Image](<lib/reportman-design-icon-select.png>) **Select** - Select an object on the report.&nbsp;
   1) ![Image](<lib/reportman-design-icon-delete.png>) **Delete** - This icon will delete a selected object. (**Keyboard**: **Ctrl + Del**).
   1) ![Image](<lib/reportman-design-icon-cut.png>) **Cut** - This will cut a selected object. (**Keyboard**: **Alt + X**).&nbsp;
   1) ![Image](<lib/reportman-design-icon-copy.png>) **Copy** - This will place a copy of the selected object on the clipboard. (**Keyboard**: **Alt + C**).
   1) ![Image](<lib/reportman-design-icon-paste.png>) **Paste** - This will paste (insert) the object from the clipboard. (**Keyboard**: **Alt + X**).

2. **Moving Object Icons**&nbsp; (also available on the **Edit *→* Move** menu)
1) &nbsp;

   1) ![Image](<lib/reportman-design-icon-move-left.png>) **Move left** - Each click moves the selected object one space to the left. (**Keyboard**: **Ctrl + Left**).
   1) ![Image](<lib/reportman-design-icon-move-right.png>) **Move right** - Each click moves the selected object one space to the right. (**Keyboard**: **Ctrl + Right**).
   1) ![Image](<lib/reportman-design-icon-move-up.png>) **Move up** - Each click moves the selected object one space to the up. (**Keyboard**: **Ctrl + Up**).
   1) ![Image](<lib/reportman-design-icon-move-down.png>) **Move down** - Each click moves the selected object one space to the down. (**Keyboard**: **Ctrl + Down**).

3. **Aligning Object icons** (also available on the **Edit *→* Align** menu) - To activate the align objects, you must select more than one (1) object (using the **Shift** + mouse click) or **Edit → Select → All / All Text** menu options.

   1. ![Image](<lib/reportman-design-icon-align-left.png>) Align left - Aligns selection to the left.
   1. ![Image](<lib/reportman-design-icon-align-right.png>) Align right - Aligns selection to the right.
   1. ![Image](<lib/reportman-design-icon-align-top.png>) Align top - Aligns selection to the top.
   1. &nbsp;![Image](<lib/reportman-design-icon-align-bottom.png>) Align bottom - Aligns selection to the bottom.
   1. ![Image](<lib/reportman-design-icon-align-space-horizontal.png>) Align horizontal space - Aligns selection distributing horizontal space.
   1. ![Image](<lib/reportman-design-icon-align-space-vertical.png>) Align vertical space - Aligns selection distributing vertical space.
