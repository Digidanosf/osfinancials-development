# Graph - Supplier returns per week

***

| ![Image](<lib/admon-tip.png>) | *The [Graph*](<documents-graphs.md>) *feature for each [document type*](<document-types.md>) *(i.e Supplier returns) in **Documents** on the **Default** ribbon, allows you to filter, analyse and view your Supplier returns in a graph using various options to include or exclude in the graph, i.e.:*&nbsp; [*Document types*](<document-types.md>) *- Select the Supplier returns document type.*&nbsp; [*Document status*](<document-status.md>) *- Select All, Posted or Unposted Supplier returns.*&nbsp; [*Filter options*](<documents-filter-search-options.md>) *- Select the various options to include or exclude data to be included in the graph, etc.*&nbsp; |
| --- | --- |


The following two (2) graph reports to analyse your Supplier returns, which are **posted**, for a specific period, may be printed:

* **Graph - Supplier returns per week - Quantities** - displays the total quantities of items on the "*Quantity shipped*" field of **Supplier return documents** per week in a bar chart.&nbsp;
* **Graph - Supplier returns per week - Amount** - displays the total amount (exclusive of tax (VAT/GST/Sales tax)) of **Supplier return documents** per week in a bar chart.

| ![Image](<lib/admon-important.png>) | *This graph only includes Supplier returns which are posted. Unposted Supplier returns will be **NOT** be included in this graph.* *To view a list of unposted documents, which is not updated to the ledger, go to **Input → [Checking unposted items***](<checking-for-unposted-items.md>) *(**Default** ribbon).* |
| --- | --- |


# Graph - Supplier returns per week options

**To print Graph - Supplier returns per week reports:**&nbsp;

1. On the **Reports** ribbon, select **User reports → Graphs → Graphs per week → Purchases → Supplier returns per period**.&nbsp;

![Image](<lib/graphs-supplier-returns-per-week-options.png>)

2. Select the following:
1) &nbsp;

   1) **Report** - Select the **Graph - Supplier returns per week - Quantities** or **Graph - Supplier returns per week - Amount** report.
   1) **Period** - Select a period on the **Per period** tab. You may click on the **Free selection** tab to select a specific date or a range of dates to include in the graph.&nbsp;

3. Click on the **OK** button.&nbsp;

# Printed : Graph - Supplier returns per week - Quantities

An example of the "*Graph - Supplier returns per week - Quantities*", is as follows:

![Image](<lib/graphs-supplier-returns-per-week-print-1.png>)

The details is as follows:

1. **Vertical Axis** - The total quantities on Supplier return documents per week.&nbsp; &nbsp;
1. **Horizontal Axis** - This is the Number of the Week in the calendar year (1 to 52).&nbsp;
1. **Totals -** **Quantities** - The total quantities of all Supplier return documents for the selected period.

# Printed : Graph - Supplier returns per week - Amount

An example of the "*Graph - Supplier returns per week - Amount*", is as follows:

![Image](<lib/graphs-supplier-returns-per-week-print-2.png>)

The details is as follows:

1. **Vertical Axis** - The total amount for Supplier return documents per week.&nbsp; &nbsp;
1. **Horizontal Axis** - This is the Number of the Week in the calendar year (1 to 52).&nbsp;
1. **Exclusive total** - The total amount (exclusive of tax (VAT/GST/Sales tax)) of all Supplier return documents for the selected period.
