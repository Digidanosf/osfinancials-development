# System requirements

***

The system requirements for osFinancials5/TurboCASH5 are as follows:

1. **Operating System**: A 32-bit or 64-bit Microsoft Windows operating system is required. Supported versions include Windows 98, Windows 2000, Windows ME, Windows NT, Windows XP, Windows Vista, Windows 7, Windows 8, Windows 10, or Windows 11.
1. **RAM**: For a single-user setup, at least 256 MB of RAM is required. For a multi-user setup, a minimum of 512 MB of RAM is recommended.
1. **Hard Disk Space**: Approximately 1 GB of available hard disk space is needed to install and run osFinancials5/TurboCASH5.
1. **USB Port and Drives**: A USB port is required, and available USB drives are needed to save and store backups.
1. **Screen Resolution**: An SVGA/LCD/LED screen with a resolution of 1440 x 900 or higher is recommended.

6. **System Number Format**: Configuring the [System number format setting](<system-setting-number-format.md>) to print reports like “[*Budget vs actual*](<ledger-analyser-budget-actual.md>)” and “[*This year vs last year*](<ledger-analyser-this-last-year.md>)”.
6. **Spreadsheet Software**: Microsoft Excel or other spreadsheet programs such as [LibreOffice](<https://www.libreoffice.org/> "target=\"\_blank\""), [OpenOffice](<https://www.openoffice.org/> "target=\"\_blank\"") or [UltraOffice](<https://www.compuclever.com/products/ultra-office/> "target=\"\_blank\"") should be installed to save reports or export data from the Ledger analyser and other options.
6. **Recommended Device**: While osFinancials5/TurboCASH5 may run on older devices, it is recommended to use a better device for more secure and efficient operation. Modern devices are significantly more powerful than older ones, making them more suitable for running osFinancials smoothly.

Please note that the information provided here is specific to osFinancials5/TurboCASH5. Always refer to the official documentation and system requirements provided by the osFinancials team to ensure accurate and up-to-date information. Additionally, consider using the most recent and supported operating system and hardware for optimal performance and security.
