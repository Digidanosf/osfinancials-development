# Reports - Page headers and Page footers

***

# Report headers

The standard data in the report headers will be displayed on all reports (except on [Debtor statements](<debtor-statement.md>), [Creditor remittance advises](<creditor-remittance-advices.md>), [Document layout files](<document-layout-files.md>) en [Spreadsheet reports](<spreadsheet-reports.md>)).&nbsp;

![Image](<lib/report-header.png>)

The company's contact details and registration numbers, as entered in the **Setup → [Company info**](<setup-company-info.md>) (**Setup** ribbon).

# Report footers

The standard data in the report footers will be displayed on all reports (except on [Debtor statements](<debtor-statement.md>), [Creditor remittance advises](<creditor-remittance-advices.md>), [Document layout files](<document-layout-files.md>) and [Spreadsheet reports](<spreadsheet-reports.md>)). &nbsp;

![Image](<lib/report-footer.png>)

At the end of each page of a report, the page footer will display the following details:

1. **Name for Set of Books -** The Company name as entered in **Setup → [Company info**](<setup-company-info.md>) (**Setup** ribbon).
1. **Printed:** Your system's date and time when the report was printed.
1. **Data** - Name of the Set of Books and the full path of the Set of Books (e.g. ."*Data: C:\\osFinancials5\\books\\MY-BOOKS/books.fdb*")
1. **System user** - The user of your computer's operating system.
1. **osFinancials user** / **TurboCASH user** - If users is added in **Setup → [Access control**](<access-control.md>) (**Setup** ribbon),the user name will be displayed.
1. **Page:** Page number and the last page number of the report.&nbsp;
