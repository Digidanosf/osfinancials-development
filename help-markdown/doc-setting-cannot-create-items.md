# Cannot create items

***

By default, if an invalid stock code, description, etc. is typed, when selecting a stock item on the Stock lookup, an confirmation screen to create a new stock item will be displayed.&nbsp;

If you click on the **Yes** button, you may first create the item and then return to continue with processing the document. To turn this feature off, select (tick) this "*Cannot create items*" option.
