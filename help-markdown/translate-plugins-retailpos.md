# RetailPOS - Reports

![Image](<lib/translate-retail-pos-9.png>)

**Notes on reports**

* These reports may not fully translatable
* Report names may need to be added to the print-outs to show if it is Cashup report, Sales per period or Posted payments report
* Address will be displayed Address1 / Postal code / Address 2 / Address 3&nbsp; for the Nederlands basetype 0&nbsp;

for international base type 1 - Address will be displayed Address1 / Address 2 / Address 3 / Postal code

* Printed from Salesperson tab options and when Close POS option prints automatically.

**Location** :&nbsp; ...\\plug\_ins\\reports\\REPORTS\\HORECAPOS\\DAYVIEW&nbsp;
