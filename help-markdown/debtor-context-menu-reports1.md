# Creditor - Context menu - Reports

***

The "*Reports*" context menu option allows you to print postal or delivery address labels for a selected creditor (supplier / vendor) account.&nbsp;

**To print labels for a selected creditor (supplier / vendor) account:**

1. On the "*Creditors*" list screen, select a creditor (supplier / vendor) account.
1. Right-click and select "*Reports → Selected creditor label*" option or the&nbsp; select "*Reports → Selected creditor delivery label*" option on the context menu. This will print the postal address label or the delivery address label of the selected creditor (supplier / vendor) account.&nbsp;

![Image](<lib/creditor-context-menu-reports.png>)

| ![Image](<lib/admon-note.png>) | You will only be allowed to print the postal address label or delivery address label for one selected creditor (supplier / vendor) account at a time. |
