# Add / Create task - Calendar

***

**To add (create) a task:**&nbsp;

1. To create a new task, simply right-click on any area of the Task list section of the Agenda screen.&nbsp;
1. On the context menu, select the **Add task...** option. The Untitled - Task screen will be displayed.
1. Enter the subject (topic) of the task.
1. Select the Due date when the task should be completed.
1. Enter any additional information or notes for the task.
1. Click on the **OK** button to save the task.&nbsp;
