# Debtor accounts - Documents tab

***

All sales documents (i.e. Invoices, Credit notes, Quotes and/or POS Invoices) which have been processed for the selected debtor (customer / client) account will be listed here.&nbsp;

| ![Image](<lib/admon-tip.png>) | *To maximise the **Documents** tab and/or Debit / Credit transactions section of the screen, click on the*&nbsp; ![Image](<lib/work-screens-drag-01.png>) *button to maximise the **Documents** tab. You may also click and drag any part of the screen up / or down.*&nbsp; ![Image](<lib/work-screens-drag-02.png>) |
| --- | --- |


![Image](<lib/debtor-tab-documents.png>)\
The options are as follows:

1. **Print** button - This will print the selected document to the default or selected output method.

| ![Image](<lib/admon-tip.png>) | *You may also double-click on a selected document to print it.* |
| --- | --- |


2. [**Layout file**](<document-layout-files.md>) - This is by default blank. The selected document will be printed using the layout file as selected for each document type (i.e. Invoices, Credit notes and Quotes) in [**Documents setup**](<documents-setup.md>) (**Setup → [Documents setup**](<documents-setup.md>)).

| ![Image](<lib/admon-note.png>) | *If specific layout files is selected on the [**Delivery address***](<debtor-tab-delivery-address.md>) *tab of a debtor (customer / client) account, te layout files will be used to print the selected documents.* |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *You may change the layout file by selecting a layout file from the list.*&nbsp; *These layout files are stored in the ...\\plug\_ins\\reports\\DOCUMENTS\\DOCUMENTS directory of your system.* |
| --- | --- |


3. **Send report to** - The default option as set in the "*Output Method"* field on **Setup → Company info** (**Options**) or **Setup → Access control** (**Reports access** tab). You any change this to any other option (i.e. "*Screen, Printer, File, E-mail*" or "*Fax*"), if required.

| ![Image](<lib/admon-tip.png>) | *You may click on the column headings (i.e. Document number, Date, Description or Reference) to change the sort sequence from the ascending to the descending sequence.* *You may also click and drag any of the column headings to the left or right to change the sequence in which the columns are to be displayed.* |
| --- | --- |


# View Transactions - Show transact context menu

| ![Image](<lib/admon-important.png>) | *The **Documents** tab lists all posted and unposted documents (i.e. Invoices, Credit notes and Quotes. The **Show transactions** context menu option will not do anything, if an order or an unposted document is selected.* |
| --- | --- |


1. Select a document (Invoices or Credit notes) and right-click, select **Show transactions**. The posted batch transactions will be displayed in the "*Transactions*" screen.&nbsp;

![Image](<lib/debtor-tab-transactions-show-transactions.png>)

2. Click on the **Print** button to print the posted batch transactions, if required.&nbsp;

![Image](<lib/debtor-tab-transactions-transactions-pint.png>)
