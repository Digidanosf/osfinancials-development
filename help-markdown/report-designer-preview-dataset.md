# Report designer - View Dataset

***

**To view the datasets:**

1. On the "*Report manager designer*" screen, click on the ![Image](<lib/reportman-design-icon-9-connections.png>) **Data access configuration** icon
1. Select the Dataset from the list and click on the **Show data** button.&nbsp;

![Image](<lib/report-desingner-dataset-view.png>)

3. In this example, the Data (for the "*COMPANYINFO*" Dataset) is as follows:&nbsp;

If you click on the ![Image](<lib/reportman-dataset-icon-close.png>) **Exit** button, you will return to the "*Database connections and datasets"* screen. You may view other Datasets.&nbsp;

4. For example, if you view the "*DOCUMENTLINE*" Dataset, and you have more than one line in the document (transactions and/or comments), you may click on the ![Image](<lib/reportman-dataset-icon-next.png>) **Next** and ![Image](<lib/reportman-dataset-icon-first.png>) **First record** buttons to view the transactions / and or comments. &nbsp; &nbsp;
4. Click on the **OK** button of the "*Database connections and datasets"* screen.&nbsp;
