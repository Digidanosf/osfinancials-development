# Creditor listing - Delivery address (Reports menu)

***

The Creditor listing - Delivery address report will list the Account code, Name of the Creditor (account description), Delivery address, Telephone 1 / 2, Fax, E-mail Address and Website. &nbsp;

# Creditor - List delivery address report options

**To print a Creditor - List delivery address report:**

1. On the **Reports** ribbon, select **Reports → Creditors → Listing**.&nbsp;
1. Select "*List delivery address*".

![Image](<lib/reports-creditor-list-delivery-address-option.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a creditor (supplier / vendor) account or a range of creditor (supplier / vendor) accounts, to include in the report.
   1) **Sequence** - Select "*Creditor code, Description, Address 3* or *Postal code*" to print the report details in the order (sort) in which you wish to list the data.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The report will only be printed for those creditor (supplier / vendor) accounts linked to the selected Creditor reporting group 1.

4. Click on the **OK** button.&nbsp;
