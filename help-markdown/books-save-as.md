# Save as

***

The "*Save as*" option in osFinancials5/TurboCASH5 allows you to save a Set of Books to any directory or folder on your system. To use this option, you must have a Set of Books open and select the option to save the entire Set of Books with a different name in the directory of your choice. This allows you to create a copy of the Set of Books with a new name in a different location without affecting the original data.

| ![Image](<lib/admon-note.png>) | *The "Save as" screen will also be launched if you download one of the free Sets of Books (templates) (using the option, [Let osFinancials help you to create a Set of Books*](<create-books-default-download.md>) */ [Let TurboCASH5 help you to create a Set of Books*](<create-books-default-download.md>)*)* *After&nbsp; downloading a Set of Books, the Set of Books (template) will automatically open in the language associated with the country and language.*&nbsp; *See a list of the [Sets of Books available for a download*](<sets-of-books-download-templates.md>)*.* |
| --- | --- |


**To save a Set of Books as:**&nbsp;

1. On the [**Start**](<start-ribbon.md>) ribbon, [**Open**](<open-set-of-books.md>) the Set of Books which you need to save under a new name.

| ![Image](<lib/admon-important.png>) | *&nbsp;If you already have an open Set of Books, please make sure that it is the actual Set of Books you wish to save. If not, close the Set of Books and open the Set of Books you wish to save.*&nbsp; |
| --- | --- |


2. On the [**Start**](<start-ribbon.md>) ribbon, select **Save as** (**Ctrl + S**). The "*Save as"* screen is displayed:&nbsp;

![Image](<lib/books-save-as-1.png>)

3. The default or installation Directory, in which osFinancials5/TurboCASH5 is installed, will be displayed.&nbsp;

| ![Image](<lib/admon-tip.png>) | *If you wish to Save the Set of Books in another Directory or location on your system, you may click on the Folder icon on the right hand side of the Directory field. The "Browse for Folder" screen will be displayed on which you may then select a valid directory or folder on your system in which you wish to Save the Set of Books.*&nbsp; |
| --- | --- |


4. Enter a name in which you wish to save your Set of Books. The name for the Set of Books must not already exist.
4. Click on the **Save** button. The full Set of Books will be saved into the specific directory with the specified name. When this process is finished, an information message will be displayed. &nbsp;

*"Set of Books copied"*

6. Click on the **OK** button to close (exit) the "*Save as"* screen.
6. The source Set of Books will be opened.&nbsp;

| ![Image](<lib/admon-note.png>) | *To open the Saved Set of Books, click on [**Open***](<open-set-of-books.md>) *([**Start***](<start-ribbon.md>) *ribbon) and select the new saved (copied) Set of Books can then be selected from the list.*&nbsp; |
| --- | --- |
