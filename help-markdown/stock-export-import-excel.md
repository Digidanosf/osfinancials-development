# Spreadsheet Export / Import - Stock items

***

osFinancials5/TurboCASH5 supports the "*Excel Workbook (\*.xlsx) file type* Spreadsheet import and export feature#8202;*.* &nbsp;

* [Spreadsheet export](<stock-export-excel.md>) - You may export stock items and save it as a "*Excel Workbook (\*.xlsx) file type.*
* [Spreadsheet import](<stock-import-excel.md>) - You may import stock items from a saved "*Excel Workbook (\*.xlsx) file type.*

This feature may be used to import stock items into:

* Into the same Set of Books.
* Into a different Set of Books.
* Into a new Set of Books -Created from scratch - [Create your own Set of Books (Advanced)](<create-books-advanced.md>) option for Firebird database types.
* Into a new Set of Books - Created from scratch - [Create your own Set of Books (Advanced)](<mssql-sql-database-create.md>) option for Microsoft SQL sever (MSSQL) database types.

| ![Image](<lib/admon-tip.png>) | *Before you may import stock items in a new Set of Books, you need to do the necessary setup and configuration for stock.*&nbsp; *Once this is done, you may start to import stock items from a saved "Excel Workbook (\*.xlsx) file type.* |
| --- | --- |


This option will allow you to Export your stock item file into an Excel spreadsheet. You may then add new stock items, edit existing stock items and import it back into your stock code file.

You may access the "*Data Import Export*" feature from the following options:&nbsp;

* [**Import / Export**](<stock-item-tab-import-export.md>) tab of the Stock items (**Advanced** mode) on the **Default** ribbon.

![Image](<lib/stock-excel-import-options-01.png>)&nbsp;

* [**Import**](<stock-import-tab-delimited.md>) option on the **Setup** ribbon.

![Image](<lib/stock-spreadsheet-export-import-setup-ribbon.png>)

Select **Stock** as the "*Data type to import*". Click on the **Excel import export** button.&nbsp; The "*Data Import Export*" spreadsheet&nbsp; will displayed column headings for the Stock items (without any data).
