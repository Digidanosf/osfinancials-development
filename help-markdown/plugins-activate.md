# Activate plugins

***

# Plugin activation

Plugins are included in the osFinancials5/TurboCASH5 installation. By default, some Plugins are not available from the **Plugins** menu on the **Setup** ribbon.&nbsp;

To add plugins to the **Plugins** menu, you need to activate the plugins, you require. This will update the "*Plugins load list*". Activated Plugins will be available on the **Plugins** menu.

**To activate plugins:**

1. On the **Setup** ribbon, select **Tools → Activate** **plugins**.&nbsp;
1. On the "*Plugin load list*" screen, you may select to activate / deactivate, the plugins you have purchased on the list.

![Image](<lib/plugin-activate-012.png>)

3. Click on the **Close** button.
3. The selected plugins will be added to the **Plugins** menu on the **Setup** ribbon.&nbsp;
3. Plugins needs a licence before these can be activated. Some plugins, if not registered (purchased on the [Webshop](<https://www.osfinancials.org/en/webshop> "target=\"\_blank\"")), can be used for a number of transactions or records only.

# Using plugins&nbsp;

Once the plug-ins are activated, you must enable (enable) the plug-ins you purchased in the **Plugins** menu (**Setup** ribbon).&nbsp;

Some plug-ins must be activated for each Set of Books in osFinancials5/TurboCASH5.

All plug-ins are enabled in the **Tools → Activate plugins** (**Setup** ribbon) and will be available in the Plug-ins menu. However, there are two exceptions:

1. [**Bank Import Plugin**](<plugin-bank-import.md>) - This plugin must be enabled in the **Setup → System parameters** (**Setup** ribbon).
1. [**Payroll Import Plugin**](<batch-payroll-import-plugin.md>) - This plugin is accessed in the **Process batch** option (**F9:Process**), provided you have a valid, and you have a valid input file from a supported payroll software.

Start osFinancials5/TurboCASH5 again to fully activate the enabled plugins.

# Retrieving plugin licences

**To retrieve plugin licences:**

1. Go to the **Start** ribbon (tab). Then click on **Online registration**.&nbsp;
1. On the “*Register your osFinancials* / *Register your TurboCASH*” screen, click on the **Get it online** button.
1. On the next screen, enter your username and password for this website.&nbsp;
1. Then click **Send** (bottom of screen) and your plugin codes are retrieved from our servers.
1. You will see all the plugins you purchased or plugins you have asked for a trial.&nbsp;
