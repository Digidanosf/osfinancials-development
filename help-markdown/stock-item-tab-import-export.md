# Import / Export tab

***

The **Import / Export** tab on the on the Stock item list screen will only be available when the mode in set to Default. In the **Advanced mode** is not selected, the **Import / Export** tab will not be displayed.

| ![Image](<lib/admon-tip.png>) | *The stock item screen may also be launched as a floating screen from the **Input → [Adjust stock***](<stock-adjustments-input-menu.md>) *menu on the **Default** ribbon.* |
| --- | --- |


![Image](<lib/stock-list-tabs-import-export.png>)

The options, is as follows:

1. [**Spreadsheet import**](<stock-export-import-excel.md>) -&nbsp;
- &nbsp;

  - [Spreadsheet export](<stock-export-excel.md>) - You may export stock items and save it as a "*Excel Workbook (\*.xlsx) file type.*
  - [Spreadsheet import](<stock-import-excel.md>) - You may import stock items from a saved "*Excel Workbook (\*.xlsx) file type.*

2. [**Export**](<stock-export-tab-delimited.md>) - Create an Export file in the "*Tab delimited (\*.txt)*" (default - recommended).

| ![Image](<lib/admon-tip.png>) | *Stock items may also be exported in [**Export***](<stock-export-tab-delimited.md>) ***- Stock items** ([**Setup***](<setup-ribbon.md>) *ribbon).* |
| --- | --- |


3. [**Import**](<stock-import-tab-delimited.md>) - Import a valid file exported via the Import option.&nbsp;

| ![Image](<lib/admon-tip.png>) | *Stock items may also be imported from [**Import***](<stock-import-tab-delimited.md>) *- **Stock items** ([**Setup***](<setup-ribbon.md>) *ribbon).* |
| --- | --- |


4. **Enter is next record** - To be advised.
4. **Auto detail on one record** - To be advised.
4. **Match on creditor code** - To be advised.
