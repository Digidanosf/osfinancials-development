# Stock - Print labels (Barcodes) (Reports menu)

***

You may print the following two (2) barcodes for stock items.&nbsp;

1. **Barcode** - Displays the stock code and stock description and the barcode.&nbsp;
1. **Yagoda barcode** - Displays the stock code, stock description and barcode as well as the selling price (exclusive of Tax (VAT/GST/Sales Tax)).&nbsp;

# Print labels options

**To print Barcodes for Stock items:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Stock → Print labels**.&nbsp;
1. Select the "*Barcode*" (default) or "*Yagoda barcode*".

![Image](<lib/reports-stock-labels-barcode-options.png>)

3. Select the following:
1) &nbsp;

   1) **From... To...** - Select the stock code(s) to print barcodes for.&nbsp;
   1) **Sequence** - Select "*Stock code* or *Stock description*" to sort the order in which you need to print the labels.
   1) **Reporting group 1** - This is optional and by default not selected (not ticked). You may select (tick) the reporting group(s) to include in the report.&nbsp;

4. Click on the **OK** button.&nbsp;
