# Contacts (Reports menu)

***

This report lists all contacts linked to debtor (customer/client) accounts as well as creditor (supplier/vendor) accounts.&nbsp;

| ![Image](<lib/admon-tip.png>) | *To list only the contacts for debtor (customer / client) accounts, you need to print it in **Reports → Debtors → Listing - [Contacts***](<debtors-list-contacts.md>) *menu (**Reports** ribbon). This report specifically focuses on contacts related to debtors (customers/clients). It provides a listing of contacts associated with your debtor accounts.* *To list only the contacts for creditor (supplier / vendor) accounts, you need to print it in **Reports → Creditors → Listing - [Contacts***](<creditors-list-contacts.md>) *menu (**Reports** ribbon). This report specifically lists contacts related to creditors (suppliers/vendors). It presents a comprehensive view of contacts associated with your creditor accounts.* |
| --- | --- |


To add new contacts, edit or delete contacts, you may use the following options on the **Default** ribbon.

1. [**Contacts**](<contacts.md>): This option will list all the contacts, where you can manage and maintain contact information for debtors (customers/clients) and creditors (suppliers/vendors). You may add new contacts, update contact details, delete contacts and print a list of all the contacts in your Set of Books.
1. [**Debtors - Contacts**](<debtor-tab-contacts.md>) tab: This tab specifically focuses on managing contacts associated with debtor accounts. You can add, edit, or delete contacts linked to debtor accounts, ensuring that the contact details are up to date.
1. [**Creditors - Contacts**](<creditor-tab-contacts.md>) tab: Similar to the [**Debtors - Contacts**](<debtor-tab-contacts.md>) tab, this tab is dedicated to managing contacts associated with creditor accounts. You can modify the contact information for your suppliers/vendors to maintain accurate records.
1. [**Calendar - Contacts**](<calendar-contacts.md>) feature: provides the ability to manage contacts, ensuring that any changes made to contact information are reflected across the system. The following options on the **Default** ribbon allow you to create, edit, or delete contacts, with all updates being synchronized with the **Contacts** in the Calendar.

By utilising these options, you can conveniently manage contacts within osFinancials5/TurboCASH5, ensuring that all contact information remains consistent and accessible across different sections in a Set of Books.

# Contacts report options

**To print a list of all contacts:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Contacts**.&nbsp;

![Image](<lib/reports-contacts-options.png>)

2. **Sequence** - Select "*First name, Name, Account"* or *"Description*".
2. Click on **OK**. &nbsp;
