# Stock - Reorder - Current stock levels (Reports menu)

***

Current Stock Levels - Displays all stock item quantities.&nbsp;

# Reorder - Current stock levels report options

**To print Reorder - Current stock levels reports:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Stock → Reorder**.&nbsp;
1. Select "*Current stock levels*" report, if not already selected.&nbsp;

![Image](<lib/reports-stock-reorder-current-levels-options.png>)

3. Select the following:
1) &nbsp;

   1) **From... To...** - Select the stock code(s) to include in the report.&nbsp;
   1) **Reporting group 1** - This is optional and by default not selected (not ticked). You may select (tick) the reporting group(s) to include in the report.&nbsp;
   1) **Sequence** - Select "*Stock code* or *Stock description*" to sort the order in which you need to print the data.
   1) **Show details** - By default, this option is not selected. It will display the reorder quantities. If this option is selected, it will display the details for each stock item. &nbsp;

4. Click on the **OK** button.&nbsp;
