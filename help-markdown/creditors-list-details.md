# Creditor listing - List details (Reports menu)

***

The Creditor listing - List details report will display the Contact details, Registration details, Banking details, Reporting details, Accounting terms and information for each creditor (supplier / vendor) account.&nbsp;

# Creditor - List details report options

**To print a Creditor listing - List details report:**

1. On the **Reports** ribbon, select **Reports → Creditors → Listing**.&nbsp;
1. Select the "*List details*".

![Image](<lib/reports-creditor-listing-details-options.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a creditor (supplier / vendor) account or a range of creditor (supplier / vendor) accounts, to include in the report.
   1) **Sequence** - Select "*Creditor code, Description,* *Reporting group 1* or*Reporting group*&nbsp; *2*" to print the report details in the order (sort) in which you wish to list the data.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The report will only be printed for those creditor (supplier / vendor) accounts linked to the selected Creditor reporting group 1.

4. Click on the **OK** button.&nbsp;

# Printed : Creditor list - Details report

![Image](<lib/reports-creditor-listing-details-print.png>)

The List details report will display the following information:

1. **Contact details** - Contactperson, Telephone 1 / 2, Fax, E-Mail, Website, Postal address, Delivery address and Language of the creditor (supplier / vendor).&nbsp;
1. **Registration details** - Company Registration number and Tax registration number.
1. **Banking details** - Bank account type, Name of Bank account, Bank account number and Branch code.&nbsp;
1. **Reporting details** - Reporting group 1 / 2, Layout file 1 (Purchases), Layout file 2 (Supplier returns) and Layout file 3 (Orders).&nbsp; If no layout files are selected, the documents will be printed for the global settings in **Setup → Documents setup**.
1. **Accounting terms and information** - Remittance message, Credit limit, Due days (payment terms), Charge amount, account settings (i.e. Open item / Balance B/F) and Account disabled / inactive) as well as the Debtor code from the creditor (supplier / vendor). &nbsp;
