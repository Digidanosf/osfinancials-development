# Translate Multi-language stock descriptions

***

The Stock Item descriptions, as entered in the Description field of the Edit Stock Items screen, are by default displayed on document layout files.

| ![Image](<lib/admon-note.png>) | *To use multi-language Stock Descriptions, Language Groups need to be created / added in **Setup → Groups** - **Languages** tab.* |
| --- | --- |


The translated stock item descriptions should print on layout files.

**To Add Translations to Stock Items:**

1. On the **Default** ribbon, select **Stock items**.
1. Select the Stock item.
1. Click on the **Edit** button (or double-click on the stock item).
1. Click on the **Languages** tab.&nbsp;

![Image](<lib/multi-language-stock-languages.png>)

5. Select the Language and enter the name (description) for each language.
5. Click on the **Save** button.
5. Repeat the process for all stock items.&nbsp;

| ![Image](<lib/admon-tip.png>) | *If you have OpenOffice installed on your system, you may double-click in the Description field of any language, to launch OpenOffice Writer/Web.* ![Image](<lib/multi-language-stock-open-office.png>) *Write your text and use OpenOffice to format your text, etc., if necessary. A message will be displayed:*&nbsp; *"Press ok if you are done with the document to continue\!*&nbsp; *Once finished, click the **OK** button to transfer your text to the "Description" field.*&nbsp; |
