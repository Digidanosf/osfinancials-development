# Context menu - Spreadsheet import

***

This option allows you to export and import documents in a Spreadsheet.

**To import documents:**&nbsp;

1. On **Input → Documents** (Documents list), select the Document type.
1. Right-click and select the **Documents → Spreadsheet import** option on the context-menu.&nbsp;
