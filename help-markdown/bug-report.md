# Bug report

***

This will land on the&nbsp; osFinancials [Forum page](<https://www.osfinancials.org/en/forum-en?view=category\&catid=8> "target=\"\_blank\"") where you can search and will probably find a solution to your bug or issue. If not available, you may report any bugs or issues you encounter while using the program.

**To access the Bug reporting system in osFinancials5/TurboCASH5:**

On the **Start** ribbon, select **Help → Bug report**.&nbsp;

This will land on the [Forum page](<https://www.osfinancials.org/en/forum-en?view=category\&catid=8> "target=\"\_blank\""). &nbsp;

Some or most bugs are resolved with a later release or an update.&nbsp;
