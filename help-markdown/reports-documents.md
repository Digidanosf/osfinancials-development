# Documents (Reports menu)

***

The following reports are available to print lists of documents:![Image](<lib/reports-documents-listing-menu.png>)

1. **Document listing** – Select Invoices, Credit notes, Quotes, Purchases, Supplier returns and Orders.&nbsp;
1. **Document listing per account**&nbsp;
1) &nbsp;

   1) **Sales documents** – Select to list only posted or unposted Invoices or Credit notes or unconfirmed / open / confirmed Quotes.
   1) **Purchase documents** – Select to list only posted or unposted Purchases or Supplier returns or unconfirmed / open / confirmed Orders.

3. **Unprinted document listing** – This is similar to the Document listing per account reports. It will list all documents which have not yet been printed.&nbsp;
1) &nbsp;

   1) **Sales documents** – Select to list the posted and unposted sales documents (i.e. Invoices, Credit notes and Quotes) for selected debtor (customer / client) account(s), which is not printed.&nbsp;
   1) **Purchase documents** – Select to list the posted and unposted purchase documents (i.e. Purchase documents, Supplier returns and Orders) for selected creditor (supplier / vendor) account(s), which is not printed.&nbsp;
