# Calendar- Resource name

***

The Resource name links the events, tasks, etc. to a specific name in the Calendar. It will also print on the [Calendar report](<calendar-hours-report.md>).&nbsp;

| ![Image](<lib/agendamultiuser1.jpg>) | ***Multi-User calendar (agenda)** - [Manual*](<https://www.osfinancials.org/en/plugin-manuals/516-multi-user-calendar> "target=\"\_blank\"") - [*Shop*](<https://www.osfinancials.org/index.php/en/webshop/other/multi-user-calendar-plug-in> "target=\"\_blank\"") - ***Licence** :* *Once-off -*&nbsp; *This plugin adds the multi-user agenda (multiple agendas) to the Calendar (Planner). You can add agendas and link them as default to the current user. For every agenda a tab will appear in the agenda Calendar so you can see what the appointments, etc. are on other agendas. You can have a agenda for each employee, salesperson, user, etc. and create invoices directly from his / her agenda with a simple click. Each agenda has it's own calendar and task list.* |
| --- | --- |


By default a Resource is already defined as osFinancials or osFinancials5/TurboCASH5 in each of the Firebird database templates you may download from the [Downloads](<create-books-default-download.md>) screen launched from the Creation wizard - [Let osFinancials help you to create a Set of Books](<create-books-default-download.md>) option.&nbsp;

After creating a Blank database on the Creation wizard - [Customise your own Set of Books (Advanced)](<create-books-advanced.md>) option, no resource is available for the blank database.&nbsp;

![Image](<lib/calendar-resource-not-available.png>)

When you start to add an Event to the Calendar for the first time, the following confirmation message is displayed:

*No resources have been defined\! Would you like to add one now?*

Click **Yes** to add a Resource name.&nbsp;

| ![Image](<lib/admon-note.png>) | *The resource name or description field can accommodate up to 255 characters in the "RESOURCES" table.*&nbsp; *The [Task*](<calendar-tasks.md>) *form can only display up to 12 characters.* |
| --- | --- |


![Image](<lib/calendar-resource-editor.png>)

Click **OK** to save the resource.&nbsp;
