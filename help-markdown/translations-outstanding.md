# Copy / Confirm documents - Comments / Remarks

## Edit → Repeating invoices - Remarks not copied

Copies Comments but No remarks is copied on Edit **→** Repeating invoices&nbsp;

In the Calendar - Repeating invoices appointment process - Copies remarks as well as comments.

## Input → Confirm - Comments and Remarks not copied

Copies no Comments and Remarks on Input **→** Confirm (Shift+F3) -&nbsp;

In Documents - Quotes context menu&nbsp; - Confirm quotes and orders - Copies remarks as well as comments.

## Documents setup - Print standard logo&nbsp;

Documents setup - Print standard logo Setting does not work

* To print the logo need to Load the logo in Setup **→** Company info - Options tab&nbsp;
* If you do not want the logo to print, simply remove (clear) the logo in Setup **→** Company info - Options tab&nbsp;
