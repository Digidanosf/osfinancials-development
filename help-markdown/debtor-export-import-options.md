# Export / Import Debtor accounts

***

You may export and import debtor (customer / client) accounts and creditor (supplier / vendor).&nbsp;

This feature may be used to import debtor (customer / client) accounts and creditor (supplier / vendor) into:

* Into the same Set of Books.
* Into a different Set of Books.
* Into a new Set of Books - Created from scratch - [Create your own Set of Books (Advanced)](<create-books-advanced.md>) option for Firebird database types.
* Into a new Set of Books - Created from scratch - [Create your own Set of Books (Advanced)](<mssql-sql-database-create.md>) option for Microsoft SQL sever (MSSQL) database types.

| ![Image](<lib/admon-important.png>) | *The debtor (customer / client) accounts (Chart of Accounts) are maintained in [**Debtors***](<debtor-accounts.md>) *(**Default** ribbon).*&nbsp; *The creditor (supplier / vendor) accounts (Chart of Accounts) are maintained in [**Creditors***](<creditor-accounts.md>) *(**Default** ribbon).*&nbsp; *Reports for the accounts (Chart of Accounts) are printed in **Reports → Ledger → Listing** - [**Chart of Accounts***](<report-ledger-chart-of-accounts.md>) *(**Reports** ribbon).* *Before using this feature, it is recommended to print the report.*&nbsp; |
| --- | --- |


&nbsp;

**Export / Import options on Setup ribbon**&nbsp;

To be able to import the data for debtor (customer / client) accounts and creditor (supplier / vendor) accounts, you need to created a "*Tab delimited*" (\*.txt) export file in the&nbsp; [**Export**](<debtor-creditor-export-excel.md>) (**Setup** ribbon).&nbsp;

It is recommended that the debtor (customer / client) accounts and creditor (supplier / vendor) accounts is exported in the "*Tab delimited*" (\*.txt) file format. This file format may be used to import the "*Tab delimited*" (\*.txt) file format in the [**Import**](<debtor-import-tab-delimited.md>) (**Setup** ribbon).

# Spreadsheet export / import&nbsp;

osFinancials5/TurboCASH5 supports the "*Excel Workbook (\*.xlsx)* file type Spreadsheet import and export feature#8202;*.* &nbsp;

* [Spreadsheet export](<debtor-creditor-export-excel.md>) - You may export debtor (customer / client) accounts and creditor (supplier / vendor) accounts and save it as a "*Excel Workbook (\*.xlsx) file type.*
* [Spreadsheet import](<debtor-creditor-import-excel.md>) - You may import debtor (customer / client) accounts and creditor (supplier / vendor) accounts from a saved "*Excel Workbook (\*.xlsx) file type.*
