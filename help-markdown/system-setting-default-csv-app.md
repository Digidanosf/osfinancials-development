# System settings - Default app for csv file types

***

# Open saved Export.txt files in your system's app (program) associated with the CSV (Comma Separated Values) file type

The **Export** button on the following features will open the saved "*Export.txt*" text file in your system's app (program) associated with the CSV (Comma Separated Values) file type:

* [**Ledger analyser 1**](<ledger-analyser.md>) - [Trial balance](<ledger-analyser-trial-balance.md>), [Income statement](<ledger-analyser-income-statement.md>), [Balance sheet](<ledger-analyser-balance-sheet.md>), [Standard column balances](<ledger-analyser-standard-ccolumn.md>) (column view of the [Trial balance](<ledger-analyser-trial-balance.md>), [Income statement](<ledger-analyser-income-statement.md>) and [Balance sheet](<ledger-analyser-balance-sheet.md>) in one report), [Budget vs actual](<ledger-analyser-budget-actual.md>), [This year vs last year](<ledger-analyser-this-last-year.md>) as well as six (6) report types for [Column balances](<ledger-analyser-column-balances.md>). which cannot be printed, but exported to a spreadsheet only.
* [**T-Account viewer**](<t-account-viewer.md>) - In addition to launching the T-Account viewer from the [**Ledger analyser 1**](<ledger-analyser.md>) / [**Ledger analyser 2**](<ledger-analyser.md>), on the **Reports** ribbon; the T-Account viewer may also be accessed on the following options on the **Default** ribbon:

  * [**Search - Investigator**](<search-investigator.md>) - Account or Batch type searched items.
  * [**Debtors** - **Transactions**](<debtor-tab-transactions.md>) tab **/ [Creditors** - **Transactions**](<creditor-tab-transactions.md>) tab - Click on the **Show transactions** button.&nbsp;

* **Open item link**&nbsp; - [**Totals**](<open-item-export-totals-tab.md>) tab and [**Transactions**](<open-item-export-transactions-ta.md>) tab or a [selected open item account](<open-item-export-account.md>).
* [**Debtors**](<debtors-export-grid.md>) - Export button on the Debtors list screen - You may enter the File name with a \*.txt file type extension.
* [**Creditors**](<creditors-export-grid.md>) - Export button on the Creditors list screen - You may enter the File name with a \*.txt file type extension.
* [**Documents**](<documents-grid-export.md>) - Export button on the Documents list screen - This will launch the "Save As" screen with the default Export.csv file type. &nbsp;

If you click on **Export** buttons on various screens, the "*Save As*" screen will be launched to save a comma separated value text file, with the default "*Export.txt*" file name.&nbsp;

![Image](<lib/file-type-default-app-csv-export-txt-save-as.png>)

Once you have saved the "*Export.txt*" file with your own file name, the Exported file will, by default open in your system's default app associated with csv files.

![Image](<lib/file-type-default-app-csv.png>)

# Open saved Export.txt files in another app (program) for CSV (Comma Separated Values) file type

If you wish to open saved Export.txt files in another app (program) for CSV (Comma Separated Values) file type, than Microsoft Excel, you may use the spreadsheet components of other apps (programs)&nbsp; e.g. [LibreOffice](<https://www.libreoffice.org/> "target=\"\_blank\""), [OpenOffice](<https://www.openoffice.org/> "target=\"\_blank\"") or [UltraOffice](<https://www.compuclever.com/products/ultra-office/> "target=\"\_blank\"") installed on your system.&nbsp;

To do this, select the saved Export.txt file and right click on [LibreOffice Calc](<https://www.libreoffice.org/discover/calc/> "target=\"\_blank\"").

![Image](<lib/file-type-default-app-csv-changed.png>)

To open a Export.txt file in&nbsp; [LibreOffice](<https://www.libreoffice.org/> "target=\"\_blank\""), it may, by default, open the saved Export.txt file in LibreOffice Writer (Word processing) component instead of the Spreadsheet component [LibreOffice Calc](<https://www.libreoffice.org/discover/calc/> "target=\"\_blank\"").&nbsp;

To add [LibreOffice Calc](<https://www.libreoffice.org/discover/calc/> "target=\"\_blank\"") to the **Open with \>** context menu, you need to select **Choose another app** on the context menu.&nbsp;

Go to **Look for another app on this PC**.&nbsp; On the "*Open with...*" screen, browse to where your preferred app is installed. In this example, LibreOffice install folder is selected.&nbsp;

Go to the program folder and select **scalc.exe**.

Click **Open** button and the app will be added to the **Open with ...** context menu of your system's File Explorer.&nbsp;

![Image](<lib/file-type-default-app-csv-change-open-with.png>)

![Image](<lib/file-type-default-app-csv-change-setting.png>)

Opening a saved Export.txt file in [LibreOffice Calc](<https://www.libreoffice.org/discover/calc/> "target=\"\_blank\"") will display the "Text Import - {Export.txt} or \[your file name.txt\].

You may need to check and configure the options for the Comma Separated Values file. &nbsp;

![Image](<lib/text-import-options.png>)

When you click **OK**, the file will be opened in a spreadsheet.
