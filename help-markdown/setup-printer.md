# Printer

***

This option allows you to select the default printer (from any available printers set up / configured on your system).

**To select the printer:**&nbsp;

1. On the **Reports** ribbon or the **Setup** ribbon, select **Printer**. &nbsp;

| ![Image](<lib/admon-tip.png>) | *This "Dialog"-screen may also be launched from the following options:*&nbsp; ***Print setup** tab - click on the [**Configure printers***](<reportman-reports-printer-setup.md>) *button.* ***Page setup** button op **Setup → [Setup documents***](<documents-setup.md>) *(**Setup** ribbon).*&nbsp; |
| --- | --- |


2. The "*Dialog*" screen is automatically displayed:&nbsp;

![Image](<lib/setup-printer.png>)

| ![Image](<lib/admon-note.png>) | ***Configuration file options - Location:*** *User configuration - e.g. - "C:\\Users\\danva\\AppData\\Roaming\\reportman.ini"*&nbsp; *System configuration - e.g. - "C:\\ProgramData\\reportman.ini"* |
| --- | --- |


On the "*Dialog*" window, you may: -&nbsp;

1. Select one of the following printers; Default printer, Reporting printer, Reporting printer 2, Ticket printer, Ticket printer 2, Graphics printer, Character based printer, User printer 1 - 9, Plain text printer or Full plain text printer.
1. **Printer Fonts (Windows GDI Only)** - The Microsoft Windows graphics device interface (GDI).
1. **Text driver only** - You may select a print driver from the list (i.e. "*PLAIN, EPSON, EPSON-MASTER, EPSON-ESCP, EPSON-ESCPQ, IBMPROPRINTER, EPSONTMU210, EPSONTMU210CUT, EPSONTM88II, EPSONTM88IICUT, HP-PCL, VT100*" of "*PLAINFULL*".&nbsp;

| ![Image](<lib/admon-note.png>) | *Oem (Original Equipment Manufacturers) tick to use the drivers from the hardware manufacturers.* |
| --- | --- |


4. **Position adjustment** - Optional - You may enter a value for the left and top of the page. &nbsp; &nbsp; &nbsp;
4. **Operations after print** -&nbsp;
- &nbsp;

  - Cut paper -&nbsp;
  - Open drawer -&nbsp;

6. **Configuration file** - Printer configuration file - Report Manager Designer will use a Printer configuration file, only used in printer selection.&nbsp;

| ![Image](<lib/admon-note.png>) | ***Configuration file options - Location:*** *User configuration - e.g. - "C:\\Users\\danva\\AppData\\Roaming\\reportman.ini"*&nbsp; *System configuration - e.g. - "C:\\ProgramData\\reportman.ini"* |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *This configuration file will be searched as Documents \& Settings\\Local Config\\All Users\\reporman.ini, if not found it will create or use Documents \& Settings\\Local Config\\YourUserName\\reporman.ini* |
| --- | --- |
