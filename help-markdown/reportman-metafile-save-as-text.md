# Reportman output - Save metafile as text file

***

Text files is a flat file, which contains text and has no special formatting such as **bold text**, *italic text*, images, etc. The text files is saved with the \*. txt file extension. It is usually opened with Notepad, or any other text file editor set as default and associated with your operating system.

# What apps or tools can be used to open text files?

Text files, which typically use extensions like \`.txt\`, \`.log\`, \`.csv\`, and others, can be opened using a wide range of applications and tools. Here are some popular options along with their operating system compatibility:

1. **Text Editors**
- &nbsp;

  - Notepad: Pre-installed on Windows.
  - Notepad++: Windows.
  - Sublime Text: Windows, macOS, Linux.
  - Visual Studio Code: Windows, macOS, Linux.
  - Atom: Windows, macOS, Linux.
  - Vim: Windows, macOS, Linux.
  - Emacs: Windows, macOS, Linux.
  - TextEdit: Pre-installed on macOS.
  - Gedit: Pre-installed on many Linux distributions.

2. **Office Suites**
- &nbsp;

  - Microsoft Word: Windows, macOS.
  - LibreOffice Writer: Windows, macOS, Linux.
  - Google Docs: Cross-platform (accessible via any modern web browser).
  - WPS Office: Windows, macOS, Linux, Android, iOS.

3. **Integrated Development Environments (IDEs)**
- &nbsp;

  - Eclipse: Windows, macOS, Linux.
  - IntelliJ IDEA: Windows, macOS, Linux.
  - PyCharm: Windows, macOS, Linux.
  - Visual Studio: Windows, macOS.

4. **Command Line Tools**
- &nbsp;

  - cat: Available on Unix-based systems (Linux, macOS).
  - less: Available on Unix-based systems (Linux, macOS).
  - more: Available on Unix-based systems (Linux, macOS).
  - type: Available on Windows.
  - more: Available on Windows.

5. **Online Tools**
- &nbsp;

  - Google Docs: Cross-platform (accessible via any modern web browser).
  - Microsoft OneDrive: Cross-platform (accessible via any modern web browser).
  - Dropbox Paper: Cross-platform (accessible via any modern web browser).
  - Notepad Online: Cross-platform (accessible via any modern web browser).

6. **Mobile Apps**
- &nbsp;

  - Google Docs: iOS, Android.
  - Microsoft OneDrive: iOS, Android.
  - Dropbox: iOS, Android.
  - WPS Office: iOS, Android.
  - Notepad+: Android.
  - Text Editor: iOS.

7. **Other Tools**
- &nbsp;

  - Adobe Acrobat Reader: Can open text files as well as PDF files. Available for Windows, macOS, Android, iOS.
  - Foxit Reader: Can open text files as well as PDF files. Available for Windows, macOS, Linux.

These tools should cover most of your needs for viewing and editing text files across different operating systems.

# Save metafile options

**To save reports or document layout files as a text file:**

1. On the "*Reportman print preview*" screen, select **Save** (**Ctrl+S**) ![Image](<lib/reportman-preview-icon-save.png>) – This will launch the “*Save metafile as*” screen.&nbsp;
1. Select the directory (folder) where you need to save this file, if necessary.&nbsp;
1. Enter a file name.&nbsp;
1. Select "*Text file (\*.txt)*" file type.
1. Click **Save**.

# Income statement - Text file

If the saved file does not open automatically; or if you need to open it at a later stage; you may open the file in your system's file explorer.

This should automatically open the saved text file of your system's default text file program associated with Text file type (\*.txt) file types (e.g. Notepad, etc).&nbsp;
