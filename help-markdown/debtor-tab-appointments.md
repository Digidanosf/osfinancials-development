# Debtor accounts - Appointments tab

***

In osFinancials5/TurboCASH5, the "*Debtor accounts - Appointments*" tab allows you to view and manage appointments specifically related to a selected debtor (customer/client) account. This tab provides a centralised view of all scheduled debtor appointments that have been added in the **Calendar** (**Default** ribbon).

By accessing the "*Debtor accounts - Appointments*" tab, you will be able to see a comprehensive list of all [debtor appointments/events](<calendar-ivoices-debtor-appointm.md>) that have been scheduled in the **Calendar**. This includes appointments for meetings, discussions, notes, or any other relevant activities associated with your debtor account.

If any Bulk time entries were processed for a selected debtor (customer/client) account these entries will only be listed on the "*Debtor accounts - Appointments*" tab, once you generated quotes on the "**Create invoices for hours**" button.

All scheduled [Debtor appointments](<calendar-ivoices-debtor-appointm.md>) (events) added in the **Calendar** (**Default** ribbon) for the selected debtor (customer / client) account will be listed here.
