# Spreadsheet - Import - General ledger accounts

***

Select the Set of Books which contains the Accounts (Chart of accounts) you need to export.

This option will allows you to import data for Account records from a valid Excel Workbook (\*.xlsx) file format into an existing Set of Books.&nbsp;

To be able to import the data for Account records, you need to [create a valid spreadsheet export file](<accounts-export-excel.md>) in the Excel Workbook (\*.xlsx) file format from a Set of Books, you wish to import your system in the&nbsp; **Export** or **Setup → [Accounts**](<accounts-edit.md>) menu (**Setup** ribbon).&nbsp;

**To import accounts to a spreadsheet:**&nbsp;

1. In the Set of Books in which which you wish to import the Accounts to a spreadsheet file, go to **Setup → [Accounts**](<accounts-edit.md>) (**Setup** ribbon).

![Image](<lib/spreadsheet-export-accounts-option.png>)

2. Click on the **Spreadsheet** icon. A blank "*Data Import Export*" spreadsheet will be displayed with the column headings of the "*Accounts*" table.

![Image](<lib/accounts-spreadsheet-import-blank.png>)

3. Click on the **Open** icon on the "*Data Import Export*" screen.&nbsp;

![Image](<lib/accounts-spreadsheet-import-open.png>)

4. Locate a valid Excel Workbook (\*.xlsx) saved file and click on the **Open** button. All the "*accounts*" (chart of accounts) included in the exported file is populated in the spreadsheet.&nbsp;

![Image](<lib/accounts-spreadsheet-import-opened.png>)

5. To do a import check of the imported file, select (tick) the "*Show data before import*" option and click on the **Import check** icon.&nbsp;

![Image](<lib/accounts-spreadsheet-import-import-check.png>)

| ![Image](<lib/admon-warning.png>) | *&nbsp;Column L - "SRECEIPT" and column N - "SPAYMENT" background is in a **Yellow** colour. This indicates that special attention is needed with the Receipt batches and Payments batches linked to Bank accounts.*&nbsp; *You may proceed with the Import, but need to pay special attention to add and link payments and receipt batches to bank accounts, after the import.* |
| --- | --- |


6. Click on the **Settings** tab.

![Image](<lib/accounts-spreadsheet-import-settings.png>)

7. Click on the **Import** icon. &nbsp;

| ![Image](<lib/admon-note.png>) | ***Optional - Show data before import setting***&nbsp; *The "ImportCheck" screen (if the Import the "Show data before import" option was selected (ticked) on the "Data Import Export" spreadsheet screen) is displayed:* ![Image](<lib/spreasheet-import-accounts-import-check.png>) Click on the **All** or **OK** buttons to start the import process.&nbsp; &nbsp; |
| --- | --- |


![Image](<lib/accounts-spreadsheet-imported.png>)

8. &nbsp;Once the import process is finalised, Column A will be displayed in a **green** background colour.
8. Close the "*Data Import Export*" spreadsheet screen.&nbsp; The "*Accounts*" screen will list the imported chart of accounts.&nbsp;
