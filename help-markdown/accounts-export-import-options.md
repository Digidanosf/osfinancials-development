# Export / Import - General ledger accounts

***

You may export and import general ledger accounts (chart of accounts).&nbsp;

This feature may be used to import ledger accounts (chart of accounts) into:

* Into the same Set of Books.
* Into a different Set of Books.
* Into a new Set of Books - Created from scratch - [Create your own Set of Books (Advanced)](<create-books-advanced.md>) option for Firebird database types.
* Into a new Set of Books - Created from scratch - [Create your own Set of Books (Advanced)](<mssql-sql-database-create.md>) option for Microsoft SQL sever (MSSQL) database types.

| ![Image](<lib/admon-important.png>) | *These accounts (Chart of Accounts) are maintained in **Setup → [Accounts***](<accounts-edit.md>) *(**Setup** ribbon).*&nbsp; *Reports for the accounts (Chart of Accounts) are printed in **Reports → Ledger → Listing** - [**Chart of Accounts***](<report-ledger-chart-of-accounts.md>) *(**Reports** ribbon).* *Before using this feature, it is recommended to print the report.*&nbsp; |
| --- | --- |


&nbsp;

**Export / Import options on Setup ribbon**&nbsp;

To be able to import the data for Account records, you need to created a "*Tab delimited*" export file in the [**Export**](<accounts-export-tab-delimited.md>) (**Setup** ribbon).&nbsp;

It is recommended that the accounts (chart of accounts) is exported in the "*Tab delimited*" (\*.txt) file format. This file format may be used to import the "*Tab delimited*" (\*.txt) file format in the [**Import**](<accounts-import-tab-delimited.md>) (**Setup** ribbon).

# Spreadsheet export / import&nbsp;

osFinancials5/TurboCASH5 supports the "*Excel Workbook (\*.xlsx)* file type Spreadsheet import and export feature#8202;*.* &nbsp;

* [Spreadsheet export](<accounts-export-excel.md>) - You may export ledger accounts and save it as a "*Excel Workbook (\*.xlsx) file type.*
* [Spreadsheet import](<accounts-import-excel.md>) - You may import ledger accounts from a saved "*Excel Workbook (\*.xlsx) file type.*
