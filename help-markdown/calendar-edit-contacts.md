# Edit contacts - Calendar

***

Once a contact has been added, you may then edit it, if necessary.&nbsp;

**To edit a contact:**&nbsp;

1. To edit an existing contact, simply select the contact, and right-click on the selected contact. On the context menu select the **Edit contact...** option.&nbsp;

| ![Image](<lib/admon-note.png>) | *The Name of the Contact - \*Contact will be displayed in the title bar.*&nbsp; |
| --- | --- |


2. Edit any of the fields you need to edit on the Main tab, Telephone tab or the Notes tab.&nbsp;
2. Click on the **OK** button.&nbsp;
