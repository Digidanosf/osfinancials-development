# Comments

Comments are additional information which can be entered when processing documents. This additional information will be printed on layout files for sales documents (e.g. Invoices, Credit note and Quote) and purchase documents (e.g. Purchases, Supplier returns and Orders).&nbsp;

Information entered in comments can be used to find and track specific invoices, etc.&nbsp;

| ![Image](<lib/admon-note.png>) | *By default, [comments*](<document-entry-insert-transactio.md>) *is inserted below a selected item. The option "[**Append notes below item***](<doc-setting-append-items-below.md>)*" (**Setup → Documents setup** (**Setup** ribbon)) is added to set the preference if [comments*](<document-entry-insert-transactio.md>) *should be added before, or below (default option), a selected item when processing documents and printing comments on layout files.*&nbsp; *The default "**Append notes below item**" setting is selected (ticked). It will display on document lines and document layout files as follows:*&nbsp; *Item*&nbsp; *Remark*&nbsp; *Comment*&nbsp; *If the "**Append notes below item**" setting is not selected. It will display on document lines and document layout files as follows:*&nbsp; *Comment*&nbsp; *Item*&nbsp; *Remark*&nbsp; |
| --- | --- |


**To insert a comment in a document:**&nbsp;

1. Select the row or line directly underneath the transaction for which you wish to insert a comment.&nbsp;

| ![Image](<lib/admon-note.png>) | *If you have any other transactions below the selected line or row, which you wish to keep, it is recommended that you click on the **\^N:Insert** icon, or press the **Ctrl** + **N** keys to insert a line or row before you change the selected line to a comment.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-warning.png>) | *You should note that if any information or a transaction is displayed in a selected line, that information would be lost for the selected line or row.*&nbsp; |
| --- | --- |


2. Click on the **F9:Comment** icon. \
![Image](<lib/documents-comments-enter-edit.png>)
2. Enter the comment for the stock item, or transaction, and click on the **OK** button. The comment will be transferred into the selected line or row and will display as follows on the Document entry form:&nbsp;

![Image](<lib/documents-comments-entered-view.png>)

| ![Image](<lib/admon-tip.png>) | *You may double-click on a selected comment to edit an existing comment.* |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *You may also click on the **+** (before stock item code) to add a **remark** (additional information) for a stock item.*&nbsp; *This **remark** will also be printed before the comment (if added) on document layout files.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-note.png>) | [***Search** (**Investigator**)*](<search-investigator.md>)&nbsp; ***Comments** entered in document lines are searchable.* ***Remarks** is not searchable.* |
| --- | --- |


4. Press the **Enter** key on your keyboard to proceed to the next line or row.&nbsp;

| ![Image](<lib/admon-note.png>) | *Although the full comment is not displayed on the comment line or row, it will be displayed as entered on the printed document (transaction section). For example;*&nbsp; ![Image](<lib/documents-comments-entered-view-layout-file.png>) |
| --- | --- |


5. You may proceed to enter any further transactions for the debtor (customer / client) or creditor (supplier / vendor). If you are finished, click on the **Esc** key on your keyboard to exit (close) the Stock item lookup screen, and to return to the Document list screen.

# Serial number tracking&nbsp;

Should you for example, enter a serial number in the comment of a document, you may locate it on the **Search** - [**Investigator**](<search-investigator.md>). To do this:&nbsp;

1. On the **Default** ribbon, select **Search** (**Ctrl + F3**). The "[*Investigator*](<search-investigator.md>)*"* screen will be displayed.&nbsp;
1. Enter the serial number.&nbsp;
1. Click on the **Search now** button.&nbsp;
1. The sales document (e.g. Invoices, Credit note or Quote) and purchase documents (e.g. Purchases, Supplier returns or Orders) containing that serial number will be listed. To display the selected document, simply double-click on it.&nbsp;
