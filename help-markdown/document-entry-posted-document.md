# Document entry form - Posted documents

***

Once a document (Invoices, Credit notes, Purchases or Supplier returns) have been updated (posted) to the ledger, you cannot edit or change these documents.

However, you may still view the documents as they were originally processed in the Document entry screen before these documents were updated (posted) to the ledger.&nbsp;

**To view the posted documents in the Document entry screen:**

1. Select the document type (Invoices, Credit notes, Purchases, or Supplier returns).
1. Select "Posted" on the Document grid screen. Only the posted documents will be listed.
1. Select the Posted document on the list.
1. Click on the **Edit** button.&nbsp;

5. To close this screen, click on the **Cancel** button.

| ![Image](<lib/admon-note.png>) | *The layout Document entry screen displays all fields as on the original Document entry screen before it was updated (posted) to the ledger.* *The transaction section lists all the stock item codes and/or comments.
