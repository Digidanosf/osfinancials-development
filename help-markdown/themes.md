# Themes

***

By default, osFinancials5 is installed in the "*Classic*" theme (default theme) and TurboCASH5 is installed in the "*Material-navy*" theme.&nbsp;

If you prefer a different look, you can choose the "*Blue, Glass, Silver or Soft-blue" or "VS2010"* theme in osFinancials5/TurboCASH5. These themes feature a navy-blue colour scheme and offer a collection of icons similar to those in the "*Material*" theme. In addition to the Material theme's black icons, you may select other coloured variants such as "*Material-blue, Material-green, Material-maroon, Material-navy"* or *"Material-pink*".&nbsp;

Selecting a different theme from the [**Theme**](<themes.md>) menu allows you to customize the interface to your preferences.

## Select a theme

**To select a theme:**&nbsp;

1. On the **Setup** ribbon, select **Theme** and select the theme of your choice.

osFinancials is by default installed in the "*Classic*" theme.

![Image](<lib/theme-classic-default.png>)

The "*Material*" theme is selected and used for most screenshots in this documentation.

![Image](<lib/osfinancials5-themes-material.png>)

2. Select the theme you like the best. The program will display the selected matching icon set.
