# Bank import plugin

***

Using the [**Bank import plugin**](<plugin-bank-import.md>), you may import a valid bank statement file (downloaded from your online banking) saved on your system into a Bank account (Cash-book).

This [**Bank import plugin**](<plugin-bank-import.md>) will save you time, since you do not need to physically enter the transactions manually in the Payments journal and Receipts journal.

This plug-in uses Bank references to automatically allocate or link payments and deposits, receipts, other debits. credits, etc. to the correct accounts.&nbsp;

It further supports the bank accounts of your debtors (customers / clients) to use the automatic receipts for debit orders, etc.&nbsp;

It further supports the bank accounts of your creditors (suppliers) to use the automatic payments to manage the payments to suppliers, etc.
