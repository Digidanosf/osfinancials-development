# Documents - Context menu

***

Right-click to display the following context menu:

![Image](<lib/invoices-context-menu-documents.png>)

The options are as follows:

1. [**Salesperson**](<document-grid-change-salesperson.md>) – Change the Salesperson (as added in **Setup → Setup →&nbsp; Salesperson** or **Setup → Groups** (**Salespersons**)).
1. [**Set reporting group 1**](<document-grid-change-groups.md>) – Change the reporting group of a document, if Reporting group 1 was set for Documents (as added in the **Setup → Groups** (**Document groups 1**)).

| ![Image](<lib/admon-tip.png>) | *The colours of the rows (list of documents) will be changed, if set in Reporting group 1.*&nbsp; &nbsp; |
| --- | --- |


3. [**Set reporting group 2**](<document-grid-change-groups.md>) – Change the reporting group of a document, if Reporting group 2 was set for Documents (as added in **Setup → Groups** (**Document group 2**)).
3. [**Documents**](<document-grid-menu-documents.md>) - The "*Documents*" context menu option on the Document list screen changes for the document type selected. Depending on whether a posted document; or an unposted document is selected; some context menu options may not be available, or additional specific context menu options may be added. Some examples are:
- &nbsp;

  - [Change account](<docs-grid-change-account.md>) - is not available for posted documents.&nbsp;
  - [Adjustment](<documents-adjustment.md>) - is not available for unposted documents. Adjustments not implemented in osFinancials5/TurboCASH5.

5. **Print**

   1. [**Print barcodes**](<docs-grid-print-barcodes.md>) – This will print only the barcodes of stock item codes included on selected document(s).
   1. [**Print labels**](<docs-grid-print-labels.md>) – This will only print the name (account description) of a debtor customer / client on a selected sales document. In the case of a purchase document, it will print the name (account description) of the creditor (supplier / vendor).

5. **Others**

   1. [**Import**](<doc-grid-menu-others-import.md>) – This option allows you to import an exported document from a valid existing export.txf file. "*TurboCASH Data Exchange files*" are exported (created) in **Tools → Export** (**Documents**).
   1. [**Totals**](<doc-grid-menu-others-totals.md>) – This will list a summary of the Exclusive, Inclusive, Open amounts and the records on an Information screen.

| ![Image](<lib/admon-tip.png>) | *The results may be selected and copied to any other text editor or other external program.*&nbsp; |
| --- | --- |


1. &nbsp;
   3. [**Show details**](<doc-grid-menu-others-show-detail.md>) – This will display a mouse-over screen displaying the details (i.e. account description,&nbsp; document number, stock description, quantity and amount (inclusive of tax) of the selected document. &nbsp;
   3. [**Open item link**](<doc-grid-menu-others-open-item.md>) – This will launch the Open item link screen displaying the transactions of the debtor (customer / client) or creditor (supplier / vendor) account on a selected document.&nbsp;

| ![Image](<lib/admon-note.png>) | *Only available for Invoices / Purchases document type.* *This option is not available for other document types (Quotes, Credit notes, Orders and Supplier return documents).* |
