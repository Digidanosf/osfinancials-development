# Report designer

| ![Image](<lib/editreport1.png>) | ***Reportman** - [Manual*](<https://www.osfinancials.org/en/plugin-manuals/550-reportman> "target=\"\_blank\"") *- **Licence** : Free.*&nbsp; *Accessible from the [**Report designer***](<reportman.md>) *icon on the **Setup** ribbon* ***Translations** : [Translations outstanding*](<translations-report-designer.md>)&nbsp; |
| --- | --- |


Unlock the power of report generation and customization with our Report designer plugin, provided free of charge. Accessible from the [**Report designer**](<reportman.md>) icon on the **Setup** ribbon, this plugin empowers users to design and print reports and document layout files effortlessly. All reports and document layout files included in the osFinancials5/TurboCASH5 installation are designed with the powerful **Report designer** plugin.

**Compatibility with Database Types:**

The Report designer plugin seamlessly integrates with osFinancials5/TurboCASH5, allowing users to edit and write reports and document layout files across different database types. Whether your Set of Books in osFinancials5/TurboCASH5 relies on Firebird or MSSQL database types, the Reportman plugin ensures smooth functionality and compatibility.

**Key Features:**

* &nbsp;
  * Powerful report generator and designer
  * Customize existing reports or create your own
  * Design document layout files for various sales and purchase documents
  * Includes default layout files and reports in the osFinancials5/TurboCASH5 installation

**Best Practices:**

* &nbsp;
  * Always make a copy of a report or layout file before editing. You can always fall back on the original report or layout if things go wrong.
  * Rename your layouts or reports to prevent overwriting during reinstallation or during installation of osFinancials5/TurboCASH5 updates.
  * Add your customised reports to the **User reports** menu

**Documentation and Resources:**

* &nbsp;
  * **osFinancials Help and Online documentation:** Comprehensive detailed information provided in the "*Working with Reportman*" section of the osFinancials5/TurboCASH5 Help file and on the osFinancials website in [English](<https://www.osfinancials.org/htmlhelp\_eng/welcome.html> "target=\"\_blank\"") and [Afrikaans](<https://www.osfinancials.org/htmlhelp\_af/welcome.html> "target=\"\_blank\"").
  * **Official offline documentation:** Navigate to the "...\\plug\_ins\\reportman\\doc" folder within your osFinancials5/TurboCASH5 installation directory. Open the index.html file located at ".../plug\_ins/reportman/doc/index.html" in your web browser to access this resource.
  * **Official online documentation:** For additional resources and information regarding the Report Manager, refer to the official online documentation available at [https://reportman.sourceforge.io/](<https://reportman.sourceforge.io/> "target=\"\_blank\""). This online documentation offers comprehensive guidance and support for maximizing the functionality of the Report Manager.

Harness the capabilities of the Reportman Plugin to create tailored reports and document layouts while maintaining flexibility and control over your printing needs.

# SQLBrowser

| ![Image](<lib/B93.png>) | ***SQLBrowser** - [Manual*](<https://www.osfinancials.org/en/plugin-manuals/507-sqlbrowser> "target=\"\_blank\"") *- [Shop*](<https://www.osfinancials.org/en/webshop/other/sqlbrowser> "target=\"\_blank\"") *- **Licence** : Once-off -* ***Documentation** : [Help documentation*](<plugin-sql-browser.md>) &nbsp; ***Translations** : [Outstanding*](<translate-plugins-drilldown.md>) &nbsp; |
| --- | --- |


The SQL Browser plugin in osFinancials empowers users with a robust tool for interacting with their data using Structured Query Language (SQL). Tailored for users familiar with SQL, this feature enables the execution of custom queries, effortless grouping, filtering, and sorting of results.&nbsp;

You can save their SQL Script files for future use and export data to various formats including XML (Extensible Markup Language), HTML (Hypertext Markup Language), Microsoft Excel, and Text File formats. Additionally, users can seamlessly export data to "*JSON*"-files and create a new table from a "*JSON*"-file, expanding the flexibility and capabilities of data management within osFinancials5/TurboCASH5.

**Compatibility with Database Types**

The SQL Browser plugin seamlessly integrates with any Set of Books utilizing either Firebird or MSSQL database types. Whether your organisation relies on Firebird or MSSQL for database management, the SQL Browser provides a unified platform for accessing and manipulating data.

This flexibility allows users to leverage the SQL Browser's powerful features across different database environments, ensuring consistent functionality and ease of use regardless of the underlying database technology.
