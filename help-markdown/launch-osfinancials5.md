# Launch osFinancials5 for the first time

In osFinancials5, the interface language will be displayed based on the language associated with the Set of Books for the country. Each Set of Books can have a specific language linked to it, and when you open that Set of Books in osFinancials5, the interface language will be displayed in the language associated with that particular Set of Books by default.

| ![Image](<lib/admon-note.png>) | *&nbsp;Default languages have already been pre-set for each Set of Books. (**Setup → [Access control***](<set-default-language.md>) *on the **Setup** ribbon).*&nbsp; *When opening a Set of Books, it will automatically load the language file. All screens, reports and layout files are translatable via the language files.* |
| --- | --- |


| ![Image](<lib/admon-warning.png>) | *Some languages may not display correctly if your system locale in the **Regional and Language Settings** option in the **Control Panel** of your system may not support the language and system locale (unicode).* |
| --- | --- |


| ![Image](<lib/admon-important.png>) | *English is the default (driver language file – used as the basis for translation, and to [Customise language*](<customise-language.md>) *files). For example, the American and the English files contain specific label, text, currency and tax terminology for each country.* *Should you select the incorrect language at this stage, you may change the language in [**Switch language***](<switch-language.md>) *on the **Start** ribbon).* |
| --- | --- |


Once osFinancials5 is launched, the "*Set of Books"* screen is displayed. To display the interface of osFinancials5 in the default "*English*" language, select the "*HANDYMAN*" Set of Books and click on the [**Open**](<open-set-of-books.md>) button. &nbsp;

| ![Image](<lib/admon-important.png>) | *When multiple copies of* **TurboCASH 5‑4** *are started from the* **same installation directory** *(for example: TCASH5-4), the* **TurboCASH icons may not appear in the Windows taskbar**#8202;*.* *This can also occur when older TurboCASH versions (such as 5.2 or 5.3) are installed in the same folder structure.* *If this happens, you can still switch between open TurboCASH windows by pressing* **ALT + TAB** *and selecting the window you want to activate.* *When TurboCASH 5‑4 is launched from* **different installation directories** *(for example: TCASH5-4, TCASH5-4-AF, TCASH5-4-HR), the icons are displayed correctly in the Windows taskbar.* *Each installation folder is treated as a separate application, which prevents the icon display issue.* |
