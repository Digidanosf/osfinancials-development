# Print the Chart of accounts&nbsp;

After translations added, the report options and printed reports will be displayed in the Language&nbsp; set for the logged in user.

**To check the translated Chart of accounts into the translated language:**

1. Click on the **Open** icon (**Start** ribbon).
1. Select the Set of Books and click on the **Open** button.
1. On the Login screen, enter the username and password for the different language.
1. Click on the **OK** button. The osFinancials5/TurboCASH5 interface and Set of Books will open in the user's language (as per this example in Afrikaans.
1. Print the Chart of accounts and check if it is correct.
1. Print the Account Group 1 / 2 report and check that the groups are correct.&nbsp;
