# Drop-down lists auto-expand according to field length

Replicated in all translations if longer translations and data is entered.

Drop-down lists auto-expand according to field length like it is implemented in osFinancials5.1.0.249. and TurboCASH5-4.

It is implemented in the following drop-down lists :

* Debtors/Creditors&nbsp;

  * Country&nbsp;
  * Language
  * Delivery Address tab Layout file 1, Layout file 2 and Layout file 3,

* Documents -&nbsp;

  * Layout file 1, Layout file 2, Layout file 3 and Layout file 4 (sidebar on the grid)
  * Salesperson
  * Country&nbsp;
  * Contact

* Stock&nbsp;

  * Sidebar (select reports)
  * Type (stock type)&nbsp;
