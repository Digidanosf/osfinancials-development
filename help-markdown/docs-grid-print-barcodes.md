# Context menu - Print barcodes

***

This option allows you to print barcodes for the stock items included on sales documents (i.e. Invoices, Credit notes or Quotes) or purchase documents (i.e. Purchases, Supplier returns or Orders).

| ![Image](<lib/admon-tip.png>) | *You may also print the Barcode or the Yagoda barcode from the **Reports → Stock → [Print labels***](<report-print-stock-barcodes.md>) *menu on the **Reports** ribbon.* |
| --- | --- |


**To print barcodes for selected stock items:**

1. On **Documents** (Documents list screen) **Default** ribbon), select the Document type, if the correct document type is not selected.
1. Select the Document(s), right-click and select **Print → Barcodes** on the context menu. This will print the labels of all the stock items included on a selected document.&nbsp;

| ![Image](<lib/admon-tip.png>) | *If you need to select multiple documents, press the **Ctrl** or the **Shift** key and click on each document, for which you need to print barcodes.* |
| --- | --- |


An example of a printed barcodes for a selected document, is as follows:

![Image](<lib/documents-context-menu-print-barcodes.png>)

| ![Image](<lib/admon-tip.png>) | *Barcodes will be printed for each stock item included in a selected document.* |
| --- | --- |


It will display / print the following for each stock item included in a document:

1. Stock code
1. Stock item description
1. Barcode
1. Selling price (exclusive of Tax (VAT/GST/Sales tax).

| ![Image](<lib/admon-note.png>) | *Labels will be printed for each stock item on a selected document.*&nbsp; *If more than one document is selected, labels will be printed for each of the selected documents.*&nbsp; |
| --- | --- |
