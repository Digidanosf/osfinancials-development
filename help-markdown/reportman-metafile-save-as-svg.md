# Reportman output - Save metafile as svg

***

The "*Scalable Vector Graphics file*" type **SVG** file type extension is a is an [*Extensible Markup Language*](<https://en.wikipedia.org/wiki/Extensible\_Markup\_Language> "target=\"\_blank\"") (XML)-based [*vector image format*](<https://en.wikipedia.org/wiki/Vector\_image\_format> "target=\"\_blank\"") for [*two-dimensional*](<https://en.wikipedia.org/wiki/Two-dimensional> "target=\"\_blank\"") graphics with support for interactivity, 

SVG images and their related behaviours are defined in [*XML*](<https://developer.mozilla.org/en-US/docs/Web/XML> "target=\"\_blank\"") text files, which means they can be searched, indexed, scripted, and compressed. &nbsp;

There are a few SVG editors available to edit SVG files.&nbsp;

# What apps or tools can be used to open svg files?

Some of the tools and apps that can be used to open SVG files, along with their cross-compatibility across different operating systems is listed as follows:

1. **Web Browsers**
- &nbsp;

  - Google Chrome: Windows, macOS, Linux, Android, iOS
  - Mozilla Firefox: Windows, macOS, Linux, Android, iOS
  - Microsoft Edge: Windows, macOS, Android, iOS
  - Safari: macOS, iOS

2. **Image Viewers and Editors**
- &nbsp;

  - Inkscape: Windows, macOS, Linux
  - Adobe Illustrator: Windows, macOS
  - GIMP: Windows, macOS, Linux
  - CorelDRAW: Windows, macOS

3. **Office Suites**
- &nbsp;

  - Microsoft Office (Word, PowerPoint, Excel): Windows, macOS
  - LibreOffice (Writer, Impress, Draw): Windows, macOS, Linux

4. **Online Tools**
- &nbsp;

  - SVG Viewer Online: Cross-platform (accessible via any modern web browser)
  - Cloud Services (Google Drive, Dropbox): Cross-platform (accessible via any modern web browser)

5. **Mobile Apps**
- &nbsp;

  - Adobe Illustrator Draw: iOS, Android
  - Inkscape Viewer\*\*: Android

6. **Command Line Tools**
- &nbsp;

  - ImageMagick: Windows, macOS, Linux

7. **Text Editors**
- &nbsp;

  - Visual Studio Code: Windows, macOS, Linux
  - Sublime Text: Windows, macOS, Linux

8. **Other Tools**
- &nbsp;

  - SVG-Edit: Cross-platform (accessible via any modern web browser)

This list should help you find the right tool for viewing and editing SVG files regardless of your operating system.

# Save metafile options

**To save reports or document layout files as a Scalable Vector Graphics file:**

1. On the "*Reportman print preview*" screen, select **Save** (**Ctrl+S**) ![Image](<lib/reportman-preview-icon-save.png>) – This will launch the “*Save metafile as*” screen.&nbsp;
1. Select the directory (folder) where you need to save this file, if necessary.&nbsp;
1. Enter a file name.&nbsp;
1. Select the "*Scalable Vector Graphics (\*.svg)*" file type.
1. Click **Save**. &nbsp;

# Income statement - SVG file

An example of the "*Income statement*" saved as a SVG file, opened in "*Microsoft Edge*", is as follows:

![Image](<lib/reportman-save-meta-file-as-svg.png>)

Since the Namespace links, by default renders an error as displayed in the "*Microsoft Edge*" Web browser, as per this example, as well as in other Web browsers ,such as Google Chrome, etc.), the report headers seem to be displayed correctly, but the data in the report details does not display.

Using a SVG editor such as [**Inkscape**](<https://inkscape.org/> "target=\"\_blank\"") is professional quality vector graphics software which runs on Windows desktop, Linux, Mac OS X and most computers, the data displays correctly.&nbsp;

![Image](<lib/reportman-save-meta-file-as-svg-inkscape.png>)

&nbsp;

If a report is saved as a "*Scalable Vector Graphics file*" file format, it will save the file in separate pages. An example of a one page saved "*Scalable Vector Graphics file*" will also export numbering bullets (numbering) as objects for each page.&nbsp;

The vertical lines, separating the columns, in the "*detail*" report used to print the Trial balance, Income statement, Balance sheet, etc. is exported as bullets (numbering) objects.&nbsp;

![Image](<lib/reportman-save-meta-file-as-svg-folder.png>)
