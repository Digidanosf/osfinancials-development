# Document print confirmation

***

By default, when a document is created or edited, it's automatically set to be printed. Upon saving the document (by clicking the **OK** button in the Document entry screen), a print confirmation message will appear. By default, documents will be printed when a document is created or edited (when clicking on the **OK** button of the Document entry screen). A print confirmation message will be displayed:

*&nbsp;"Do you wish to print this Invoice now?"* \
*For instance, if the document type is an "Invoice," the message might read: "Do you wish to print this Invoice now?" The same applies to other document types like credit notes, quotes, purchases, supplier returns, or orders.*

If you click on the **Yes** button, the document will be printed immediately. If you click on the **No** button, you have the option to print the document at a later stage.

This will not print any documents when processing documents.* *To disable the print confirmation mode, select (place a tick) in the "No print confirmation" field. This can be done through the **Setup → [Stock information***](<stock-information.md>) *menu under the **Setup** ribbon.* *This means that documents will not be automatically printed when created or edited. There won't be any print confirmation message displayed.* |

This can be done through the **Setup → [Documents setup***](<documents-setup.md>) *menu under the **Setup** ribbon. Enabling this option will automatically post the documents when you print them, eliminating the need for a separate posting step.* |

By utilizing these settings, you can customize your document creation and printing process to match your preferred workflow, ensuring efficiency and accuracy in your business operations.
