# Context menu - Convert quote to one unprocessed invoice per client

***

Convert quote to one unprocessed invoice per client.&nbsp;

*The Quote number and date of the selected quote which is converted will be added as a comment.*

*Each time this option is selected for a selected Quote a new line with the Stock item and Quote number and date will be added.*&nbsp;

**To convert quote to one unprocessed invoice per client:**

1. Select the "*Quotes*" document type.
1. Select the Quote to convert quote to one unprocessed invoice per client.
1. Right-click and select **Documents →** **Convert quote to one unprocessed invoice per client** option on the context menu.&nbsp;

![Image](<lib/documents-context-menu-new-1.png>)

4. A new Invoice for the selected debtor (client) will be generated.
4. Select the "*Invoices*" document type and select the Invoice.

![Image](<lib/documents-context-menu-new.png>)
