# Report designer -  Data access configuration

***

The Database connections and datasets is the heart of the Report manager. Parameters needs to be passed from osFinancials5/TurboCASH5 to Reportman for the various reports.&nbsp;

For documents, the ***TCVAR\_DOCID*** parameter is used. This parameter contains the document number in the Dochead table.&nbsp;

![Image](<lib/report-designer-parameter-definition.png>)

For example, in the case of Stock reports, you may need to use the SQL parameter for the data in **Stock items**&nbsp; ([**Default**](<default-ribbon.md>) ribbon) to activate your search action. This will enable you to define and add various criteria to be used in a report.&nbsp;

To access this you may click on the ![Image](<lib/reportman-design-icon-9-connections.png>) icon on icon toolbar or go to the **Report → Data access configuration** menu.&nbsp;

![Image](<lib/report-designer-report-menu.png>)

![Image](<lib/reportman-database-connections-datasets.png>)
