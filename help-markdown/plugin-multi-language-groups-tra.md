# Translate Multi-language groups

***

The names or descriptions for Groups, as entered / edited in the **Setup → Groups** menu for all the Groups tabs, are by default displayed on screens, lookups, documents, reports, etc.

| ![Image](<lib/admon-note.png>) | *To use multi-language Groups, Language Groups need to be created / added in **Setup → Groups** - **Languages** tab.* |
| --- | --- |


The following groups need to be translated, where necessary:

* Account group 1
* Account group 2
* Debtor group 1
* Debtor group 2
* Creditor group 1
* Creditor group 2
* Stock group 1
* Stock group 2
* Document group 1
* Document group 2
* Payment types&nbsp;
* Salesperson&nbsp;
* Projects&nbsp;
* Cost centre 1
* Cost centre 2&nbsp;

**To translate groups:**

1. Go to **Setup → Groups** menu.
1. Click on the **Account Group 1** tab.&nbsp;

 ![Image](<lib/multi-language-groups-accounts-languages.png>)

3. Select a Group (e.g. Income, etc.) on the list and right-click to select the Languages on the context menu.
3. Enter the Name and Description for the Account group 1 item.
3. Click on the **OK** button.
3. Repeat the process to add all required items on Account group 1.
3. Repeat the process for all other Groups (e.g. Account group 2, etc.) where required.
3. Click on the **OK** button to save your changes and to close the "*Groups*" screen.&nbsp;

**After translation**&nbsp;

After translations added, the group names will be displayed in the Language set for the logged in user.

## Logged in as English user

![Image](<lib/multi-language-groups-english.png>)

To print the groups, click on the **Print** button.

![Image](<lib/multi-language-groups-english-print.png>)

## Logged in as Afrikaans user

&nbsp;

![Image](<lib/multi-language-groups-afrikaans.png>)

To print a list of the groups, the groups, click on the **Print** button. The translated groups will be listed.
