# Debtor - Context menu - Reports

***

The "*Reports*" context menu option allows you to print postal or delivery address labels for a selected debtor (customer / client) account.&nbsp;

**To print labels for a selected debtor (customer / client) account:**

1. On the "*Debtors*" list screen, select a debtor (customer / client) account.
1. Right-click and select "*Reports → Selected debtor label*" option or the&nbsp; select "*Reports → Selected debtor delivery label*" option on the context menu. This will print the postal address label or the delivery address label of the selected debtor (customer / client) account.&nbsp;

| ![Image](<lib/admon-note.png>) | *You will only be allowed to print the postal address label or delivery address label for one selected debtor (customer / client) account at a time.* |
