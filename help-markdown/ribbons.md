# Ribbons

***

The functions and features and utilities are grouped in five Ribbon tabs.&nbsp;

![Image](<lib/ribbon-start.png>)

| ![Image](<lib/admon-tip.png>) | ***Show / Hide Ribbon options*** *You may need some extra space when working with, Documents, Stock, Debtors, Creditors, Ledger analyser, etc.*&nbsp; *To Auto hide the ribbons options, double-click on any of the ribbon tabs. This will hide the options on the ribbon and only display the ribbon tabs and auto-resize (larger) the screens in osFinancials5/TurboCASH5. You may then click on a ribbon tab to select and access the options. Once you have selected an option, the ribbon options will auto hide and only display the ribbon tabs.*&nbsp; *To Show the ribbon options and resize the screens (smaller), you may double-click on a ribbon tab.*&nbsp; |

These ribbon tabs are:

1. [**Start**](<start-ribbon.md>) - This ribbon tab is used to manage your sets of books, backups, registration and language.
1. [**Default**](<default-ribbon.md>) -&nbsp;
1) &nbsp;

   1) **Records** - Debtors, creditors, contacts, stock items, appointments, tasks, etc.&nbsp;
   1) **Processing of transactions** in sales documents (i.e. invoices, credit notes and quotes) and purchase documents (i.e. purchases, supplier returns and orders), batches (journals), managing open item accounts (adjust payment dates and/or process receipts / remittance advice for payments and/or refunds, etc.
   1) **Periodic processing** – Open item, Reconciliation, Repeating transactions Repeating invoices.
   1) **Search** – Search for data in the central search and print transactions and documents, etc.

3. [**Reports**](<reports-ribbon.md>) - Print reports, User reports, Spreadsheet reports, Ledger analyser, T-account viewer, Age analysis, Printer setup, etc. &nbsp;
3. [**Setup**](<setup-ribbon.md>) - This ribbon tab is used to access the **Setup**, **Tools**, **Global processes**, **Themes** and **Plugins**.
3. [**User tab**](<user-ribbon-tab.md>) - This ribbon tab is a custom (quick menu). It is used to manage user navigation in a Set of Books. You may select (tick) the "*Auto-learn*" feature to add the last accessed options in up to six (6) groups. &nbsp;
