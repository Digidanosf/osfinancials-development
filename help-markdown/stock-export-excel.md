# Spreadsheet - Export - Stock items

***

This option will allow you to Export your stock code file into an Excel spreadsheet. You may then add new stock items, edit existing stock items and import it back into your stock code file.

**To export and stock items to a Excel Workbook (xlsx) file:**&nbsp;

1. Open the Set of Books from which, you wish to create a spreadsheet export file.
1. On the **Setup** ribbon, select **Import**. The "*Import Data into osFinanacials"* screen is displayed:&nbsp;

![Image](<lib/stock-spreadsheet-export-import-setup-ribbon.png>)&nbsp;

3. Select **Stock** as the "*Data type to import*".
3. Click on the **Excel import export** button. The Excel spreadsheet will be displayed with the column headings for the Stock items (without any data).
3. Click on the **Export** icon. The data from your stock code file will be imported (populated) into the spreadsheet.&nbsp;

![Image](<lib/stock-excel-export-01.png>)

6. You may add new stock items, edit the stock items, change the quantities, selling prices, whatever field, is necessary.&nbsp;
6. Once you are done, check the following options:&nbsp;
1) &nbsp;

   1) Import check&nbsp;
   1) Profile &nbsp;

8. Click on the **Save as...** icon (**View** ribbon) to save the data in the "*Data Import Export*" spreadsheet as an "*Excel Workbook (\*.xlsx)*" file.

![Image](<lib/stock-excel-export-02.png>)

9. Select the folder and enter a name for the file.
9. Click on the **Save** button.&nbsp;
9. Click on the **Close** icon in the titlebar of the "*Data Import Export*" screen.&nbsp;
