# Context menu - Others - Show details

***

By default, the details of each document is listed on the Documents list screen in columns. You may select to show the basic details of the items, quantities, remarks and comments added to documents. When a document is selected or the scrolled on the list, the details will be displayed.&nbsp; &nbsp;

| ![Image](<lib/admon-note.png>) | *The detail will be displayed regardless of whether you select other document types.*&nbsp; *The "Show detail" will only be disabled when you:*&nbsp; *Select the **Other → Hide** option context menu option on the Document entry list screen.*&nbsp; *Select the **Tools → [Close active forms***](<close-active-forms.md>) *(**Setup** ribbon) option.* *Close and reopen the Set of Books.* |
| --- | --- |


**To show or hide the details on a selected document:**&nbsp;

1. On the **Default** ribbon, select **Documents** (Documents list screen), select the Document type, if the correct document type is not selected.
1. Right-click anywhere on the Documents list screen or on a selected document. On the context menu, select the **Others → Show details** option.&nbsp;

![Image](<lib/documents-context-menu-show-details.png>)

3. When a document is selected, it will display the details of the document.&nbsp;

![Image](<lib/docs-context-menu-show-details-02.png>)

&nbsp;

The details will display the following:

* Name of debtor (customer / client) or creditor (supplier / vendor)
* Document number
* Stock item description
* Quantity
* Price (Inclusive of tax)
* Remarks (if entered)
* Comments (if entered)

**To turn the Show details off:**

Right-click and select the **Others → Hide** option on the context menu.&nbsp;
