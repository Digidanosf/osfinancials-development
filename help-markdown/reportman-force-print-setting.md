# Reportman - Force print setting

***

The Page footer may print on the first page only. It may not print on the last page or on consecutive pages, if it is a multi-page report.&nbsp;

If this is the case, the default **Force print** setting for the Page footer is set to **False**.&nbsp;

You need to set the **Force** print setting to **True**.&nbsp;
