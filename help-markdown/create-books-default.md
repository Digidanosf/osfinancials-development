# Let osFinancials5/TurboCASH5 help you to create a Set of Books

***

osFinancials5/TurboCASH5 will guide you through the steps to create a new Set of Books with the "*Creation wizard*" in a matter of minutes.&nbsp;

It is a good idea to check that you have all the necessary information regarding your business at hand before starting to create your Set of Books.&nbsp;

Once you have the necessary information available, you may proceed to create your Set of Books. Should you not have all the information available at this stage, you may go back and enter or change some of the information at a later stage.&nbsp;

The following options to create a Set of Books using the "*Let osFinancials help you to create a Set of Books?" /*&nbsp; "*Let TurboCASH help you to create a Set of Books?" wizard*, is available:

* To create a Set of Books using a template on your system.&nbsp;
* To create a Set of Books downloading a template from osFinancials5.

**To create a Set of Books using a template:**&nbsp;

1. On the **Start** ribbon, you can select the **Create Set of Books** option or simply press the **F10** key on your keyboard.

![Image](<lib/books-creation-wizard-default-option.png>)

2. osFinancials5.1 automatically defaults to the option "*Let osFinancials help you to create a Set of Books?"* or in TurboCASH5-3 "*Let TurboCASH5-3 help you to create a Set of Books?"* as this is the recommended option for most users. You only need to check that it is selected.&nbsp;
2. Click on the **Next** button. The Download books - "*Which Chart of Accounts do you want to use?*" screen will automatically be launched, if you have an Internet connection:&nbsp; ![Image](<lib/create-set-of-books-download.png>)

| ![Image](<lib/admon-note.png>) | *osFinancials5/TurboCASH5 supports 79 different languages and counties / tax regimes languages liked to 87 Sets of Books (databases) which may be downloaded and customised to meet your requirements.*&nbsp; *If the set of books for your language and or country, is not available in the books directory, you may select any of the available Firebird [Sets of Books and linked languages available as a free download*](<sets-of-books-download-templates.md>)*.*&nbsp; |

It will usually be the default directory in which osFinancials5.1/TurboCASH5-3 is installed or other directory, which you have specified when you have installed osFinancials5.1/TurboCASH5-3.*&nbsp; *If you wish to create your Set of Books in any other directory or location on your system, you may click on the Browse button to select any folder or drive on the Pick a directory screen.*&nbsp; |

7. Click on the **Next** button. &nbsp;

![Image](<lib/create-boolld-default-setup-options.png>)

8. By default, all options is selected. It is recommended that you keep all setup options selected to confirm or change the basic settings. The screens, based on this selection, will launch when you click on the **Finish** button. The confirmation message will be displayed.

*"Are you sure you want to create the MY-BOOKS&nbsp; Set of Books?" (where "MY-BOOKS" is the name of the Set of Books entered)* &nbsp;

9. Click on the **Yes** button. The following screens from the **Setup** ribbon will automatically be displayed as selected on the last screen of the Creation wizard.
9. Enter and select the necessary information and settings for the following screens:&nbsp;
1) &nbsp;

   1) [**Company info**](<setup-company-info.md>) - Enter all your business' details such as the address, e-mail address, contact numbers and Company and Tax Registration numbers, which you need to display on all documents, reports, and lists. You may also select whether your default output option for documents, reports and lists are on: screen, printer, file, e-mail or fax. You may also load a logo picture to be displayed on your reports or documents, and set your Windows operating system's default date and currency formats.
   1) [**Reporting dates**](<setup-reporting-dates.md>) - Set up the number of reporting dates required, and the start date of your financial year period. The periods or dates will be displayed in the format in which you have set your Windows Operating System's default date on the previous screen.
   1) [**Batch types**](<batch-types-setup.md>) - Check the default batch types, add (create) new batch types, delete an existing batch type, or change the name of existing batch types in which you wish to enter and update or post your transactions to the ledger.
   1) [**Groups**](<setup-groups.md>) - Set up to two Reporting groups for your General ledger, Debtors, Creditors accounts, Stock and Document groups.
   1) [**Documents setup**](<documents-setup.md> "TC4:Setup Documents") - This option allows you to set up your documents Invoices, Credit notes, Quotes, Orders, Purchases and Supplier return documents, you wish to hand to customers (debtors) or suppliers (creditors) when you sell, or quote, or order and purchase, stock items and services.
   1) [**Stock information**](<stock-information.md>) - Set the selling price descriptions, default selling price, cost of sales, options for documents, unit sizes, tax classes, etc.&nbsp; This option should be used if you use stock and/or wish to generate documents (Invoices, Credit notes, Quotes, Orders, Purchases and Supplier returns).&nbsp;
   1) [*Salesperson*](<setup-salespersons.md>) - This option allows you to enter or add the salespersons who will be processing documents.

11. After the last setup screen (i.e. [*Salesperson*](<setup-salespersons.md>)) is closed, the new Set of Books, is automatically opened and ready to work in.
