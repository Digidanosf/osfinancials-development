# Budgets - Transactions details tab (Reports ribbon)

***

The **Transaction details** tab lists all the budget records for the selected periods or dates selected on the options section of the **Budget** tab.&nbsp;

You may overtype any amount, add or edit references and descriptions, as necessary. &nbsp;

![Image](<lib/budget-transactions-tab.png>)

The details of the columns, is as follows:

1. **id** - The ID or record ID is automatically generated for each budget entry.&nbsp;
1. **Name** - Account name or description&nbsp;
1. **Date** - The date for the period of the budget entry.&nbsp;
1. **Reference** - The default Reference "*BUDG*" or your reference as entered on the options section of the **Budget** tab.&nbsp;
1. **Description** -&nbsp; The default Description "*Auto budget*" or your description as entered on the options section of the **Budget** tab. &nbsp;
1. **Amount** - The amount for the budget entry. The&nbsp; total debit amounts an credit amounts (prefixed by a - (minus sign)) will be listed.&nbsp;
1. **User** - If Users were added in **Setup** **→** **Access control** (**Setup** ribbon), the name of the user or users will be listed.&nbsp;
1. **Change date** - The date and timestamp on which the budget amount is entered or changed.&nbsp;
1. **Project** - If budgets were entered or processed for [Projects](<projects.md>), the project code will be listed. You may select a different Project code, if required.&nbsp;
1. **Cost centre 1** - If budgets were entered entered for [Cost centres](<cost-centres.md>) (Cost centre 1), the Cost centres for Cost centre 1 will be listed. You may select a different Cost centre 1 code, if required.
1. **Cost centre 2** - If budgets were entered entered for [Cost centres](<cost-centres.md>) (Cost centre 2), the Cost centres for Cost centre 2 will be listed. You may select a different Cost centre 2 code, if required.&nbsp;
