# Stock item listing - Stock grid reports&nbsp;

The "*Stock item* listing" report, will list the Stock code, Stock description, Quantity on hand, the Unit description&nbsp; and the total value for each stock item. All active (enabled) Stock items as well as inactive (disabled) Stock items will be listed. &nbsp; &nbsp;

| ![Image](<lib/admon-tip.png>) | *You may select the "[**Stock item listing***](<report-stock-item-listing1.md>)*" report on the **Reports → Stock →** **Stock item listing** menu (**Reports** ribbon). You can choose to print a report for a specific stock item or a range of stock items. You can also filter the report to include only stock items that are linked to selected stock groups for [stock group 1*](<stock-item-reporting-groups.md>)*.* |
