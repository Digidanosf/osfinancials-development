| ![Image](<lib/B-multi-language.jpg>) | ***Multi-Language accounts** - [Manual*](<https://www.osfinancials.org/en/plugin-manuals/515-multi-language-plugin> "target=\"\_blank\"") *- [Shop*](<https://www.osfinancials.org/en/webshop/accountancy/multi-language-set-of-books> "target=\"\_blank\"") *- **Licence** : Once-off -* *This plugin makes it possible to translate the ledger and group names in multiple languages. It is already possible to choose different languages for the interface (the buttons). But this plugin makes it possible to do the accounting in different languages. Per user you can select a language, so the user can account in his/her own language. The set of books can be viewed in different languages, therefore it is possible an employee enters information into an English Set of Books and a bookkeeper, auditor, etc. can see the set of books in a set language, e.g. Afrikaans, Dutch, etc.* |

Should you have selected the incorrect language when installing osFinancials5/TurboCASH5, you may select the language applicable to your country.

The osFinancials5/TurboCASH5 interface will display in the selected language in [**Switch language**](<switch-language.md>) (**Start** ribbon). Depending on the Set of Books template downloaded when [Creating a Set of Books (Creation wizard)](<create-books-default-download.md>); or downloaded from&nbsp; the [Product information → Technical information → Databases](<http://www.osfinancials.org/en/productinformation/technical-information/databases> "target=\"\_blank\"") page on the osFinancials website); osFinancials5/TurboCASH5 will automatically display the interface language of the language file which have already been linked to each Set of Books. (**Setup → [Access control**](<set-default-language.md>)).&nbsp;

When opening a Set of Books, it will automatically load the language file. All screens, reports and layout files are translatable via the language files.

The following options are available to change the language:

1. Set the global language: [**Switch language**](<switch-language.md>) (**Start** ribbon) and select the correct language for your country, state, province, etc. &nbsp;
1. Set the language for a specific Set of Books: **Setup** **→ Access control** - [**System language**](<set-default-language.md>) field on the **Standard security** tab (**Setup** ribbon).
1. Languages may also be customised in **Tools → [Customise language**](<customise-language.md>) (**Setup** ribbon).
