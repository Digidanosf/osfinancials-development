# Context menu - Others - Open item link

***

The Open item link option will launch the "*Open item link*" form for the debtor (customer / client) account associated with the selected document, if Invoices, Credit notes or Quotes document type is selected.&nbsp;

If purchase documents, (Purchases, Supplier returns or Orders document type is selected, the "*Open item link*" form for the creditor (supplier / vendor) account associated with the selected document will be launched. &nbsp;

You may view all the transactions and process open item links for the selected account. See - [Open item link](<open-item-link.md>).

**To launch the Open item link for a selected document:**&nbsp;

1. On **Default** ribbon, select **Documents** (Documents list screen), select the Document type, if the correct document type is not selected.
1. Right-click on a selected document. On the context menu, select the **Others → Open item link** option.&nbsp;

This will launch the "*Open item link*" form for the debtor (customer / client) account; or the creditor (supplier / vendor) account; associated with the selected document.
