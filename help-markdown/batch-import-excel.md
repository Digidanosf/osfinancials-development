# Batch transactions - Spreadsheet Import

***

You may import batch transactions from a valid spreadsheet saved as a "*Excel Workbook (\*.xlsx) file type* into a batch.&nbsp;

You may then add new transactions, edit existing transactions, etc.&nbsp;

**To import batch transactions from a Excel Workbook (xlsx) file:**&nbsp;

1. On the **Default** ribbon, select **Batch entry** (**F2**).&nbsp;
1. Select a blank Batch, and click **Open**.&nbsp;

![Image](<lib/batch-transactions-spreadsheet-import-option.png>)

3. Click on the **Spreadsheet import** icon.&nbsp;
3. The "*Data Import Export*" be launched without any data.&nbsp;

![Image](<lib/batch-transactions-spreadsheet-import-blank.png>)

5. On the "*Data Import Export*" screen, click **Open** to select a valid saved exported file.&nbsp; &nbsp;

![Image](<lib/batch-transactions-spreadsheet-export-open.png>)

| ![Image](<lib/admon-tip.png>) | *You may click on the "Show the preview pane" icon on the "Open" screen to display a preview of the selected file to open (import) in the "Data Import Export" screen..* |
| --- | --- |


6. Select the folder and file. Click on the **Open** button. The batch transactions included in the selected "Excel Workbook (\*.xlsx) file opened will be imported and listed in the "*Data Import / Export*" screen.&nbsp;

![Image](<lib/batch-transactions-spreadsheet-imported.png>)

| ![Image](<lib/admon-tip.png>) | *Should you need to change an account (in column B - "SACCOUNT" or column E - "STAX" or column G - "SCONTRAACCOUNT"),&nbsp; you may click on the **Data** tab and click on the "Find account" option.* ![Image](<lib/batch-transactions-spreadsheet-lookup.png>) *This will launch the "Select account" lookup screen.*&nbsp; ![Image](<lib/batch-transactions-spreadsheet-lookup-select.png>) |
| --- | --- |


7. You may select the following options:

   1. **Show data before import** - If this option is selected, the *"ImportCheck"* screen will be launched, before the import process is initialised.
   1. **Import check** - Click on this option to populate the values of the various columns on the **Settings** tab to be imported.&nbsp;

![Image](<lib/batch-transactions-spreadsheet-import-setting.png>)

| ![Image](<lib/admon-tip.png>) | *See - [Spreadsheet export*](<batch-export-excel.md>) *- The data of the imported accounts is imported (populated) in the following eleven (11) columns in the spreadsheet on the "Data Import Export" screen..* |
| --- | --- |


8. Click on the **Import** tab and on the **Import** icon. If the "*Show data before import" option is selected, the "ImportCheck" screen will be launched*: &nbsp;

![Image](<lib/batch-transactions-spreadsheet-export-check.png>)

9. The transactions that will be imported into the batch will be listed. You may edit these transactions, if necessary. Click on the **All** or **OK** button to start the import process.&nbsp; &nbsp;

![Image](<lib/batch-transactions-spreadsheet-imported-done.png>)

10. Click on the **Import** icon. The colour of the cells in the rows for the transactions in the "*SREFERENCE*" column should be green. This means that the import process is finalised.&nbsp;
10. Click on the **Close** icon in the titlebar of the "*Data Import Export*" screen. The imported transactions will be listed in the "*Batch entry*" screen.&nbsp;

![Image](<lib/batch-transactions-spreadsheet-imported-batch.png>)

12. You may edit the transactions if necessary.&nbsp;

| ![Image](<lib/admon-note.png>) | *Once you have finished importing and/or editing the transactions in the batch you need to: -* [*Change the Alias (Batch name)*](<batch-change-alias.md>)*.* [*Balance the batch*](<batch-balance.md>)*.* [*List the batch*](<batch-list.md>)*.* [*Post the batch*](<batch-post.md>)*.* |
| --- | --- |
