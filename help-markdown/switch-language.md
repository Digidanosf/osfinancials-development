# Switch language

***

When you install and start osFinancials5/TurboCASH5 for the first time, the select a Language window is displayed on which you may select the language and country. If you have selected the incorrect language / country, when launching osFinancials5/TurboCASH5 for the first time, you may change it.&nbsp;

| ![Image](<lib/admon-important.png>) | *English - This is the driver language file.* |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *You may customise any language for your own personal or industry specific needs.*&nbsp; |
| --- | --- |


**To choose a language:**&nbsp;

1. On the [**Start**](<start-ribbon.md>) ribbon, select [**Switch language**](<switch-language.md>). The "*Choose a language*" screen is displayed:&nbsp;

![Image](<lib/language-switch-v.png>)

2. Select the Language of your choice.
2. Click on the **OK** button. A progress indicator at the bottom of the screen will display the progress. When the process is completed, the ribbons and the screens will be displayed in the selected language.&nbsp;

| ![Image](<lib/admon-tip.png>) | *Users in **Access control** can have their language set in the "System lang" field. Upon logging into osFinancials5/TurboCASH5, the user interface and reports will be displayed in the language assigned to that user.* ***To access the [Customise language***](<customise-language.md>) ***option:***&nbsp; *Right-click in any area of the list of languages (on the "Choose a language" screen).*&nbsp; *Click on the **Customize** context menu option. This will directly access the selected (highlighted) language on the "Customize Languages" screen (**Tools → [Customise language***](<customise-language.md>) *menu on the **Setup** ribbon).*&nbsp; |
