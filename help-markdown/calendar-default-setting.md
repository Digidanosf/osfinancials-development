# Set the Calendar as default

***

In osFinancials5/TurboCASH5, the default behaviour is for the Welcome screen to be launched when opening a Set of Books. However, you have the option to activate the Calendar to be launched instead. This means that whenever you open a Set of Books, the Calendar will be automatically displayed as the default screen.

Activating the Calendar as the default screen can be useful if you frequently rely on the Calendar feature and want quick access to it whenever you open your Set of Books. It allows you to directly view and manage your scheduled events, appointments, and other calendar-related tasks without having to navigate through the Welcome screen first.

**To set the Calendar as the default when opening the Set of Books:**&nbsp;

1. On the **Setup** ribbon, select **Setup → System parameters**. &nbsp;

![Image](<lib/calendar-show-default-setting.png>)

2. Select (tick) the “*Show calendar*” field.
2. Click on the **OK** button.
