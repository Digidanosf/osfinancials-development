# Batch Entry - Trial balance option - Batch entry selection screen context menu

On the Batch entry selection screen's context menu, Select the "[*Trial balance*](<ledger-analyser-unposted-reports.md>)" option. You may select to include only unposted batches or to include posted batches.&nbsp;

The following known issues:

1. The **Print** button will by default, not be displayed. To initialise the **Print** button, select Income statement or Balance sheet and then Trial balance in the "*Report type*" field.
1. Click report type select Income statement or Balance sheet to add the print button - then report can be printed.
1. If there no unposted transactions in batches, (and the "*Unposted only?*" option is selected) the Unposted batches trial balance will still be launched with a zero.&nbsp;

![Image](<lib/bug-batch-unposted-trial-balance.png>)

**FIXED** - The "*Unposted batches*" screen may disappear behind other programs or applications. If this happens, click on the osFinancials5 icon on the Windows task bar.&nbsp;
