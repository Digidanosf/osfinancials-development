# Show welcome

***

The "*Welcome to osFinancials*" screen will by default be displayed. It will list the latest News from osFinancials.&nbsp;

To close or exit the "*Welcome to osFinancials*" screen, click on **X** (close).

To display the "*Welcome to osFinancials*" screen again, select **Help → Show welcome** on the **Start** ribbon.&nbsp;

| ![Image](<lib/admon-tip.png>) | *You may set the [Calendar*](<calendar-default-setting.md>) *as the default to launch in stead of the "Welcome to osFinancials" screen, when you open a Set of Books.*&nbsp; |
