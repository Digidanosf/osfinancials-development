# Spreadsheet - Sales analysis - Stock movement

***

The Spreadsheet report - Stock movement will list the total stock quantities for each stock item in and out for the selected period.&nbsp;

The data in this spreadsheet is similar to the "[*Stock In / Out*](<report-stock-in-out.md>)" report (**Reports → Stock items → Stock In / Out** - (**Reports** ribbon)).

| ![Image](<lib/admon-important.png>) | ***The following will be included in the spreadsheet:*** *Only documents (i.e. invoices, credit notes, purchase and supplier return documents) which is posted (updated to the ledger) will be included.*&nbsp; *Only the default stock type (trading stock items (Stock item type = 0))&nbsp; will be included.*&nbsp; ***The following will be excluded from the spreadsheet:*** *All the other ten (10) stock types (e.g. Sales, Sales (no stock), Purchases, Purchases (no stock), Financial entry, Option item, Option item (no stock), BOM, BOM (Production) and Linked item) will be excluded in the spreadsheet import.* |
| --- | --- |


| ![Image](<lib/admon-important.png>) | *This report only includes document transactions which are posted. Unposted documents will be **NOT** be included in this report.* *To view a list of unposted documents, which is not updated to the ledger, go to **Input → [Checking unposted items***](<checking-for-unposted-items.md>) *(**Default** ribbon).* |
| --- | --- |


**To import Stock movements (Stock In / Out) into a spreadsheet:**&nbsp;

1. On the **Reports** ribbon, select **Spreadsheet reports → Sales analysis → Stock movement**.&nbsp;
1. On the parameters ("*Params*") screen, select the period and click **OK**. The total quantities for each stock item will be listed in the spreadsheet:

![Image](<lib/spreadsheet-reports-stock-movement.png>)

&nbsp;

| ![Image](<lib/admon-note.png>) | *The Document type in column, E is as follows:* *10 - Invoice* *11 - Credit note* *12 - Purchase*&nbsp; *13 - Supplier return* *14 - Quote* *15 - Order*&nbsp; *16 - Adjustment* *Quotes and Orders will be excluded in the spreadsheet import.* |
