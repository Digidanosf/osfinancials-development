# Edit task - Calendar

***

Once a task has been added, you may then edit it, if necessary.&nbsp;

**To edit a task:**&nbsp;

1. Select a task on the Task list section of the Agenda screen, and right-click to select the **Edit task...** option.&nbsp;

| ![Image](<lib/admon-note.png>) | *The subject (topic) of the task will be displayed in the title bar of the Task screen. If you create a new task it will be displayed as Untitled - Task.* |
| --- | --- |


2. Edit any of the fields you need to edit. You may also add any additional information, or edit any notes or information already added when the task was created, or previously edited.
2. If the task is completed, select the Complete field (tick in box). When a Task is set as completed, the Task will be ticked and striped through on the Task list.
