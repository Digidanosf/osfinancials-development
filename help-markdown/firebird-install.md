# Firebird install

***

Copy Firebird client library to \<system\> directory = Tick

Firebird 1.5 creates databases with an ODS (On-Disk Structure) version of 10.1.&nbsp;

Firebird 2.1 creates databases with an ODS (On-Disk Structure) version of 11.1.&nbsp;

Select the "*Firebird.exe*" file in installs folder of the osFinancials5/TurboCASH5 installation directory.&nbsp;
