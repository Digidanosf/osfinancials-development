# Creditor listing - Credit Control Listing (Reports menu)

***

The "*Creditor listing - Credit control listing (Credit limit vs Current balance) as at...*"&nbsp; report will list the Account code, Name of the Creditor (account description), Postal address, Telephone 1 / 2, Credit limit, Current balance and the Available balance as at the date and time of printing this report.&nbsp;

| ![Image](<lib/admon-important.png>) | *This report only includes batch and document transactions which are posted. Unposted batches and documents will be **NOT** be included in this report.* *To view a list of unposted batches and documents, which is not updated to the ledger, go to **Input → [Checking unposted items***](<checking-for-unposted-items.md>) *(**Default** ribbon).* |
| --- | --- |


# Creditor listing - Credit control listing report options

**To print a Creditor listing - Credit control listing (Credit limit vs Current balance) as at... report:**

1. On the **Reports** ribbon, select **Reports → Creditors → Listing**.&nbsp;
1. Select "*Credit control listing (Credit limit vs Current balance) as at*".

![Image](<lib/reports-creditor-list-credit-control-options.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a creditor (supplier / vendor) account or a range of creditor (supplier / vendor) accounts, to include in the report.
   1) **Sequence** - Select "*Creditor code, Description, Credit limit* or *Disabled*" to print the report details in the order (sort) in which you wish to list the data.
   1) **Show contact details** - You may select this option to view the postal address, delivery address, telephone numbers, Fax number, email address, website and more for each creditor account.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The report will only be printed for those creditor (supplier / vendor) accounts linked to the selected Creditor reporting group 1.

4. Click on the **OK** button.&nbsp;
