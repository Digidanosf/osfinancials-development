# Show / Hide ribbon options

***

**To hide the Ribbon** **options**

Select any of the ribbon tabs.

Right-click and select "*Minimize the ribbon*" (a tick will be added before this option).

![Image](<lib/ribbon-minimize-context-menu.png>)

**To show the Ribbon options**

Select any of the ribbon tabs.

Right-click and select "*Minimize the ribbon*" (the tick before this option will be removed).&nbsp;
