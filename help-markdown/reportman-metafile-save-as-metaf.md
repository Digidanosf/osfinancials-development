# Reportman output - Save metafile as metafiles

***

The following Meta file file types is available to save reports or layout files:

* Report Metafile \*.rpmf&nbsp;
* Executable Meta file uncompressed \*.rpmf&nbsp;
* Executable meta file \*.exe &nbsp;

| ![Image](<lib/admon-note.png>) | *These File types could not be opened, if the required applications are not available on your system.*&nbsp; |
| --- | --- |


Executable meta file \*.exe may produce a system error as follows:

*"System Error.&nbsp; Code: 2.*\
*The system cannot find the file specified"*&nbsp;
