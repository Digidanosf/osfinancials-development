# Stock - Stock item listing - All selling prices (Reports menu)

***

You may print the following two (2) reports to manage the stock items.&nbsp;

1. **All selling prices** - Displays a list of Selling prices 1 / 2 / 3. The details option will display the Extra description, Manufacturer, Gross and Net weight.
1. **Stock item listing** - Displays a list of the stock items and the quantities and cost prices. The details option will display the Extra description, Manufacturer, Unit size, Gross and net weight, Selling price 1 / 2 / 3, Average cost, Unit cost, Barcode, Reporting group 1 / 2, Preferred supplier 1 and Supplier 1's Stock code.&nbsp;

# Stock item listing - All selling prices report options

**To print Stock item listing - All selling prices reports:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Stock → Stock item listing**.&nbsp;
1. Select "*All selling prices"* report.&nbsp;

![Image](<lib/reports-stock-list-selling-prices-options.png>)

3. Select the following:
1) &nbsp;

   1) **From... To...** - Select the stock code(s) to include in the report.&nbsp;
   1) **Reporting group 1** - This is optional and by default not selected (not ticked). You may select (tick) the reporting group(s) to include in the report.&nbsp;
   1) **Show details** - By default, this option is not selected. If this option is selected, it will display the details for each stock item. &nbsp;

4. Click on the **OK** button.&nbsp;
