# Customise osFinancials5/TurbCASH5-4

***

| ![Image](<lib/admon-tip.png>) | *osFinancials5 installs, by default, in the Classic theme. The **Material** theme (black icons) is used for most of the topics in this documentation. This is set in [**Theme***](<themes.md>) *on the **Setup** ribbon.* *The Blue, Glass, Silver, Soft-blue theme includes a similar set of icons as in the Material theme (navy-blue icons).* *The "Classic" theme (default theme) and "VS2010" theme displays the same set of icons (located in the "...\\bin\\themes\\Classic" folder.* |
| --- | --- |


The osFinancials interface will display in the default "*English*" language.&nbsp;

osFinancials5 supports 79 different languages and tax regimes languages liked to 87 Sets of Books (databases) which may be downloaded and customised to meet your requirements.&nbsp;

You may [**Switch language**](<switch-language.md>) (**Start** ribbon) to choose a different language.&nbsp;

![Image](<lib/language-switch-classic.png>)

Depending on the Set of Books template downloaded when [Creating a Set of Books (Creation wizard)](<create-books-default-download.md>); or downloaded from&nbsp; the [Product information → Technical information → Databases](<http://www.osfinancials.org/en/productinformation/technical-information/databases> "target=\"\_blank\"") page on the osFinancials website); osFinancials will automatically display the interface language of the language file which have already been linked to each Set of Books. (**Setup → [Access control**](<set-default-language.md>)).&nbsp;
