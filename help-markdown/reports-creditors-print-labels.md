# Creditors - Print labels (Reports menu)

***

The Creditor labels allows you to print the following two (2) types of labels:

1. **Delivery address** - This will print the Account code, Account description, Contact name and Delivery address for each selected creditor (supplier / vendor) account.
1. **Postal address** - This will print the Account code, Account description, Contact name and Postal address for each selected creditor (supplier / vendor) account.

# Creditor labels options

**To print creditor labels:**

1. On the **Reports** ribbon, select **Reports → Creditors → Print labels**.&nbsp;
1. Select "*Delivery address labels*" (default) or "*Postal address labels*".&nbsp;

![Image](<lib/reports-creditor-labels-01.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a creditor (supplier / vendor) account or a range of creditor (supplier / vendor) accounts, to print labels for.
   1) **Sequence** - Select "*Creditor code, Description, Address 3* or *Postal code*" to order (sort) the labels when printing.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The address labels will only be printed for those creditor accounts linked to the selected Creditor reporting group 1.&nbsp;

4. Click on the **OK** button.&nbsp;
