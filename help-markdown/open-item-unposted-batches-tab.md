# Open item link - Unposted batches tab

***

The **Unposted batches** tab will list all transactions linked to open item accounts (i.e. ledger, debtors and creditors) in batches, which have not yet been posted (updated) to the ledger. The transactions in unposted batches&nbsp; for open item accounts need to be linked to transactions in posted batches.

![Image](<lib/open-item-unposted-batches-tab.png>)

| ![Image](<lib/admon-note.png>) | *If any transactions in unposted batches are not linked to transactions in posted batches, these transactions will not be listed on the **Unposted batches** tab.* &nbsp; |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *You may remove linked transactions (right-click on selected transaction(s) and select **Remove** on the context menu. A confirmation message is displayed:*&nbsp; *"Continue action Unlink transaction?"*&nbsp; *Click **Yes** to remove the link. This will clear the links in the “Batch entry” screen.* |
| --- | --- |


When processing transactions in batches and an open item account is selected, the “*Open item selection*” screen list all the outstanding transaction. For example, if a payment is received, you may select to link the receipt to outstanding amount(s) in batches as well as documents.&nbsp;

![Image](<lib/open-item-unposted-batches-receipts.png>)

| ![Image](<lib/admon-note.png>) | *If a transaction in unposted batches is linked to outstanding amounts in the Open item selection screen the transaction line will be displayed in a peach background.* |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *You may also select an account on the "Batch entry" screen (right-click on selected transaction) and select the "[Select from account open item list*](<batch-entry-menu-select-open-ite.md>)*" option. The "Open item selection" screen will list the unlinked transactions as well as transactions with outstanding amounts.* &nbsp; |
