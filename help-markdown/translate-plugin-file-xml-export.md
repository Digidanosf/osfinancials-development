# FileAndXMLImportExport plugin errors

## FileAndXMLImportExport plugin error - folder not found

**Menu** : **Plugins → Tools → FileAndXMLImportExport**&nbsp;

*Import export folder not found\!*\
*C:\\osfinancials5\\importdefs\\FXMLEXPORT*&nbsp;

**Solution** : Added a folder "*importdefs*" and subfolder "*FXMLEXPORT*" in the osFinancials5 root directory.

Then the plugin setup could be launched.

## List index of bounds error

*List index out of bounds (-1).&nbsp; TComboBoxStrings is empty*
