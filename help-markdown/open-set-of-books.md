To access the Set of Books option in osFinancials5/TurboCASH5, you need to click on the **Open** icon on the **Start** ribbon. This option will display a list of all valid Sets of Books (databases) that are located in the "*books*" folder of the osFinancials5 installation folder. Additionally, it will also show any Sets of Books that are stored in other locations, such as on a different part of the hard disk or on a mapped drive on the network, which have been added to the list. The path or directory of these Sets of Books can be set in the **Properties** of the Set of Books.

# Icons on Open Set of Books screen&nbsp;

## Browse&nbsp;

The **Browse** icon will allow you to add a Set of Books to the list.&nbsp;

If the Set of Books you want to open is stored in a location other than the "*books*" folder in the osFinancials5/TurboCASH5 installation folder, you can use the **Browse** icon to locate and select it. Clicking on the **Browse** icon will allow you to navigate to the directory or folder where the Set of Books is stored and select it from there.

**To browse and add a valid osFinancials5/TurboCASH5 Set of Books to the Set of Books list:**&nbsp;

1. If you click on the **Browse** button, you may select from any other directory. The '*Browse for Folder'* screen is displayed.&nbsp;
1. You must pick a valid osFinancials5/TurboCASH5 Set of Books. If the directory does not contain a valid Set of Books, it will not be added to the list.&nbsp;

osFinancials5/TurboCAH5 supports 79 different languages and counties / tax regimes languages liked to 87 Sets of Books (databases) which may be downloaded and customised to meet your requirements.&nbsp;

2. **bin** directory - The bin directory consists of the following three folders:&nbsp;

The remaining Chart of Accounts can be fully customised according to your specific requirements. This flexibility enables you to tailor the Sets of Books to suit your unique accounting needs. ## Purge list&nbsp;

The **Purge** list icon is used to remove a selected Set of Books from the list, but it does not delete the Set of Books from the directory. It's important to note that you cannot purge a Set of Books located in the osFinancials5/TurboCASH5 directory (books folder). To remove a Set of Books from the books folder, you need to use the [**Delete Set of Books**](<delete-set-of-books.md>) option located on the **Start** ribbon. If you want to remove a Set of Books from the list but keep the Set of Books, you will need to move the entire Set of Books folder out of the osFinancials5/TurboCASH5 directory using File Explorer or any other file/folder management program.

When you click on the **Purge icon**, the following confirmation message will be displayed:&nbsp;

*"This will seriously affect your Set of Books\! Are you sure you want to clear this Set of Books?"*

## Properties&nbsp;

The **Properties** icon displays the name of the Set of Books (e.g. "*MYBOOKS*") and its holding directory in the list.

*e.g. C:\\osFinancials5\\books\\MYBOOKS\\books.fdb.*&nbsp;

1. If you select a Set of Books and click on the **Properties** icon, the following screen is displayed: \
![Image](<lib/open-books-properties.png>)
1. If you need to change any of the properties for the Set of Books, follow these steps: &nbsp;
1) &nbsp;

   1) **Enter the name of the Set of Books** - Enter a valid name for the Set of Books.&nbsp;
   1) **Server** - If osFinancials5 is configured as a Multi-user product, enter the server details.
   1) **Enter the path where the Set of Books are stored** - Enter the path or directory where the Set of Books is stored on your system..&nbsp;
   1) **Username** - Enter a valid username for the Set of Books
   1) **Password** - Enter a valid password for the Set of Books.
   1) **Databasetype** - Select the appropriate database type from the following options:

      1) **Firebird** - This is the default database type for all Sets of Books included in the osFinancials5 installation in the "books" and the "...bin/Tutorials" folder.&nbsp; Sets of Books (Firebird database type) may also be downloaded when you create a Set of Books using the "[*Let osFinancials help you to create a Set of Books?*](<create-books-default-download.md>)" / "[*Let TurboCASH help you to create a Set of Books?*](<create-books-default-download.md>)" option.
      1) SharedDisk (Firebird) - For sharing in multi user setup for multi-user environment for Firebird databases.
      1) MSSQL -&nbsp; [Microsoft SQL server](<mssql-sql-database-create.md>) -&nbsp;
      1) Mysql -&nbsp; Not fully supported.
      1) Postgress - Not fully supported.

## Cancel button (right-click context menu)

The **Cancel** button on the Set of Books screen has a right-click context menu that provides four options:

![Image](<lib/open-books-cancel-menu.png>)

1. **Purge invalid** - This option deletes Sets of Books that are listed in **Red - Font colour** on the "*Set of Books*" open screen, indicating that they are either deleted, moved to another location, or stored on an unavailable or unmounted drive or network.
1. **Purge not in this folder** - This option does not delete the files from the directory. Sets of Books located in the "*books*" folder of the osFinancials5/TurboCASH5 directory cannot be purged using this option. To remove them, use the [**Delete Set of Books**](<delete-set-of-books.md>) option in the **Start** ribbon.
1. **Delete not local (Books this PC can not find)** - This option removes Sets of Books that are not available on the current device.
1. **Reinit version** - This option updates the database of the selected Set of Books. Once finished, a confirmation message will be displayed.

*Database version updated.*
