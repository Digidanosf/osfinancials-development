# Options setup

***

If you wish to create the [Option item](<stock-item-new-option-item.md>) and [Option item (no stock)](<stock-item-new-option-no-stock.md>) stock types, you may select items that are a combination of options (like colour and size) options.&nbsp;

These options are then linked to the Options stock item type (**Stock items** on the **Default** ribbon). When processing documents, the available options (options list) will be launched from the quantity fields when selecting documents.&nbsp;

The Options with stock allows every entry to have a sales price, cost price, quantity and minimum stock level. This will also be used in generating purchases on supplier 1.&nbsp;

**To create / add Options for option items:**&nbsp;

1. On the **Setup** ribbon, select **Setup** **→** **Stock information**. The "*Stock information*" screen will be displayed.

2) Click on the **Setup options** button. The "*Setup options"* screen is displayed:&nbsp;

![Image](<lib/stock-options-setup-01.png>)

3. Click on the ![Image](<lib/record-icons-05.png>) (**Add**) button. A record number will automatically be assigned. Enter the name for the Option, e.g. Colour, Size, etc.&nbsp;
3. Click on the ![Image](<lib/record-icons-08.png>) (**Save**) icon.
3. Double-click on the item to add the list of options and the extra prices. This may be the size, colours, etc. The selling price of the first option (default option) should be zero, as this is the default selling price of the stock item to which this is linked. The extra prices for the other options (on the options list) are the extra prices to be added to the stock items default selling price.
3. Once finished with the options list, click on the **OK** button to save the list. \
![Image](<lib/stock-options-setup-02.png>)
3. Close the "*Setup options*" screen (click **Close** button on the title bar).
3. Close the "*Stock information*" screen and return to the **Stock items** screen (**Default** ribbon). You may then link this option on the **Options** tab of the Stock item.&nbsp;
