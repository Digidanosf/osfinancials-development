# Graphs

***

[Graphs per week](<graphs-week.md>) and [Graphs per month](<graphs-month-sales.md>) is available to view the data in a graph on the **User reports** menu.

[**Graphs per week**](<graphs-week.md>)**:**&nbsp;

1. **Graphs → Ledger → [Profit / Loss per week**](<graphs-week-profit-loss.md>)&nbsp;
1. **Graphs → Debtors → [Debtors per week**](<graphs-week-debtors.md>)&nbsp;
1. **Graphs → Creditors → [Creditors per week**](<graphs-week-creditors.md>)&nbsp;
1. **Graphs → Graphs per week → Sales menu**: &nbsp;

   1. [Sales (Invoices - Credit notes) per week](<graphs-week-invoice-credit-notes.md>)&nbsp;
   1. [Invoices per week](<graphs-week-invoices.md>)&nbsp;
   1. [Credit notes per week](<graphs-week-credit-notes.md>)&nbsp;
   1. [Quotes per week](<graphs-week-quotes.md>)&nbsp;

1. **Graphs → Graphs per week → Purchases menu:** &nbsp;

   1. [Purchases (Purchases - Supplier returns) per week](<graphs-week-purchases-returns.md>)&nbsp;
   1. [Purchases per week](<graphs-week-purchases1.md>)&nbsp;
   1. [Supplier returns per week](<graphs-week-supplier-returns.md>)&nbsp;
   1. [Orders per week](<graphs-week-orders.md>)&nbsp;

[**Graphs per month**](<graphs-month-sales.md>)**:**

1. **Graphs → Debtors → [Debtors per month**](<graphs-month-debtors.md>)&nbsp;
1. **Graphs → Creditors → [Creditors per month**](<graphs-month-creditors.md>)&nbsp;
1. **Graphs → Graphs per month → Sales menu:** &nbsp;

   5) [Quotes per month](<graphs-month-quotes.md>)&nbsp;

1) **Graphs → Graphs per month → Purchases menu:** &nbsp;

   1) [Purchases (Purchase - Supplier returns) per month](<graphs-month-purchases-returns.md>) &nbsp;
   1) [Purchases per month](<graphs-month-purchases1.md>) &nbsp;
