# Debtor listing - Contacts (Reports menu)

***

The "*Debtor listing&nbsp; - Contacts*" report will list the Name of the Contact, Telephone 1 / 2 and the debtor (customer / client) account code and name (description) linked to the contact.

# Debtor listing - Contacts report options

**To print a Debtor&nbsp; - Listing - Contacts report:**

1. On the **Reports** ribbon, select **Reports → Debtors → Listing**.&nbsp;
1. Select "*Contacts*".

&nbsp;![Image](<lib/reports-debtor-listing-contacts-options.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a debtor (customer / client) account or a range of debtor (customer / client) accounts, to include in the report.
   1) **Sequence** - Select "*First Name, Name (Surname)*, *Debtor code"* or "*Description*" to print the report details in the order (sort) in which you wish to display the data.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The report will only be printed for those debtor (customer / client)&nbsp; accounts linked to the selected Debtor reporting group 1.

4. Click on the **OK** button.&nbsp;
