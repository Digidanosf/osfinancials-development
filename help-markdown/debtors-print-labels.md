# Debtors - Print labels (Reports menu)

***

The Debtor labels allows you to print the following two (2) types of labels:

1. **Delivery address** - This will print the Account code, Account description, Contact name and Delivery address for each selected debtor (customer / client) account.
1. **Postal address** - This will print the Account code, Account description, Contact name and Postal address for each selected debtor (customer / client) account.

# Debtor labels options

**To print Debtor labels:**

1. Click on the **Reports → Debtors → Print labels**.&nbsp;
1. Select "*Delivery address labels*" (default) or "*Postal address labels*".&nbsp;

&nbsp;![Image](<lib/reports-debtor-labels-options.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a debtor (customer / client) account or a range of debtor (customer / client) accounts, to print labels for.
   1) **Sequence** - Select "*Debtor code, Description, Address 3"* or "*Postal code*" to order (sort) the labels when printing.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The address labels will only be printed for those debtor (customer / client) accounts linked to the selected Debtor reporting group 1.&nbsp;

4. Click on the **OK** button.&nbsp;
