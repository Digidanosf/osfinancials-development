# Reportman output - Print

***

**Print** (**Ctrl+P**) – This will launch the “*Print*” screen. You may select another printer (than your system's default printer), select a specific page and change the number of copies, if necessary.&nbsp;

In this example, the "*Microsoft Print to PDF*" is the default printer.&nbsp;

![Image](<lib/reports-screen-print-options.png>)

Save the print output file.

![Image](<lib/reports-screen-print-options-save-as.png>)

Open the printed pdf file in your system's default PDF App. &nbsp;

![Image](<lib/reportman-save-metafile-pdf-edge.png>)

&nbsp;

If the printed file does not open automatically; or if you need to open it at a later stage; you may open the file in your system's file explorer.

![Image](<lib/reports-screen-print-options-open.png>)

&nbsp;
