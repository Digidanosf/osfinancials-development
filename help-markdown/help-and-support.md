# Help and Support

***

The **Help** on the **Start** ribbon, lists access to help and support information and options to access the osFinancials5/TurboCASH5 community.&nbsp;

![Image](<lib/menu-help.png>)

&nbsp;

The **Help** on the **Start** ribbon, lists the following seven menu items:&nbsp;

1. [**Help**](<help.md>) - Launches the context-sensitive Help file. You may press **F1** to launch the help file from most screens and fields to launch the topic relating to that screen or field. This help system is designed to assist users in understanding and navigating the osFinancials5/TurboCASH5 accounting software, as well as to provide guidance on accounting principles and practices.&nbsp;

2. [**Support**](<support.md>) - This option will display the support information.
2. [**Show welcome**](<show-welcome.md>) - The "*Welcome to osFinancials*" screen will by default be displayed. It will list the latest News from osFinancials5/TurboCASH5.&nbsp;

| ![Image](<lib/admon-tip.png>) | *You may set the [Calendar*](<calendar-default-setting.md>) *as the default to launch in stead of the "Welcome to osFinancials" screen, when you open a Set of Books.*&nbsp; |
| --- | --- |


4. [**Updates**](<install-osfinancials-updates.md>) - This option will list all the installation files available for download. By default, the Update will be launched when starting osFinancials5/TurboCASH5. To disable the check for update at start, select this field and remove the tick.
4. [**Forum**](<forum.md>) - You may access and view the latest and older [Forum](<https://www.osfinancials.org/en/forum-en> "target=\"\_blank\"") posts as well as various categories. Should you have a query or an issue, you may search and will probably find a solution to your query or issue. If not, you may post your query or issue. Registered osFinancials5/TurboCASH5 users will soon help by posting answers or solutions to your query or issue.
4. [**Webshop**](<webshop.md>) - Access to the [osFinancials webshop](<https://www.osfinancials.org/en/webshop> "target=\"\_blank\"").&nbsp;
4. [**Bug report**](<bug-report.md>) - Access to the osFinancials [Forum page](<https://www.osfinancials.org/en/forum-en?view=category\&catid=8> "target=\"\_blank\"").&nbsp; Should you have a bug or an issue, you may search and will probably find a solution to your bug or issue in the Questions and Answers page. If not, you may post your bug or issue.
