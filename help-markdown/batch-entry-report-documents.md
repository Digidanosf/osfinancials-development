# Batch report - Documents (Reports menu)

***

Print the transactions of all posted invoices, credit notes, purchase and supplier return documents for a specific selected period.

| ![Image](<lib/admon-important.png>) | *This report only includes document transactions which are posted. Unposted documents will be **NOT** be included in this report.* *To view a list of unposted documents, which is not updated to the ledger, go to **Input → [Checking unposted items***](<checking-for-unposted-items.md>) *(**Default** ribbon).* |
| --- | --- |


# Batch entry - Documents report options

**To print Batch entry - Documents reports:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Batch entry**.&nbsp;
1. Select "*Documents*", if not already selected.

![Image](<lib/reports-batch-period-documents-options.png>)

3. Select the **Period** - Select the period on the **Per period** tab. You may click on the **Free selection** tab to select a specific date or a range of dates.
3. Click on the **OK** button.&nbsp;

Page 1 -

Page 2 -&nbsp;

This report will list the following:

1. **Document heading** – Document number, Batch type and Reference (as entered in the “*Your reference*” field of “*Document entry*” screens).
1. **Transaction details** -&nbsp;
1) &nbsp;

   1) **Batch id.** – The batch id or number as automatically generated.&nbsp;
   1) **Date** – Date of the transaction.&nbsp;
   1) **Reference** – The document number will be listed.&nbsp;
   1) **Description** – Document number / Your reference (as entered in the “*Your reference*” field of “*Document entry*” screens).
   1) **Account** – The account which was selected for documents.
   1) **Contra account** – The account for the document type.&nbsp;
   1) **Tax account** column – Tax account code.
   1) **Tax** – Tax amount.
   1) **Debit** column – Debit transaction amount.
   1) **Credit** column – Credit transaction amount.&nbsp;

3. **Total amount** – This amount is the accumulative sum of all document transactions. &nbsp;

| ![Image](<lib/admon-note.png>) | *The total in the Tax column, is the accumulative total of all Output tax on sales documents (Invoices and Credit notes) minus the accumulative total of all Input tax for purchase documents (Purchases and Supplier returns).* |
