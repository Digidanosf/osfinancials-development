# Tools

***

This menu allows you to perform a Data integrity check, customise a language for your specific needs or industry. You may close any active screens, etc. in the Set of Books. You may also activate various utilities and plugins / extensions.

**To access the Tools menu:**&nbsp;

1. On the **Setup** ribbon, select **Tools**. A list of the menu options will be displayed.

![Image](<lib/menu-setup-tools.png>)

2. Select one of the available options:&nbsp;
1) &nbsp;

   1) **Blockchain** - By default, Blockchain is not activated. Blockchain needs to be activated in the [**Setup**](<setup-menu.md>) **→ [System parameters**](<setup-system-parameters.md>) on the [**Setup**](<setup-ribbon.md>) ribbon.

| ![Image](<lib/admon-video.png>) | *See [video on osFinancials and Blockchain*](<https://www.youtube.com/watch?v=cVr9dbFM-Es> "target=\"\_blank\"")*.* |
| --- | --- |


1. &nbsp;
   2. [**Data integrity check**](<data-integrity-check.md>) - Run a Data integrity check and print a Data integrity check report.
   2. [**Customise language**](<customise-language.md>) - Customise (edit, change and translate) language files.&nbsp;
   2. [**Close active forms**](<close-active-forms.md>) - Close any open forms (e.g. Documents, Debtors, Creditors, Stock items screens, etc.)
   2. [**Activate plugins**](<plugins-activate.md>) - By default, no plugins is activated when osFinancials5/TurboCASH5 is installed. This option allows you to activate / deactivate all plugins, registered plugins or specific plugins. The activated plugins will be available from the Plugins menu (on the **Setup** ribbon).&nbsp;

| ![Image](<lib/admon-plugin.png>) | *Most of these plugins are licenced under commercial licences. You need to purchase a licence on which a registration code and instructions will be mailed to you to unlock the full features of these plugins. Multi-User and Plugins need to be registered in **Plugins&nbsp; → Tools → Register plugins** menu (**Setup** ribbon).* |
