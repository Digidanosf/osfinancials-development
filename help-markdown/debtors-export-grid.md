# Export Debtor accounts - Export grid

***

The **Export** button of the sidebar of the "*Debtor accounts*" list screen, will create a text file (tab delimited), which will open in your system's default Microsoft Excel (if installed on your system).&nbsp;

# Export options

**To export debtor (customer / client) accounts:**

On **Debtors** (**Default** ribbon).&nbsp;

| ![Image](<lib/admon-note.png>) | *Select the Filter and search options in the [Filter column*](<debtor-filter-search-options.md>) *and click on the **Filter** button to include only those filter criteria to be included in the Export file.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-important.png>) | *By default, the export file will include all the debtor (customer / client) accounts (excluding disabled accounts).*&nbsp; *To include disabled debtor (customer / client) accounts in the export file, you need to select the "Show disabled" option, and click on the **Filter** button.* |
| --- | --- |


&nbsp;

Click on the **Export** button.

![Image](<lib/debtors-export-grid.png>)

&nbsp;

The "*Save As*" screen is displayed:

![Image](<lib/debtors-export-grid-save-as.png>)

&nbsp;

Enter a name to save the file.&nbsp;

| ![Image](<lib/admon-note.png>) | *"**Save as type :**" options will be empty. A text (\*.txt) file type will automatically be created with your file name.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-warning.png>) | *If you do not enter a unique name in the selected folder, and save the export.txt file name, you may replace existing (previously exported) files.*&nbsp; *If an existing exported file is opened, and you are trying to save a export file with the same name, a similar error message as the following will be displayed:* *Cannot create file "F:\\exports\\your-file-name.txt". The process cannot access the file because it is being used by another process.* |
| --- | --- |


&nbsp;

Click on the **Save** button.&nbsp;

# Example : Exported text file - LibreOffice Calc

If the exported text file does not open in Microsoft Excel; or if you do not have Microsoft Excel installed on your system, you may open the file with OpenOffice, LibreOffice, Ultra Office, etc.&nbsp;

Locate the exported text file in your System's File Explorer.

In this example, Open with \> [LibreOffice Calc](<https://www.libreoffice.org/discover/calc/> "target=\"\_blank\"") is selected. The "*Text Import*" screen is selected.&nbsp;

![Image](<lib/debtors-export-grid-libre-office-open.png>)

| ![Image](<lib/admon-important.png>) | *If LibreOffice and not [LibreOffice Calc*](<https://www.libreoffice.org/discover/calc/> "target=\"\_blank\"") *is selected, the exported text file will open in LibreOffice Writer as a text file.* |
