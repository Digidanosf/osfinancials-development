# Context menu - Create account from document details

***

This documents context menu option allows you to create a new debtor (customer / client) account for unposted sales documents (i.e. Invoices, Credit notes or Quotes) or an new creditor (supplier / vendor) account for unposted purchase documents (i.e. Purchases, Supplier returns or Orders) using the document details.&nbsp;

This may be necessary if a Document is imported and the account does not exist.&nbsp;

# Create a new debtor (customer / client) account using the document details

**To create a new debtor (customer / client) account using the document details:**

1. Select the Invoices, Credit notes or Quotes document type.
1. Select the document to create an account for.
1. Right-click and select **Documents → Create account from document details** option on the context menu. The "*Debtor accounts"* edit form with the details on the document header is displayed:

![Image](<lib/documents-context-menu-create-debtor-details.png>)

4. Enter a "*Debtor code*" and enter or edit the fields, as necessary.&nbsp;

| ![Image](<lib/admon-important.png>) | ***Debtor code** - Enter a prefix followed by 5-digits. In this example, the Debtor code is prefixed with "1".&nbsp; Creditor codes are prefixed with "2" in this documentation.*&nbsp; *This is important to distinguish accounts for debtors (customers / clients) from accounts for creditors (suppliers / vendors). In some features and on some screens and reports this makes it easier to identify the account type.* &nbsp; |
| --- | --- |


5. Click on the **OK** or **Save** button. A confirmation message will be displayed.

*Create this account?"*

6. Click on the **Yes** button to continue.&nbsp;

| ![Image](<lib/admon-note.png>) | *Once the New account is saved the document will be listed in the [Documents*](<debtor-tab-documents.md>) *tab of the new debtor (customer / client) account.* |
| --- | --- |
