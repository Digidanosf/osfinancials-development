# Start / Launch osFinancials5

When you first install the full version of osFinancials5 or install updates for it, and you select the "*Launch osFinancials 5*" option on the "*Completing the osFinancials 5 Setup Wizard*" screen, osFinancials5 will automatically start up. However, after that initial launch, you will need to manually launch osFinancials5 each time you want to use it.

**To start osFinancials5:**

1. To start osFinancials5, double click on the ![Image](<lib/icon-shortcut-osf5-1.png>) icon created for you on your Desktop (if the "*Create a desktop*" icon option was selected during the installation).

**Or** select **osFinancials5** on the **Start menu - All apps (Programs) → osFinancials5** (or **Pinned**, if you have already pinned **osFinancials5** to the Start.

| ![Image](<lib/admon-tip.png>) | ***Pin osFinancials5 to the Quick access area (Windows 11):*** ![Image](<lib/osfinancials5-pin-to-start-windows-11.png>) *After installation, you may need to add the osFinancials5 to the Quick access area of the **Start** button.*&nbsp; *The **osFinancials5**&nbsp; icon should be displayed as Recommended or Recently added. On the osFinancials5 icon, right-click and select Pin to Start.*&nbsp; *You may also select to add **osFinancials5** to the taskbar (Pin to taskbar).*&nbsp; |
| --- | --- |


2. Once osFinancials5 is started, the osFinancials5 splash-screen will display the version number and a progress indicator.

![Image](<lib/splash.png>)

3. Once you have opened osFinancials5, the **Start** ribbon tab is displayed.
3. Click on the **Reports** ribbon to load the reports. &nbsp;

| ![Image](<lib/admon-note.png>) | *At first only the items under the **Start** ribbon tab is available. Before you can select any of the other ribbon tabs or ribbon options, you will need to “[Open a Set of Books*](<open-set-of-books.md>)*” (Click **Open** on the **Start** ribbon).* |
| --- | --- |
