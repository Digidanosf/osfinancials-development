# Tax list debtor with taxid (Reports menu)

***

You may print the "*Tax list debtor with tax id*" report, to check your tax transactions of posted posted documents (i.e. invoices and credit notes).&nbsp;

This report will display a summary of the totals of the invoices and credit notes for each debtor account for a selected period or date(s).

| ![Image](<lib/admon-important.png>) | *Batch transactions are excluded from this report.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-important.png>) | *This report only includes document transactions which are posted. Unposted documents will be **NOT** be included in this report.* *To view a list of unposted documents, which is not updated to the ledger, go to **Input → [Checking unposted items***](<checking-for-unposted-items.md>) *(**Default** ribbon).* |
| --- | --- |


# Tax list debtor with tax id. report options

**To print Tax list debtor with tax id reports:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Tax**.&nbsp;
1. Select "*Tax list debtor with tax id*", if not already selected.

![Image](<lib/reports-tax-list-debtor-with-tax-id-options.png>)

3. **Period** - Select the period on the **Per period** tab. You may click on the **Free selection** tab to select a specific date or a range of dates.
3. **Sequence** - Select "*Debtor code, Tax reg. no.*" or "*Company reg. no.*" to print the report detail by the selected sequence (sort order).
3. Click on the **OK** button.&nbsp;
