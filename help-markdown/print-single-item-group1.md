# Reporting group 1&nbsp; - Stock grid reports&nbsp;

The [Reporting group 1](<stock-item-reporting-groups.md>) for stock items, will list the Stock code, Stock description, Quantity on hand as well as Selling price 2 for each stock item. These stock items will be grouped and listed for each Stock reporting group 1 and the Unit description. &nbsp;

| ![Image](<lib/admon-tip.png>) | *You may select the "[**All Selling prices***](<report-stock-item-listing-sellin.md>)*" or the "[**Stock item listing***](<report-stock-item-listing1.md>)*" report on the **Reports → Stock →** **Stock item listing** menu (**Reports** ribbon) to print all three (3) selling prices.&nbsp; You may select a specific stock item or a range of stock items; or all stock items linked to a specific Stock reporting group 1.*&nbsp; &nbsp; |
| --- | --- |

# Print single item (context menu)&nbsp;

Prints directly to default printer and not to the Reportman preview screen.&nbsp;

In this example, the system's default printer is set to "*Microsoft print to PDF*"&nbsp; where you need to enter a file name to save the file print as "*PDF Document (\*pdf)*" file type on the " *Save Print Output As*" screen.
