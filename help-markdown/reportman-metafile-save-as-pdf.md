# Reportman output - Save metafile as PDF

***

"***PDF***" is the acronym for "*Portable Document Format*". The "*PDF*" file format is generally used to save files that cannot be modified. The "PDF files can be saved as a hardcopy of reports and document layout files can be saved as a hardcopy for audit and record purposes.&nbsp;

PDF files can be easily shared across all devices (e.g. desktops, laptops, mobile devices (tablets, cellular phones, etc.). It is also compatible and readable with the default pdf app or pdf reader app on all operating system platforms (e.g. Windows, Linux, Mac, Android, etc.).&nbsp;

You may also open saved PDF files on your system, in the your system's associated default Web browser , e.g. Google Chrome, Firefox, Microsoft Edge, etc.&nbsp;

PDF files may easily be sent via e-mail, read, printed on demand according to your own requirements; or the recipient's requirements. &nbsp;

# What apps or tools can be used to open pdf files?

A list of tools and apps that can be used to open PDF files, along with their cross-compatibility across different operating systems:

1. **Desktop Applications**
- &nbsp;

  - Adobe Acrobat Reader: Widely used and available for Windows, macOS, and Linux.
  - Foxit Reader: A fast and lightweight PDF reader available for Windows, macOS, and Linux.
  - Nitro PDF Reader: Available for Windows, it offers features like PDF creation, editing, and conversion.
  - Sumatra PDF: A free, open-source PDF reader for Windows.
  - PDF-XChange Editor: Available for Windows, it offers advanced editing and annotation features.

2. **Web Browsers**
- &nbsp;

  - Google Chrome: Can natively open and view PDF files. Available for Windows, macOS, Linux, Android, and iOS.
  - Mozilla Firefox: Can natively open and view PDF files. Available for Windows, macOS, Linux, Android, and iOS.
  - Microsoft Edge: Can natively open and view PDF files. Available for Windows, macOS, Android, and iOS.
  - Safari: Can natively open and view PDF files. Available for macOS and iOS.

3. **Online Tools**
- &nbsp;

  - Google Drive: Can open and view PDF files. Accessible via any modern web browser.
  - Dropbox: Can open and view PDF files. Accessible via any modern web browser.
  - Smallpdf: An online PDF tool that allows you to view, edit, and convert PDF files. Accessible via any modern web browser.
  - PDFescape: An online PDF editor and viewer. Accessible via any modern web browser.

4. **Mobile Apps**
- &nbsp;

  - Adobe Acrobat Reader: Available for iOS and Android.
  - Google Drive: Available for iOS and Android.
  - Dropbox: Available for iOS and Android.
  - Microsoft OneDrive: Available for iOS and Android.
  - WPS Office: Available for iOS and Android, it includes a PDF reader and editor.

5. **Command Line Tools**
- &nbsp;

  - Poppler: A suite of free software tools for PDF manipulation, available for Windows, macOS, and Linux.
  - PDFtk: A command-line tool for working with PDF files, available for Windows, macOS, and Linux.

6. **Text Editors**
- &nbsp;

  - Visual Studio Code: With the right extensions, you can view PDF files in VS Code. Available for Windows, macOS, and Linux.
  - Sublime Text: Similar to VS Code, you can install extensions to view PDF files. Available for Windows, macOS, and Linux.

7. **Other Tools**
- &nbsp;

  - PDF.js: An open-source JavaScript library for rendering PDF files in web browsers. Accessible via any modern web browser.

This list should help you find the right tool for viewing and editing PDF files regardless of your operating system.

# Save metafile options

**To save reports or document layout files as a PDF file:**

1. On the "*Reportman print preview*" screen, select **Save** (**Ctrl+S**) ![Image](<lib/reportman-preview-icon-save.png>) - This will launch the “*Save metafile as*” screen.&nbsp;

![Image](<lib/reports-screen-save-metafile-as.png>)

2. Select the directory (folder) where you need to save this file, if necessary.&nbsp;
2. Enter a file name.&nbsp;
2. Select the "*PDF File - compressed (\*.pdf)*" (default option) or the "*PDF File – uncompressed (\*.pdf)*" file type.
2. Click **Save**.

| ![Image](<lib/admon-note.png>) | *You may select the compressed or uncompressed PDF file types* *"PDF File – compressed" (default) - The file size and quality may be reduced (smaller) for sending these files via e-mail, etc.*&nbsp; *"PDF File – uncompressed"*&nbsp; *In osFinancials5/TurboCASH5, the average file pdf file size and quality for reports and document layout files, saved to the PDF compressed or PDF uncompressed, is hardly noticeable.* &nbsp; |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *The default print (**Print** (**Ctrl+P**) option where the “[Print*](<reportman-output-printer.md>)*” screen is set to "Microsoft Print to PDF" option is much smaller in size (263K) than the "PDF File – compressed" size (51K) or "PDF File – uncompressed" size (59K) PDF file types "Save metafile as" option.* |
| --- | --- |


The PDF file format may be selected for all reports and document layout files to external recipients.
