# Purchases per product (Reports menu)

***

You may print the following two (2) reports to analyse the purchase and supplier return documents to analyse your purchases.&nbsp;

1. Purchases per Product - Stock group 1
1. Purchases per Product - Stock group 2

| ![Image](<lib/admon-important.png>) | *This report only includes Purchase and Supplier return documents which are posted. Unposted Purchase and Supplier return documents will be **NOT** be included in this report.* *To view a list of unposted documents, which is not updated to the ledger, go to **Input → [Checking unposted items***](<checking-for-unposted-items.md>) *(**Default** ribbon).* |

# Purchase analysis - Purchases per product - Stock group 1 report options

**To print Product by Stock group reports:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Purchase analysis → Purchases per product**.&nbsp;

![Image](<lib/reports-purchase-analysis-product-options.png>)

2. Select the following:
1) &nbsp;

   1) **Report** - Select the Purchases per product - Stock group 1 (default) or Purchases per product - Stock group 2 report.&nbsp;
   1) **Period** - Select the period on the **Per period** tab. You may click on the **Free selection** tab to select a specific date or a range of dates.
   1) **From... To...** - Select the stock code(s) to include in the report.&nbsp;
   1) **Reporting group 1** - This is optional and by default not selected (not ticked). You may select (tick) the reporting group(s) to include in the report.&nbsp;
   1) **Sequence** - Select "*Stock code* or *Stock description*" to sort the order in which you need to print the data. &nbsp;
   1) **Show details** - By default, this option is not selected. It will display the total quantities and cost prices of each stock item in posted purchase and supplier return documents. If this option is selected, it will display a list of document numbers, account code and description and the quantities and cost prices for each document. &nbsp;
   1) **New page per account** - If this option is selected (ticked), it will print the transactions for each account on a new page.

| ![Image](<lib/admon-note.png>) | *The Company details will not be printed as in the Page header. The company name is displayed in the Page footer.* |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *It is recommended that this option is only selected if the Show details option is selected.* |
| --- | --- |


3. Click on the **OK** button.&nbsp;
