# Supplier returns - Purchases (Reports menu)

***

This option allows you to print the following two (2) reports:

1. Supplier returns per period - Quantities - displays the Stock code, Stock description, Quantities and Cost prices (if applicable to stock items).&nbsp;
1. Supplier returns per period - Turnover - displays the Stock code, Stock description, Quantities and Cost prices.&nbsp;

| ![Image](<lib/admon-important.png>) | *This report only includes Supplier returns which are posted. Unposted Supplier returns will be **NOT** be included in this report.* *To view a list of unposted documents, which is not updated to the ledger, go to **Input → [Checking unposted items***](<checking-for-unposted-items.md>) *(**Default** ribbon).* |
| --- | --- |


# Supplier returns per period - Quantities report options

**To print Supplier returns per period - Quantities reports:**&nbsp;

1. On the **Reports** ribbon, select **Reports → Purchases → Supplier returns → Purchases**.&nbsp;

![Image](<lib/reports-purchase-supplier-return-purchase-opt.png>)

2. Select the period on the **Per period** tab. You may click on the **Free selection** tab to select a specific date or a range of dates.
2. Click on the **OK** button.&nbsp;
