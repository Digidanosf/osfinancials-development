# Clear/Reset

***

This option allows you to clear all transactions and balances in a Set of Books. By selecting this option, you will effectively reset the Set of Books, deleting all transactions and balances. However, you also have the option to retain all documents and set them to unposted status.

This process is similar to the [**Delete Set of Books**](<delete-set-of-books.md>) option from the **Start** ribbon, as all data related to transactions and batch entries will be permanently removed from the system.&nbsp;

The only difference is that if you use the **Global processes → Clear/Reset** option from the **Setup** ribbon, you will still be able to access the Set of Books from the [**Open Set of Books**](<open-set-of-books.md>) option on the **Start** ribbon. All the original setup data for your Set of Books, such as accounts, stock and selling prices, debtor and creditor accounts, contacts, reporting groups, etc., will still be available.

| ![Image](<lib/admon-warning.png>) | *It is highly recommended to create sufficient backups of your data before using this option. In case you accidentally use the **Global processes → Clear / Reset** option on the **Setup** ribbon, you will need to restore your Set of Books from the backup disks. Additionally, you may need to manually enter any transactions that were not included in your last backup until you have used the **Clear / Reset** option, unless you have not processed any transactions since your last backup.* |

| ![Image](<lib/admon-important.png>) | *Before clearing or resetting a Set of Books from your system, it is crucial to ensure that all necessary reports have been printed and retained, along with supporting documents such as slips, cheques, invoices, etc. This is essential to substantiate your profit from the business and ensure compliance with tax return requirements.*&nbsp; *As a small business owner, you must keep all necessary records for five (5) years, starting from the date of receipt of the last applicable tax return for audit and record purposes. Additionally, there may be other legislative and/or statutory requirements that apply to your specific business type and/or industry.* |

| ![Image](<lib/admon-warning.png>) | *It is essential to be absolutely certain before using the Clear / Reset option on a Set of Books, as this action will permanently delete all transactions, balances, stock quantities, and cost prices.* |
| --- | --- |


**There are two options to clear or reset a Set of Books:**&nbsp;

1. Clear / Reset all transactions and balances in a Set of Books. No transactions, documents or any balances will be available in the Set of Books.&nbsp;
1. To Clear / Reset all the transactions and and retain all sales documents (i.e. invoices, credit notes, quotes) and all purchase documents (purchases, supplier returns and orders). &nbsp;

**To Clear / Reset all transactions and balances:**&nbsp;

1. On the **Setup** ribbon, select **Global processes → Clear / Reset** menu. The following confirmation screen is displayed:&nbsp;

![Image](<lib/clear-reset-books-confirmation.png>)

2. If you are absolutely sure, tick the "*I have read this instruction and understand the action\!*" field. The **Yes** and **No** buttons will be available on this screen.

![Image](<lib/clear-reset-books-confirmation-yes-no.png>)

| ![Image](<lib/admon-note.png>) | *If you click on the **No** button, the process will be cancelled.*&nbsp; |
| --- | --- |


3. Click on the **Yes** button only if you are absolutely sure. Another confirmation screen is displayed:&nbsp;

![Image](<lib/clear-reset-books-confirmation-1.png>)

| ![Image](<lib/admon-warning.png>) | *On this screen you have the last chance to cancel the clearing process. If you click on the **Yes** button, the process will be terminated.* |
| --- | --- |


4. Click on the **No** button to clear / reset the Set of Books. The following confirmation screen is displayed:

&nbsp;![Image](<lib/clear-reset-books-confirmation-2.png>)

5. If you do not want to reset clear the quantities, etc. of stock items, click on the **No** button.&nbsp;

| ![Image](<lib/admon-warning.png>) | *If you click on the **Yes** button, all stock values will permanently be lost.*&nbsp; |
| --- | --- |


6. The following confirmation screen is displayed:&nbsp;

![Image](<lib/clear-reset-books-confirmation-3.png>)

7. If you click on the **Yes** button, all transactions, documents and balances in a Set of Books will be deleted. No transactions, documents or any balances will be available in the Set of Books.

| ![Image](<lib/admon-note.png>) | *If you click on the **No** button, all the transactions and balances will be cleared reset to zero. All sales documents (i.e. invoices, credit notes, quotes) and all purchase documents (purchases, supplier returns and orders) will be retained and be available in **Documents** (**Default** ribbon).* &nbsp; |
