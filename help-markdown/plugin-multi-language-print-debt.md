# Print documents in debtor (customer / client) language

## Set debtor (customer / client) language

**To select / set / change a debtors (customers / clients) preferred language:**

1. Click on the **Default** ribbon, select **Debtors**; or press the **F4** key.
1. Select the debtor (customer / client) account and click on the **Edit** button; or double-click on a selected debtor (customer / client) account.![Image](<lib/multi-language-debtor-set-language.png>)

&nbsp;

3. Select the Debtor's Language.  
3. Click on the **Save** or **OK** button.

| ![Image](<lib/admon-note.png>) | *On the **Delivery address** tab Layout file 1 (Invoice), Layout file 2 (Credit note) and Layout file 3 (Quote) is empty for this this debtor customer / client) account.*&nbsp; *The layout files , selected on the "Layout file" field in the **Setup → [Documents setup***](<documents-setup.md>) *on the **Setup** ribbon, for these document types, will be used for printing documents.* *If you have specific or your own custom layout files, you may select your own layout files.*&nbsp; |
