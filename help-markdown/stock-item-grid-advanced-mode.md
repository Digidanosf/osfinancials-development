# Stock list screen - Advanced mode

***

The stock item list screen has two (2) mode options available:&nbsp;

* **Default mode** option - This is a plain list view of the stock items on which the data in the columns is not editable.
* **Advanced mode** option - This mode will enable the following options:

  * The data in the columns on the stock item grid can be edited (like you would in a spreadsheet).
  * Add the [**Export / Import**](<stock-item-tab-import-export.md>), [**Adjust stock**](<stock-item-tab-adjust-stock.md>) and [**Settings**](<stock-item-tab-settings.md>) tabs. &nbsp;

By default, the "*Single line mode*" option is selected (ticked). The details for each of the stock items in the list will display in a single line.&nbsp;

If this&nbsp; option is selected, a column header "*Use your mouse to pull a column here to group on that column*" will be displayed and the similar data in columns can be grouped.

# Advanced mode : Single line mode - Selected

If the "*Advanced mode*" option is selected, the **Advanced mode** button's caption, will change to **Default**. It will enable the following options:

* The data in the columns on the stock item grid can be edited.
* Add the [**Export / Import**](<stock-item-tab-import-export.md>), [**Adjust stock**](<stock-item-tab-adjust-stock.md>) and [**Settings**](<stock-item-tab-settings.md>) tabs.

![Image](<lib/stock-single-line-selected-advanced.png>)

# Advanced mode : Single line mode - Not selected&nbsp;

The data for each of the stock items in the list can not be edited. You need to select a stock item and edit the details in the Stock item form.&nbsp;
