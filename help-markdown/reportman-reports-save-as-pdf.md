# Reportman reports - Save as PDF

***

| ![Image](<lib/admon-tip.png>) | *If you print any report or document layout file to the screen, you may [save the report as a compressed or uncompressed PDF file*](<reportman-metafile-save-as-pdf.md>) *from the "Reportman print preview" screen.* |
| --- | --- |


By default the "*Open in PDF*" option is not selected on the report options (parameter) screens.&nbsp;

If the "*Open in PDF*" option is selected on any report options (parameter) screen, it will print all reports on the [Reports menu](<reports-menu.md>) or [User reports menu](<user-reports-menu.md>) directly as a PDF file.&nbsp;

![Image](<lib/reports-print-direct-to-pdf.png>)

The PDF file will automatically be opened in your system's default PDF viewer or app.

![Image](<lib/reports-print-direct-to-pdf-printed.png>)

| ![Image](<lib/admon-note.png>) | *The file name will automatically be generates as your system's date / time stamp. In this example, the format of the timestamp is "YYYY/MM/DD/hour/minute/second".* |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *To deactivate this option, remove tick in the "Open in PDF" option on any report options (parameter) screen accessed from the [Reports menu*](<reports-menu.md>) *or [User reports menu*](<user-reports-menu.md>)*.* |
| --- | --- |
