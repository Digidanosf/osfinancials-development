# Let osFinancials5/TurboCASH5 help you to create a Set of Books - Download

***

osFinancials5/TurboCASH5 will guide you through the steps to create a new Set of Books with the "*Creation wizard*" in a matter of minutes.&nbsp;

It is a good idea to check that you have all the necessary information regarding your business at hand before starting to create your Set of Books.&nbsp;

Once you have the necessary information available, you may proceed to create your Set of Books. Should you not have all the information available at this stage, you may go back and enter or change some of the information at a later stage.&nbsp;

The following options to create a Set of Books using the "*Let osFinancials5-1 help you to create a Set of Books?" /* "*Let TurboCASH5-3 help you to create a Set of Books?" wizard*, is available:

* To create a Set of Books using a template on your system.&nbsp;
* To create a Set of Books downloading a template from osFinancials5.

**To create a Set of Books downloading a template:**&nbsp;

1. On the **Start** ribbon, you can select the **Create Set of Books** option or simply press the **F10** key on your keyboard.

![Image](<lib/books-creation-wizard-default-option.png>)

2. osFinancials5.1 automatically defaults to the option "*Let osFinancials help you to create a Set of Books?"* or in TurboCASH5-3 "*Let TurboCASH5-3 help you to create a Set of Books?"* as this is the recommended option for most users. You only need to check that it is selected.
2. Click on the **Next** button. The "*Which Chart of Accounts do you want to use?*" screen on which you may select a Set of Books (template) for your country, language. country or industry to download will automatically be launched, if you have an internet connection.&nbsp;
2. If you do not wish to download a Set of Books from the osFinancials website, you may close this online screen on the titlebar.

| ![Image](<lib/admon-important.png>) | *If you do not have an Internet connection, clicking on the **Online** button will automatically close. You may click on the **Browse** button to elect a Set of Books in your osFinancials5.1/TurboCASH5-3 installation folder.*&nbsp; ![Image](<lib/tc-5-3-create-boolld-default-03.png>) |
| --- | --- |


5. If you have an Internet connection, "*Which Chart of Accounts do you want to use?*" download screen will automatically be launched.&nbsp; ![Image](<lib/create-set-of-books-download.png>)

| ![Image](<lib/admon-note.png>) | *osFinancials5/TurboCASH5 supports 79 different languages and counties / tax regimes languages liked to 87 Sets of Books (databases) which may be downloaded and customised to meet your requirements.*&nbsp; *If the set of books for your language and or country, is not available in the books directory, you may select any of the available Firebird [Sets of Books and linked languages available as a free download*](<sets-of-books-download-templates.md>)*.*&nbsp; |
| --- | --- |


6. Select the Chart of Accounts (Set of Books) and click on the **Download** button. Once the download, is completed, the "*Save as*" screen will display the Name of the selected downloaded Set of Books.

![Image](<lib/books-creaye-download-save-as.png>)

7. Enter a unique name for the Set of Books.

| ![Image](<lib/admon-important.png>) | *If the name for the new Set of Books already exists, an information message is displayed:*&nbsp; *That Set of Books already exists\! Please re-enter\!* *Enter a new name for the Set of Books.* |
| --- | --- |


8. Click **Save**. The following information message is displayed:

*Set of Books created\!*

9. Click **OK**. The Set of Books will automatically be opened.

| ![Image](<lib/admon-note.png>) | ***Year-end confirmation***&nbsp; *Should you open the Set of Books, and the date of the end of the year, is in the past, a confirmation message will be displayed, for example;*&nbsp; *This year Ends on 2022/02/28* *Today 2022/04/04* *Warning, this date does not fall into any of this or last year's reporting dates\!* *Do year end?* *You need to&nbsp; adjust the Start of the financial year in one of the following options on the **Setup** ribbon:* &nbsp; *Setup **→ [**Reporting dates*](<setup-reporting-dates.md>) *(if no transactions were processed for the Set of Books).*&nbsp; [*Do year end*](<do-year-end.md>)*.*&nbsp; |
| --- | --- |


10. To complete and configure the downloaded Set of Books for your needs, it is recommended that you first check enter, add and select the necessary information and settings:&nbsp;
1) &nbsp;

   1) [**Accounts**](<ledger-accounts.md>) - Check the Chart of Accounts. Add any accounts required. You may also Disable or Delete any accounts that you do not require. Specific attention is required with the following account types:
1. &nbsp;

   1. &nbsp;

      1. [**Tax accounts**](<tax-accounts-add-edit.md>) - Check that the correct tax codes and tax percentages are applicable to your country. You may need to change the descriptions, add tax codes, change the tax percentages, etc.
      1. [**Bank accounts**](<bank-accounts-new.md>) - Rename an existing bank account, check your Payments and Receipt batches linked to the bank account. Add any additional bank accounts or delete unnecessary bank accounts.&nbsp;
1) &nbsp;

   2) [**Company info**](<setup-company-info.md>) - Enter all your business' details such as the address, e-mail address, contact numbers and Company and Tax Registration numbers, which you need to display on all documents, reports, and lists. You may also select whether your default output option for documents, reports and lists are on: screen, printer, file, e-mail or fax. You may also load a logo picture to be displayed on your reports or documents, and set your Windows operating system's default date and currency formats.
   2) [**Reporting dates**](<setup-reporting-dates.md>) - This option allows you to set up the number of reporting dates required, and the start date of your financial year period. The periods or dates will be displayed in the format in which you have set your Windows Operating System's default date on the previous screen.
   2) [**Batch types**](<batch-types-setup.md>) - This option allows you to create, delete an existing batch type, or change the name of an existing batch type in which you wish to enter and update or post your transactions to the ledger.
   2) [**Groups**](<setup-groups.md>) - This option allows you to set up to two Reporting groups for your General ledger, Debtors, Creditors accounts, Stock and Document groups.
   2) [**Documents setup**](<documents-setup.md> "TC4:Setup Documents") - This option allows you to set up your documents Invoices, Credit notes, Quotes, Orders, Purchases and Supplier return documents, you wish to hand to customers (debtors) or suppliers (creditors) when you sell, or quote, or order and purchase, stock items and services.&nbsp;
   2) [*Salesperson*](<setup-salespersons.md>) - This option allows you to enter or add the salespersons who will be processing documents.
   2) [**Stock items**](<stock-inventory.md>) - Add your stock items (inventory items / products).&nbsp;
   2) [**Debtors**](<debtor-accounts.md>) - Add your debtor (customer / client) accounts.
   2) [**Creditors**](<creditor-accounts.md>) - Add your creditor (supplier / vendor) accounts.
