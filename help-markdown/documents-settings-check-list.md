# Documents settings - Check list

***

The setup of documents in osFinancials5/TurboCASH5 is a one-time process that involves several steps:

1. Check the document numbers, messages, and set layout files on the **Setup → [Documents setup**](<documents-setup.md>) menu of the **Setup** ribbon.
1. Select the "*Append notes below item*" option (default setting) or choose to add comments before a selected item on documents and layout files.

| ![Image](<lib/admon-note.png>) | *By default, [comments*](<document-entry-insert-transactio.md>) *is inserted below a selected item. The option "[**Append notes below item***](<doc-setting-append-items-below.md>)*" (**Setup → Documents setup** (**Setup** ribbon)) is added to set the preference if [comments*](<document-entry-insert-transactio.md>) *should be added before, or below (default option), a selected item when processing documents and printing comments on layout files.*&nbsp; *The default "**Append notes below item**" setting is selected (ticked). It will display on document lines and document layout files as follows:*&nbsp; *Item*&nbsp; *Remark*&nbsp; *Comment*&nbsp; *If the "**Append notes below item**" setting is not selected. It will display on document lines and document layout files as follows:*&nbsp; *Comment*&nbsp; *Item*&nbsp; *Remark*&nbsp; |
| --- | --- |


3. Add the Cost of sales reporting group under Account group 1 (Reporting group 1) on the **Setup → [Groups**](<reporting-groups-setup-accounts.md>) menu of the **Setup** ribbon. Also, add reporting groups 1 and 2 for [Document groups](<document-reporting-groups.md>) and reporting groups 1 and 2 for [Stock groups](<stock-item-reporting-groups.md>), where applicable.
3. Add Salespersons on the **Setup → [Salesperson**](<setup-salespersons.md>) menu of the **Setup** ribbon.
3. Add a Cost of sales batch (journal) on the **Setup → [Batch types**](<batch-types-setup.md>) menu of the **Setup** ribbon (only to be used with Cost of sales settings for the "*Default*" stock item type).
3. Set up unit prices, selling price descriptions, and default selling price on the **Setup → [Stock information**](<stock-information.md>) menu of the **Setup** ribbon.
3. Create a Cost of sales account for Trading stock on the **Setup → [Accounts**](<ledger-accounts.md>) menu of the **Setup** ribbon.
3. Create a Sundry Supplier account under the [**Creditors**](<creditor-accounts.md>) section on the **Default** ribbon.
3. Create a Cash Customer and linking debtors to a Salesperson under the [**Debtors**](<debtor-accounts.md>) section on the **Default** ribbon.

By completing these steps, you will have all the necessary settings in place to efficiently manage and process documents in osFinancials5/TurboCASH5.
