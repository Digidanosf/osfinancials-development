# Create backorders to deliver

***

When processing documents, and there are any Backorders outstanding, a list of Backorders will be displayed.&nbsp;

![Image](<lib/document-entry-backorder-selection.png>)

&nbsp;

You may select the item on the list and the quantities (in the Items) will be added to the document lines and will be removed from this list.&nbsp;

# Backorders to deliver - Invoices

If an outstanding backorder exists for a debtor (customer / supplier account), you may use this option to create a new Invoice for the outstanding backorders to deliver.&nbsp;

&nbsp;

**To create backorders to deliver:**

1. Select the "*Invoices*" document type.
1. Select the document to create backorders to deliver.
1. Right-click and select **Documents →** **Backorders to deliver** option on the context menu. A new Invoice will be created.
1. You may need to edit the Invoice.&nbsp;

![Image](<lib/docs-backorders-to-deliver.png>)

&nbsp;

| ![Image](<lib/admon-warning.png>) | ***WARNING - Input Tax / Output tax*** *In the case of Invoices, the Input tax is used instead of the Output tax. You need to check this, and change it to the Output Tax.*&nbsp; |
| --- | --- |


The Invoice will reflect the Selling prices and the quantities as it was on the Backorders. The stock item will be removed from the Backorders screen as well as on the [**Backorders**](<stock-item-tab-backorders.md>) tabs for the Stock item and the [**Backorders**](<debtor-tab-backorders.md>) tab of the Debtor (customer / client) account.

You may click on the Backorder icon and select another stock item to add to the Invoice.

# Backorders to deliver - Quotes&nbsp;

| ![Image](<lib/admon-warning.png>) | ***WARNING - INVOICE under Quotes document type*** *This process is not recommended, as it will create an Invoice on the Quotes document type.* ![Image](<lib/documents-create-backorders-to-deliver-quote.png>) |
| --- | --- |


&nbsp;

**To create an Invoice from the Quotes document type for Backorders to deliver:**

**To create backorders to deliver:**

1. Select the "*Quotes*" document type.
1. Select the document to create backorders to deliver.
1. Right-click and select **Documents →** **Backorders to deliver** option on the context menu. A new Invoice will be created in the Quotes document type.
1. You may need to edit the Invoice / Quote.&nbsp;

![Image](<lib/documents-create-backorders-to-deliver.png>)

| ![Image](<lib/admon-warning.png>) | ***WARNING - Input Tax / Output tax*** *In the case of Invoices, the Input tax is used instead of the Output tax.&nbsp; You need to check this, and change it to the Output Tax.*&nbsp; |
| --- | --- |


The **Add to backorder** icon is added (after the Backorder icon).

![Image](<lib/quote-backorders-select.png>)

This will list the Outstanding items on Backorder. You may click on the **list** tab to view a list of the debtor (customer / client) accounts.&nbsp;

![Image](<lib/quote-backorders-select-list-tab.png>)
