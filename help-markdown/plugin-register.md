# Register Plugins

Upon purchasing plugin licences, you will receive the licence codes via email along with instructions for activation. Simply add the licence codes provided to access the licenced version of the plugin.

**To register plugins**:&nbsp;

1. On the **Setup** ribbon, select **Plugins → Tools → Register plugins**.&nbsp;

![Image](<lib/plugins-register.png>)

2. Enter or copy the provided licence code into the designated field.
2. Click on the **Add** button to register the plugin. The registered plugin will be added to the "*Registered*" list.
2. After registering, click **Close**. To ensure the registered plugin takes effect, you may need to close and restart osFinancials5/TurboCASH5.
