# Account groups (Setup - Groups)

***

You have the option to create two (2) reporting groups (Account group 1 and/or Account group 2) that can be used to categorise the various types of accounts in your chart of accounts, including [ledger](<accounts-edit.md>), [bank](<bank-accounts-edit.md>) and [tax](<tax-accounts-add-edit.md>) accounts. Each account must be linked to one of these reporting groups in order to be properly classified.

These reporting groups can be linked to a [financial category](<financial-categories.md>) that will help structure your options for viewing, analysing, and printing financial reports such as the [Trial balance](<ledger-analyser-trial-balance.md>), [Income statement](<ledger-analyser-income-statement.md>) and [Balance sheet](<ledger-analyser-balance-sheet.md>),&nbsp;

**To set reporting groups for General ledger accounts:**&nbsp;

1. On the **Setup** ribbon, select **Setup → Groups**. The "*Groups* (*Account group 1)"* default screen is displayed:

2. If you wish to rename the Reporting group name - Account group 1, over-type the description with your new description. This will change the description of the button for Account group 1.
2. To add a new Group to the root directory or list, select the "*REPORTING GROUP ROOT*" on the list and click on the **New** button.
2. Enter the name of your Account group and click on "*REPORTING GROUP ROOT*" on the list.&nbsp;
2. Repeat the process until you have entered all the Account groups, etc.&nbsp;
2. If you need to add Reporting group 2 for Account groups, click on the **Account group 2** button and repeat the process as for Account group 1.
2. Click on the **OK** button to save your changes and close (exit) this screen.
