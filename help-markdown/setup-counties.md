# Countries

***

ISO CODE2 and ISO CODE3 are codes used to represent countries and regions. ISO CODE2 is a two-letter code, and ISO CODE3 is a three-letter code.&nbsp;

Add and manage the ISO CODE2 and ISO CODE3 as well as ICL CODES for each country. See [Country codes](<https://www.iban.com/country-codes> "target=\"\_blank\"").

![Image](<lib/setup-countries.png>)

ISO CODE2 and ISO CODE3 are standardized codes used to represent countries and regions, created by the International Organization for Standardization (ISO). Here's a brief overview of their uses:

* &nbsp;

- **ISO CODE2**: This is a two-letter code (e.g., "US" for the United States, "JP" for Japan) used in various international contexts, such as internet domains (.us, .jp), vehicle registration plates, and international shipping.
- **ISO CODE3**: This is a three-letter code (e.g., "USA" for the United States, "JPN" for Japan) used in more specific contexts like international data exchange, customs declarations, and financial services.
- **ICL CODE** This is likely industry-specific codes that complement the ISO codes for more detailed categorization and management.

These codes ensure consistency and clarity in global communication. Managing these codes in software like osFinancials5 and TurboCASH5 helps standardize country and region identifiers across various modules and reports.
