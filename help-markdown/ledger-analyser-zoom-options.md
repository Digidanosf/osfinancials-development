# Zoom in / Zoom out + / - buttons 

***

**Zoom in / Zoom out** **+** / **-** buttons added to resize (increase / decrease the font) the reports (i.e. [Trial balance](<ledger-analyser-trial-balance.md>), [Balance sheet](<ledger-analyser-balance-sheet.md>) and [Income statement](<ledger-analyser-income-statement.md>)) on screen.&nbsp;

These are similar to the **Zoom +** (Zoom in) and the **Zoom -** (Zoom out) buttons on the **View** ribbon on Spreadsheets.

* [**Ledger analyser**](<ledger-analyser.md>) - Available for Trial balance, Balance sheet and Income statement on screen. Not implemented on the other report types (i.e. [Column balances](<ledger-analyser-column-balances.md>), [Standard column balances](<ledger-analyser-standard-ccolumn.md>), [This year vs last year](<ledger-analyser-this-last-year.md>), [Budget vs actual](<ledger-analyser-budget-actual.md>)).&nbsp;
* [**Spreadsheet reports**](<spreadsheet-reports.md>) (**Reports** ribbon) - All spreadsheet reports.
* [**Data Import / Export** - **Spreadsheet import** button](<export-import-stock.md>) on **Import / Export** tab of Stock items (**Default** ribbon).
* **Setup → [Accounts**](<accounts-export-import-excel.md>) - Spreadsheet icon (**Setup** ribbon).
* [**Batch entry** - **Spreadsheet import** icon](<batch-export-import-transactions.md>) on all batches entry screens.&nbsp;
* [**Spreadsheet sales**](<spreadsheet-sales.md>) (**Default** ribbon).
