# Add Multi-language users

***

By adding users and selecting different system language, and User language for each user, you may login to different languages.

| ![Image](<lib/admon-note.png>) | *The MULTI-LANG-DEMO Books is included in the ...\\Bin\\Tutorials\\ folder. To use this login with **e** (Username and Password) for English and **a** (Username and Password) for Afrikaans.* |
| --- | --- |


**To add Multi-language users:**

1. Go to **Setup → Access control** menu.
1. Click on the **Add** button and rename the User.
1. Enter a Password for the User.
1. Click on the **Others** tab.

&nbsp;![Image](<lib/multi-language-user1-language-setting.png>)

5. Select the System Language.  

| ![Image](<lib/admon-note.png>) | *System Language will list all the available languages in the **Start → Switch Language** menu.* *Any language may be customised / edited for your specific requirements on the **Setup → Tools → Customise Language** menu.* *The available languages is stored in the ... \\ bin \\ languages folder of your osFinancials5/TurboCASH5 installation directory. The default language is selected "System lang" field. This is the language in which osFinacials5 will open the Set of Books.* &nbsp; |
| --- | --- |


6. Select the User Language.

| ![Image](<lib/admon-note.png>) | *Language will list all languages created / added in **Setup → Groups** - **Languages** tab.*&nbsp; |
| --- | --- |


7. Click on the **Apply** button.
7. Click on the **Add** button to add the next User and rename the User.
7. Enter a Password for the User.
7. Click on the **Others** tab.

&nbsp; ![Image](<lib/multi-language-user2-language-setting.png>)

11. Select the System Language for the user interface. The language set in the "*System lang*" field, will be displayed as the osFinancials in interface. In this example, the osFinancials user interface will be displayed in Afrikaans if User "a" opens the Set of Books. &nbsp;
11. Select the User Language.
11. Click on the **Apply** button.
11. Click on the **Close** button.&nbsp;

| ![Image](<lib/admon-note.png>) | [***Access control - log report***](<access-control-log-report.md>) *- It will list when the users logged in and out, on which device, etc.*&nbsp; *Important functions executed from the **Tools → Global processes** menu (**Setup** ribbon), will also be listed to display the date and type as well as which user and on which computer these important actions were performed. The **Type** of actions is as follows:*&nbsp; ***OPEN YEAR - Open new year&nbsp; ([Do year end***](<do-year-end.md>)***)**.* ***REVBATC - [Reverse posted batch***](<batch-posted-reverse-cancel.md>)&nbsp; *- Reverse batch\! and the Batch ID.* ***REVDOC - [Reverse posted document***](<reverse-cancel-posted-documents.md>) *- Reverse doc\! and the Batch ID.* |
