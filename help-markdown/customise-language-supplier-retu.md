# Customising document type terminology - Supplier return

***

The language associated with each Set of Books determines the specific labels used to cater to different countries or tax regulations. In most languages, the term *'Supplier return'* or its equivalent will be displayed. For instance, en-UK and en-South Africa use terminology related to *'Supplier return'*, while in South Africa, Afrikaans refers to *'Verskafferretoer'* or *'Verskafferretoere'*.

These purchase-related labels are visible within the osFinancials5/TurboCASH5 interface when you access a Set of Books. They are also present in reports and document layout files generated by the system.

To adapt the *'Supplier return'* document type terminology to suit your country, jurisdiction, and industry, you can customize and modify the *'Supplier return'*-related labels via the "*Customize Languages*" screen. This screen is accessible through the **Tools → [Customise Language**](<customise-language.md>) menu on the **Setup** ribbon.

You may also find it helpful to utilize the search options available. This enables you to search for a specific label either in the currently active language file you're editing or in the default "*English*" language file. This assists you in identifying and modifying specific text to better suit your requirements.
