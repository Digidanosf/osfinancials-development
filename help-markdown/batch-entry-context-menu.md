# Batch entry screen - Context menu

***

In addition to the standard account lookup in the "*Account*" column, you may use the right-click context menu and select options to select accounts in batches, to edit an account, etc.&nbsp;

You may also copy and paste the "*Contra account, Date*, *Reference*" and "*Description*" from one line to another.

![Image](<lib/batch-entry-context-menu-paste.png>)

&nbsp;

The options are as follows:&nbsp;

1. [**Lookup document**](<batch-entry-menu-lookup-docummen.md>) - In addition to the lookup function in the "*Accounts / Debtors / Creditors*" column in the ""*Batch entry*" screen you can choose to search for a specific document. You can use the advanced search and filter options in the "Document entry" screen to select a specific document.&nbsp;
- &nbsp;

  - Once you have selected a document (invoice or credit note), the debtor's (customer/ client) account associated with the document (invoice or credit note) will appear in the "*Accounts / Debtors / Creditors*" column of the "*Batch entry*" screen, be selected.&nbsp;
  - Once you have selected a document (purchase or supplier return), the creditor (supplier / vendor) account associated with the document (purchase or supplier return) will be selected in the "*Accounts / Debtors / Creditors*" column of the "*Batch entry*" screen .&nbsp;

2. [**Lookup document with amount**](<batch-entry-menu-lookup-documme1.md>) - The option is similar to the [**Lookup document**](<batch-entry-menu-lookup-docummen.md>) context menu option.
2. [**Lookup debtor**](<batch-entry-menu-lookup-debtor.md>) - In addition to the lookup function in the "*Account*" column in the "*Batch entry*" screen, you can choose to lookup a debtor (customer / client) account from the "*Debtor accounts*" screen. On the "*Debtor accounts*" screen, you can use the filter and search options to select a specific debtor (customer / client)&nbsp; account.
2. [**Lookup creditor**](<batch-entry-menu-lookup-creditor.md>) - In addition to the lookup function in the "*Account*" column in the "*Batch entry*" screen you may choose to lookup a creditor (supplier / vendor) account from the "*Creditor accounts*" screen. On the "*Creditor accounts*" screen, you can use the filter and search options to select a specific creditor (supplier / vendor) account.&nbsp;
2. [**Copy**](<batch-entry-menu-copy-paste.md>) - Copy the "*Contra account, Date, Reference*" or "*Description*" from one transaction in a batch to another transaction.
2. [**Select from account open item list**](<batch-entry-menu-select-open-ite.md>) - You may select a debtor (customer / client) or creditor (supplier / vendor) account on the "*Batch entry*" screen (right-click on selected transaction) and select the "*Select from account open item list*" option. The "*Open item selection*" screen will list the unlinked transactions as well as transactions with outstanding amounts for the relevant debtor (customer / client) or creditor (supplier / vendor) account. You then choose to link this transaction to a specific outstanding transaction on this open item list.
2. [**Edit accounts**](<batch-entry-menu-edit-accounts.md>) - If you have selected a debtor (customer / client) or a creditor (supplier / vendor) account, the details (ie Account Code, Name and Address) will be displayed on the top left of the "*Batch entry*" screen. If it does not match the source document, or something has changed, you can choose to edit the account.
