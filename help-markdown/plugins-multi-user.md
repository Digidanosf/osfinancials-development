Do you want to use osFinancials5 or TurboCASH5 with multiple users at the same time? That is possible with the Multi-workstation plugin. The number of users is unlimited, as long as the hardware sustains. To use the Multi-workstation a network with a shared folder is required.&nbsp;

The plugin doesn't need to be downloaded. It will be activated by a code.

You will receive the code(s) and instructions through mail within two workdays after payment.

Using osFinancials or TurboCASH with multiple users simultaneously via the Multi-workstation plugin sounds like a convenient solution for collaborative financial management.&nbsp;

Unlimited user capability, contingent on hardware capabilities, provides scalability. Activation via a code simplifies the process, and instructions delivered via mail streamline setup.&nbsp;

This approach ensures efficient utilization of resources while accommodating diverse user needs.

No more limitations - choose any PC to serve as your network server, and experience the freedom of collaborative finance management. The Multi-User Plugin for osFinancials5/TurboCASH5 offers the capability to configure and set up a network server to accommodate multiple users. However, to unlock the full functionality for multi-user access, purchasing a multi-user licence and registering osFinancials5/TurboCASH5 as a multi-user product is necessary.

Documentation is provided for installing and configuring multi-user setups on both Windows and Linux servers, which can assist users in setting up their network environment effectively.

It's important to note that the multi-user functionality is limited in this edition, indicating that additional licensing is required to fully utilize it. Therefore, acquiring a multi-user licence is essential for organizations or teams requiring simultaneous access to osFinancials5/TurboCASH5.
