# Append notes below item

***

The **F9:Comments** is, by default, inserted after a selected stock item.&nbsp;

The "***Append notes below item***" (**Setup → Documents setup** (**Setup** ribbon)) option is added to set the preference if comments should be added before, or after, a selected stock item when processing documents and printing comments on layout files.&nbsp;

| ![Image](<lib/admon-important.png>) | *If you change this "Append notes below item" setting when a Document entry form is open or active, you may need to go to* **Tools → [Close active forms**](<close-active-forms.md>) (**Setup** ribbon)*, before this setting will be activated.* |
| --- | --- |


# Insert comments below a selected item

"Append notes below item" = TICKED (selected). This is the default setting.

![Image](<lib/documents-setup-append-below-item-on.png>)

1. Item&nbsp;

# Insert comments before a selected item

"*Append notes below item*" = NOT TICKED.&nbsp;

![Image](<lib/documents-setup-append-below-item-off.png>)
