# Bulk time entry

***

The bulk time entry feature in the Calendar of osFinancials5/TurboCASH5 allows you to efficiently add multiple time entries for a salespersons and a specific debtor (customer/client) account. This feature is particularly useful when scheduling recurring appointments or events.

To use bulk time entry, you can select the desired dates and set the time for a salesperson's hours event on the Calendar. For example, if a debtor (customer/client) has booked a training course or workshop for a week, you can easily schedule appointments for each day of that week. These bulk time entries will create multiple appointments on the calendar as hours-type events for the selected dates and time slots.

Once the bulk time entries are created, you have the option to generate quotes for these events. Quotes can be customised and edited as needed before sending them to the debtor (customer/client). This allows for clear communication and documentation of the scheduled appointments or services.

After the appointments are completed, you have the flexibility to convert the quotes to invoices. This streamlined process eliminates the need for manual data entry and ensures accuracy in billing and invoicing. By converting the quotes to invoices, you can efficiently generate invoices based on the scheduled appointments and bill the debtor (customer/client) for the services provided.

Overall, the bulk time entry feature in osFinancials' Calendar provides a convenient way to schedule and manage multiple appointments for salespersons and debtors (customers/clients). It enables the creation of quotes, smooth transition to invoices, and simplifies the workflow for tracking and billing services.

**To process bulk entries:**&nbsp;

1. On the **Default** ribbon, select **Calendar**.&nbsp;
1. Click on the **Bulk time entry** button.
1. Select the dates / days and a time slot to schedule the hours.
1. "*Week*" is the default option - it displays Monday to Sunday. If the "*Week*" option is not selected it will list only the selected date.
1. Select the Salesperson and enter the start dates and number of hours.&nbsp;
1. Select a stock item (usually sales no stock type)
1. Select the debtor (customer / client) account.&nbsp;

![Image](<lib/calendar-bulk-time-entry-options.png>)

8. Click on the **Create entries** button. A confirmation message will be displayed.

*"Entries created\!"*&nbsp;

&nbsp;![Image](<lib/calendar-bulk-time-entry-created-entries.png>)

9. Once the entries are created, these entries will be listed in the calendar. &nbsp;

![Image](<lib/calendar-bulk-time-entry-view.png>)

10. You may edit these events, set reminders and add notes.&nbsp;
10. On the top of the screen, click on the **Create invoices for hours** button.&nbsp;

![Image](<lib/calendar-create-invoices-hours-button-x.png>)

The "*Hours between date*" options screen is displayed:&nbsp;

![Image](<lib/calendar-bulk-time-select-hours-dates.png>)

12. Select the dates (from date and to date) and a Stock item.&nbsp;
12. Click **OK**. An information message will be displayed.&nbsp;

*"Quote(s) generated\!"*&nbsp;

A quote is generated for the entries on the Bulk time entries for the selected dates and number of hours.&nbsp;

14. You may edit the quote&nbsp;

![Image](<lib/calendar-bulk-time-qoute.png>)

15. Check and edit the quote, if necessary. You may wish to:
1) &nbsp;

   1) Enter a reference.
   1) Enter a message.
   1) Add comments and remarks, as needed.&nbsp;
   1) Change the number of hours and selling prices.
   1) Add discount, etc.

16. Click **OK** to save the quote.&nbsp;
16. Once these appointments bookings are done, you may print the quote and forward it to the debtor (customer / client). The quote needs to be converted to an invoice. &nbsp;
