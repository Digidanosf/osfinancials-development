# Context menu - Copy document

***

You may create exact copies of the selected document(s) of a specific document type. &nbsp;

**To create copies of selected documents:**&nbsp;

1. On **Documents** (Documents list screen) (**Default** ribbon), select the Document type, if the correct document type for sales documents (i.e. Invoices, Credit notes or Quotes) or purchase documents (i.e. Purchases, Supplier returns or Orders).
1. Select the Document(s), right-click on **Documents → Copy document** on the context menu. Exact unposted copies of the selected documents will be created for the same document type.

| ![Image](<lib/admon-tip.png>) | *If you need to select multiple documents, press the **Ctrl** or **Shift** key and click on each document, which you need to copy.* |
| --- | --- |


Once this is done, you may edit / post these documents.

![Image](<lib/documents-context-menu-copy-document.png>)

| ![Image](<lib/admon-note.png>) | *Once a document is copied, the selected source document from which it is copied, will be added to the third message line.*&nbsp; *You need to edit the first and second message line, if required.*&nbsp; *Note the following:* ***Date** - The system date will be used. You may edit the date, if required.* ***Message lines 1 and 2** - Document message (set in the **Setup → Documents** setup on the **Setup** ribbon) will also be added to the copied message of the selected source document.*&nbsp; ***Message line 3** - Up to 2 References for Document numbers (e.g. IN000022 (the latest document reference from which the document is copied will automatically be inserted before any existing document numbers on the source document (from which this document is copied). Only 5 digits of the 3 document number will be available.* &nbsp; ![Image](<lib/docs-copy-invoice-to-invoice-01.png>)&nbsp; |

You may need to check and edit these, if necessary.* *This is important if specific information for the stock item (such as, for example, series or serial numbers, etc.) may not be applicable in the copied document.* |
