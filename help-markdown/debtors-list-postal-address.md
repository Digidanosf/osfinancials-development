# Debtor listing - Postal address (Reports menu)

***

The Debtor listing - Postal address report will list the Account code, Name of the Debtor (account description), Postal address, Telephone 1 / 2, Fax, E-mail Address and Website.&nbsp;

# Debtor - List postal address report options

**To print a Debtor - List postal address report:**

1. On the **Reports** ribbon, select **Reports → Debtors → Listing**.&nbsp;
1. Select "*List postal address*".

&nbsp;![Image](<lib/reports-debtor-listing-postal-address-options.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a debtor (customer / client) account or a range of debtor (customer / client) accounts, to include in the report.
   1) **Sequence** - Select "*Debtor code, Description, Address 3* or *Postal code*" to print the report details in the order (sort) in which you wish to display the data.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The report will only be printed for those debtor (customer / client)&nbsp; accounts linked to the selected Debtor reporting group 1.

4. Click on the **OK** button.&nbsp;
