# Internet

***

This option enables you to configure your Internet settings so that you can send reports, statements, and lists to any person or organization via email. To do this, you will need to specify your email's outgoing mail server as set up by your Internet Service Provider on your computer system.&nbsp;

You can also test your email server and view the test log file from this option. It is necessary to perform internet setup in order to send document layout files. If the Internet setup is not completed, an information message will be displayed.

*"Internet setup information is incorrect\! Run Setup -\> Setup -\> Internet."*&nbsp;

If you wish to:

* Send document layout files form the [**Documents**](<debtor-tab-documents.md>) tab on **Debtors** form the [**Documents**](<creditor-tab-documents.md>) tab on **Creditors** (**Default** ribbon) via email to any person or institution.
* If you have set up any other users on the system, that user's User ID (that is logged on in this Set of Books) and their e-mail address will be displayed on this screen. &nbsp;

This option allows you to set up your Internet connection in order to send reports, lists and documents to any person or institution by e-mail. You need to specify your e-mail's outgoing mail server, as setup in your e-mail account on your computer system by your Internet Service Provider (ISP).&nbsp;

You need to then test your E-mail server, and view the test log file from this option, before you can send any reports, lists and documents via e-mail. The test to your e-mail server must be successful before this option will work.&nbsp;

**To configure the Internet:**&nbsp;

1. On the **Setup** ribbon, select **Setup → Internet**. &nbsp;

![Image](<lib/setup-internet.png>)

2. Enter your E-mail account's outgoing mail server's identification as provided by your Internet Service Provider (with whom you already have an account).
2. Enter your User ID and a valid Password as provided by your Internet Service Provider (with whom you already have an account).&nbsp;
2. Default Text is displayed in the "*Subject*" field on the "*Sending E-mail and Fax Report*" screen. This *Sending E-mail and Fax Report*" screen, is launched if you click on the **Print** button when the Send report to E-mail or Fax option is selected. You may over type this with your own text (subject), if necessary.
2. Click on the **Test Mail Server** button. This will test your e-mail connection. If successful, a similar result would be displayed:&nbsp;

*&nbsp;"Verifying osfinancials1@gmail.com on server smtp.saix.net*\
*Connected.*\
*Disconnected..."*

6. View the results of your settings for the e-mail server. If your e-mail server is set correctly, a similar result should be displayed:&nbsp;

*"Test Finished. Success ."*&nbsp;

7. Once the test is finished, a confirmation e-mail is sent to the Inbox of your system's default e-mail program.&nbsp;
7. Click on the **Apply** button.
7. Click on the **Close** button to close this screen.

| ![Image](<lib/admon-note.png>) | ***Server Name** - You may enter a name for your outgoing mail server, which can be up to 30 characters long and can include both letters and special characters.*&nbsp; ***User ID** - You may enter a user ID for your email account, which can be up to 20 characters long and can include both letters and special characters. If you have previously entered user IDs in the "**Setup → [Access control***](<access-control.md>)*" section, the user ID of the currently logged-in user will be displayed here. You cannot change user IDs in this field and can only modify them in the "**Setup → Access Control**" section.* ***E-Mail Address** - This field displays your registered email address, which can be up to 30 characters long and can include both letters and special characters. If your email address is not displayed, you need to create a user and enter the email address in the "**Setup → [Access control***](<access-control.md>)*" section. You can also enter your email address in the "**Setup → Company Info**" section on the [**Address***](<setup-company-info.md>) *tab.* |
