# Install osFinancials5 Updates - Manually

**Installation files / changelog link :** [osFinancials-Turbocash-Business-version](<https://www.osfinancials.org/en/productinformation/technical-information/changelog2/81-osfinancials-turbocash-business-version> "target=\"\_blank\"")&nbsp;

| ![Image](<lib/admon-warning.png>) | *It is important to make a backups of your Set of Books. It is strongly recommended that you make a backup copy of the entire installation folder of your existing installation osFinancials5 before proceeding with the update.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *The Updates option may manually be launched from **Help → Updates** (**Start** ribbon).*&nbsp; *The Update option will list all the installation files available for download. By default, the Update will be launched when starting osFinancials5. To disable the check for update at start, select this field and remove the tick.* ![Image](<lib/osfinancials5-help-menu-updates.png>) |
| --- | --- |


**To Install** **osFinancials5** **Updates from a downloaded file**

1. Once the osFinancials5 Update installation file is downloaded, select the file and open it. The "*Select Setup Language*" screen is displayed: \
![Image](<lib/osfinancials-install-setup-language-x.png>)

| ![Image](<lib/admon-tip.png>) | *The default installation language is set to English. "Nederlands" language is also available to launch the osFinancials5 install options.* &nbsp; |
| --- | --- |


2. Select your country and language and click **OK**. The "*License Agreement"* screen will be displayed.
2. Please read the agreement carefully. If you agree with the terms and conditions of the licence agreement, select the “*I accept the agreement*” option and click **Next**. The "*Select Destination Location"* screen is displayed:&nbsp;

![Image](<lib/osfinancials-install-destination-updates.png>)

| ![Image](<lib/admon-important.png>) | *The default path is C:/osFinancials5 on your system's default drive.* *This should be the same folder in which the Full version of osFinancials5 is installed..*&nbsp; |
| --- | --- |


4. Once finished, click **Next**. The "*Ready to Install"* screen is displayed: \
![Image](<lib/osfinancials-isntall-update-ready.png>)
4. Please check the settings.&nbsp;

| ![Image](<lib/admon-warning.png>) | *This is your last chance to change anything to be installed. If you are not satisfied with your selection, click on the **Back** buttons to change the Installation directory, or the Start menu folder or select / deselect the necessary options.* |
| --- | --- |


6. Click on the **Install** button. The osFinancials5 update installation process will start.&nbsp;
6. Once the osFinancials5 installation process is finished, the "*Completing the osFinancials 5 Setup Wizard"* screen is displayed:&nbsp;

![Image](<lib/osfinancials-isntall-update-finished.png>)

8. Please select or deselect (remove the tick) to **Launch osFinancials 5** - If this option is selected (ticked), the *osFinancials5* program will automatically be launched, when you click on the **Finish** button.
8. Once finished selecting or deselecting the necessary options, click on the **Finish** button. The *osFinancials5* program will be started if you did not remove (deselect) the tick on the "*Launch* *osFinancials 5"* option.
