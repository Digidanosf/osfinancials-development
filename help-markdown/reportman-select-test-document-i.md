# Reportman - Select test document id

***

You can regularly preview your document layout file on screen. This will point out possible errors as well as where you can make the document layout file neater by moving fields, enlarging, setting the font properties, etc.&nbsp;

When you want to preview a document layout file, the last or most recently processed document will be available for you to view or when you:&nbsp;

* click on the **Print preview** button from the "*Report man*" screen, or
* click the ![Image](<lib/reportman-design-icon-preview.png>) **Preview** report on screen icon) from the "*Report manager designer*" screen.&nbsp;

&nbsp;

When you are editing a document layout file and you want to preview the document layout file, you can click on the **Select test document id** button to select a specific document layout file.&nbsp;

You can select any sales documents (i.e. [Quotes](<quotes.md>), [Invoices](<invoices.md>) and [Credit notes](<credit-notes.md>)) or purchase documents (i.e. [Orders](<orders.md>), [Purchases](<purchases.md>) and [Supplier returns](<supplier-returns.md>)) document type.&nbsp;

These document types may be posted or unposted, confirmed or unconfirmed.

![Image](<lib/reportman-select-test-document.png>)

In the case of sales documents (i.e. [Quotes](<quotes.md>), [Invoices](<invoices.md>) and [Credit notes](<credit-notes.md>)), the document number, debtor (customer / client) account and total will list the document.&nbsp;

In the case of purchase documents (i.e. [Orders](<orders.md>), [Purchases](<purchases.md>) and [Supplier returns](<supplier-returns.md>)), the document number, creditor (supplier / vendor) account and total will list the document.&nbsp;

The selected document id (which was automatically generated when documents were created, copied or imported) will be displayed, for example 72 - IN000022.
