# Change stock code

***

**To change the stock code:**

1. Click on the **Change stock code** button (next to the "*Stock code*" field).&nbsp;
1. A confirmation message will be displayed.&nbsp;

*"Changing the stock code will result in all reports to use this code\! Are you sure you want to change the code?"* &nbsp;

| ![Image](<lib/admon-warning.png>) | *Make absolutely sure that the correct stock item is selected.*&nbsp; |
| --- | --- |


3. If you click **Yes**, the following screen is displayed:&nbsp;

![Image](<lib/stock-change-code.png>)

4. Enter a new code for the Stock item.&nbsp;
4. Click **OK**.
