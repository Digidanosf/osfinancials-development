# Context menu - Set reporting group 1/2

***

You may change the Document group 1 and/or Document group 2&nbsp; for a selected document or documents. These Groups can be changed for posted as well as unposted documents.&nbsp;

**To change the Document group 1 or 2 on documents:**&nbsp;

1. On **Documents** (Documents list screen) (**Default** ribbon), select the Document type, if the correct document type is not selected.
1. Select the Document(s), right-click on "*Document group 1*" or "*Document group 2*" context menu.

| ![Image](<lib/admon-tip.png>) | *If you need to select multiple documents, press the **Ctrl** or **Shift** key and click on each document, for which you need to change the document reporting group.* |
| --- | --- |


3. &nbsp;Select "*Empty*" or the available document group from the sub-menu. The "*Document group 1*" or "*Document group 2*" will change on documents and the reports.&nbsp;

![Image](<lib/documents-context-menu-change-group2.png>)
