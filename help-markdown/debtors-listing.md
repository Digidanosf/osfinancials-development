# Debtor listing (Reports menu)

***

The Debtor listing report will list the Account code, Name of the Debtor (account description), Delivery address, Telephone 1 / 2, Fax, E-mail Address and Website.&nbsp;

# Debtor listing - Listing report options

**To print a Debtor listing - Listing report:**

1. On the **Reports** ribbon, select **Reports → Debtors → Listing**.&nbsp;
1. Select "*Listing*".

&nbsp;![Image](<lib/reports-debtor-listing-listing-options.png>)

3. Select the following:&nbsp;
1) &nbsp;

   1) **From account / To** **account** - Select a debtor (customer / client) account or a range of debtor (customer / client) accounts, to include in the report.
   1) **Sequence** - Select "*Debtor code* or *Description*" to print the report details in the order (sort) in which you wish to display the data.
   1) **Reporting group 1** - Select a reporting group or more reporting groups. The report will only be printed for those debtor (customer / client)&nbsp; accounts linked to the selected Debtor reporting group 1.

4. Click on the **OK** button.&nbsp;

**Printed : Debtor list report**&nbsp;

![Image](<lib/reports-debtor-listing-listing-print.png>)
