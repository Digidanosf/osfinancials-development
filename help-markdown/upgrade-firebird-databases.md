# Upgraded to Firebird 2.1 ODS 11.1.

[Creating a blank osFinancials5 database](<create-books-advanced.md>), defaults to Firebird 2.1 ODS 11.1. &nbsp; Also implemented in TurboCASH5-3

**Firebird 1.5 is designated ODS 10.1**&nbsp;

All other older databases, including tutorials, is [Upgraded to Firebird 2.1 ODS 11.1](<firebird-database-version-upgrad.md>).

All databases (Sets of Books) available for download in the [Creation wizard](<sets-of-books-download-templates.md>), is updated to Firebird 2.1 ODS 11.1 and the start of the Financial year is set to 2022/XX/XX.&nbsp;

If the set of books for your language and or country, is not available in the books directory, you may select any of the available Firebird [Sets of Books and linked languages available as a free download](<sets-of-books-download-templates.md>).&nbsp;

All firebird database types available for download ([Create a Set of Books -default option](<create-books-default-download.md>)), is upgraded from firebird version 1.5 (ODS version - 10.1) to Firebird version 2.1 - (ODS version - 11.1)&nbsp;
