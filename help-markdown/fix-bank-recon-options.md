# FIXED - Bank recon options - label

***

**Resolved in osFinancials 5.1.0.189**

**Cash/Bank entry** (**Default** ribbon) - Bank account label changes to "read 21" / "read 58" / "read 61" When selecting a bank account from the list = looks like the Id of the bank account name.

It is replicated irrespective of the selection on the "*Default bank reconciliation method*" option on **Setup → System parameters** menu:

* Selected - [*Bank import plugin*](<bank-reconciliation-bank-stateme.md>) not activated &nbsp;
* Not selected - [Bank import plugin](<plugin-bank-import.md>) activated &nbsp;
