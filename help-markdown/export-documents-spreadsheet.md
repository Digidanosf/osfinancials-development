# Exported documents in Spreadsheets

***

Using this export feature, you may also export the data for sales documents (Invoices, Credit notes and Quotes) as well as purchase documents (Purchases, Supplier returns and Orders) in an open Set of Books to the available file formats.&nbsp;

All the information for each document as entered in **Documents** (**Default** ribbon), will be exported in the selected file format for the selected dates.&nbsp;

To illustrate the exported fields and data, the exported text file is opened in a spreadsheet.&nbsp;

Once the exported documents is saved in the "(\*.XLS) MS Excel SpreadSheet" file format, it will create 2 files:&nbsp;

1. [Headers](<export-documents-header-file.md>) file&nbsp;
1. [Lines](<export-documents-lines-file.md>) file&nbsp;

You may open these files with your spreadsheet application i.e Excel, LibreOffice, OpenOffice, Ultra Office, etc.&nbsp;
