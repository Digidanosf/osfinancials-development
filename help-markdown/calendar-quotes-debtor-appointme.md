# Create quotes - Debtor appointments

***

You may also create quotes for any of the six (6) categories of appointments scheduled with debtors (customers / clients).

**To create quotes for debtor (customer / client) appointments:**

1. On the **Calendar**, select any of the six (6) categories of debtor appointments and right-click.

![Image](<lib/calendar-quote-select-context-menu.png>)

2. On the context menu, select the “*Quote*” option. A new Quote entry screen, displaying all the information and details for the debtor (customer / client) account will be displayed.

![Image](<lib/calendar-debtor-appointment-quote.png>)

3. Enter and/or select the necessary fields and select a stock item. &nbsp;
3. Click on the **OK** button.

| ![Image](<lib/admon-note.png>) | *This Quote may be viewed on the following options:*&nbsp; *Document entry screen - Quotes document type - as Unposted (i.e. Not converted to an Invoice).*&nbsp; [***Projects***](<projects.md>) *on the list of a selected Project.*&nbsp; [***Documents***](<debtor-tab-documents.md>) *tab of the Debtor (customer / client) account. Once the Quote is confirmed (converted to an invoice and the Invoice is posted (updated to the ledger), it will be listed as a debit transaction on the [**Transactions***](<debtor-tab-transactions.md>) *tab of the Debtor account.*&nbsp; *You may need to edit the quote, e.g. add comments, change the quantities, etc. change the Project code (if projects are activated), before converting it to an Invoice.*&nbsp; |
