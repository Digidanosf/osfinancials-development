# Delete reports

***

You may delete a report from the [**User reports**](<user-reports-menu.md>) menu, or your own reports.

If a report, which contains parameters, is deleted both the report (\*.rep) and parameter (\*.dfm) files will be deleted.

| ![Image](<lib/admon-warning.png>) | *All reports which are required to manage the reporting is listed [Reports - File structure*](<reoorts-file-structure.md>) *and should not be deleted or changed.*&nbsp; *Do not select any of the standard reports in the Reports menu (stored in the .../plug\_ins\\reports\\systemreports folder). See [Reports - System reports - File names*](<reports-file-names-systemreports.md>) *should not be deleted or renamed.* |
| --- | --- |


**To delete a report:**

1. On the "*Report man*" screen, select the report you wish to delete.

![Image](<lib/reportman-delete-report1.png>)

2. Click on the **Delete** button. A confirmation message will be displayed.

*"Are you sure?"*&nbsp;

3. Only if you are absolutely sure, click **Yes** to delete the selected report.&nbsp;

| ![Image](<lib/admon-warning.png>) | *Once a report is deleted, the report is lost.*&nbsp; |
| --- | --- |
