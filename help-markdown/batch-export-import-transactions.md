# Export / Import batch transactions

***

The following options is available to export and import batch transactions:

* **Posted batches** - [**Import a posted batch**](<batch-import-a-posted-batch.md>) option (**F9:Process**) of the "*Batch entry*" screen. You may select a posted batch on the **Posted batches** tab of the "*Batch entry*" screen to import.&nbsp;
* **Spreadsheet Import Export** - osFinancials5/TurboCASH5 supports the "*Excel Workbook (\*.xlsx) file type* Spreadsheet import and export feature#8202;*.* &nbsp;
- &nbsp;

  - [**Spreadsheet export**](<batch-export-excel.md>) - You may export unposted batch transactions and save it as a "*Excel Workbook (\*.xlsx) file type.*
  - [**Spreadsheet import**](<batch-import-excel.md>) - You may import batch transactions from a saved "*Excel Workbook (\*.xlsx) file type.*

* **Unposted batches**&nbsp;

  * Export batch transactions - [**Write delimited file**](<batch-export-write-delimited-fil.md>) option (**F9:Process**) of the "*Batch entry*" screen. This will create a Comma Separated Value (\*.csv) file.&nbsp;
  * Import batch transactions - [**Delimited file** **import**](<batch-import-delimited-fille.md>) option (**F9:Process**) of the "*Batch entry*" screen. Select the Comma Separated file (\*.csv) file.&nbsp;
