# Translate Multi-language batch types

***

The Batch type names or descriptions, as entered in the **Setup → Batch Types** menu, are by default displayed on the following options: launched from the **Default** ribbon:

* **Batch Entry** menu.
* **Edit → Repeating transactions**.&nbsp;

It also displays on various reports, lookup and parameter screens.

| ![Image](<lib/admon-note.png>) | *To use multi-language Batch types, Language groups need to be created / added in **Setup → Groups** - **Languages** tab.* |
| --- | --- |


**To add translations to batch types:**

1. On the **Setup** ribbon, select **Setup → Batch types**.
1. Select the Batch type.

![Image](<lib/batch-type1.png>)

3. Click on the **translations** button.

![Image](<lib/batch-type.png>)

4. Select the Language and enter the name or description for each language.
4. Click on the save ![Image](<lib/record-icons-08.png>) icon to save the record.
4. Click on the **Close** button (on the translations screen).
4. Repeat the process for all Batch types.&nbsp;
4. When the process is completed, click on the **Close** button on the "*Batch type*" screen.

**Selecting batches**&nbsp;

After translations added, the batch names will be displayed in the Language&nbsp; set for the logged in user.
