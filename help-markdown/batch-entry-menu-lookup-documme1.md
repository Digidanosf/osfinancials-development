# Batch entry - Context menu - Lookup document with amount

***

The option is similar to the [**Lookup document**](<batch-entry-menu-lookup-docummen.md>) context menu option.

**To select an account from documents in a batch:**

1. Select a transaction in the batch.
1. Right-click and select the **Lookup document with amount** option on the context menu. The "*Document entry"* list screen for *Invoices* will by default be displayed. You may select any other document type (i.e. credit notes, quotes, purchases, supplier returns or orders).&nbsp;

![Image](<lib/batch-lookup-document.png>)

3. Select the document from the list. (You may use the sort and filter options to locate a specific document).
3. Click on the **OK** button. The transaction with the system date will automatically be inserted in the batch. The document number as well as the Your reference (if entered in documents) will be displayed in the Description column of the Batch entry screen. &nbsp;
