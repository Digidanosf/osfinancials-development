# Reportman output

***

**Output Icons** – See Data access configuration. If you cannot establish a database connection, you will not be able to preview or print the report.

* ![Image](<lib/reportman-design-icon-preview.png>) **Preview** – View the report on the screen. &nbsp;
* ![Image](<lib/reportman-design-icon-print.png>) **Print** – Opens the Print dialog on which you may select a printer, printer properties, select pages (if required).&nbsp;

**Preview**

Once you can view data (using the **Show data** button), you may click on the ![Image](<lib/reportman-design-icon-preview.png>) **Preview** icon on the icon toolbar of the "*Report manager designer*" screen to preview the report or document layout file. &nbsp;

![Image](<lib/reportman-preview-tutorial-invoice.png>)

| ![Image](<lib/admon-important.png>) | ***Preview in PDF format*** *It may be important to [check your document layout file in PDF format*](<report-designer-output1.md>)*, as PDF format for document layout files, may in most cases, be sent to debtors (customers / clients) and to creditors (suppliers / vendors).*&nbsp; *If there are any adjustments, you can edit the document.* |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *The data displayed in the preview of the invoice, should be your own data as entered in the **Company info** ([**Setup***](<setup-ribbon.md>) *ribbon) and **Debtors** (**Default** ribbon), and in your document header and document lines file.* &nbsp; |
