# Export - Documents grid

***

The **Export** button on the Document entry list screen allows you to export the detail of a selected document type to a CSV (Comma Separated Value) file type. The saved exported file will automatically open the "*Export.txt*" file in a default spreadsheet app (program) ([associated with the CSV (Comma Separated Values](<system-setting-default-csv-app.md>)) file type.&nbsp;

You may use the following options to filter the data which you need to include in the spreadsheet:

1. [**Document type**](<document-types.md>) - Select one of the six (6) document types (i.e. Invoices (default), Credit notes, Quotes, Purchases, Supplier returns or Orders) you need to export.
1. [**Document status**](<document-status.md>) - Select "*All, Posted, Unposted*" or "*Confirmed*" for selected document types (i.e. Invoices (default), Credit notes, Quotes, Purchases, Supplier returns or Orders) you need to export.
1. [**Search and filter options**](<documents-filter-search-options.md>) - **Search by** – Once you have selected a document type, you may filter and search by columns. For example, if you wish to export documents for a specific Salesperson, you may filter for that Salesperson.&nbsp;
1. [**Document groups**](<documents-filter-search-options.md>) - You may select any specific document groups for Document group 1 you need to export.

Once you have made your choices, click on the **Export** button.

On the "*Save As*" screen, select the folder to store the file and enter name.

Click on the **Save** button.

&nbsp;

![Image](<lib/documents-grid-export-spreadsheet-select.png>)

Once the file is saved, it will automatically be opened in your system's default spreadsheet program, e.g. Microsoft Excel.
