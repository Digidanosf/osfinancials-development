# Documents setup - General numbers

***

The "*General numbers*" option will only be available and displayed in a multi-user environment. Once users have been set up, in **Setup → [Access control**](<access-control-documents-tab.md>) (**Setup** ribbon). the "*General numbers*" option can be assigned specific users.

**To set the General numbers:**

1. Once users are add in **Setup → Access control** ([**Setup**](<setup-ribbon.md>) ribbon), select **Setup → Documents setup** on the [**Setup**](<setup-ribbon.md>) ribbon.

![Image](<lib/users-documents-setup-set-numbers.png>)

2. Select (tick) the "*General numbers*" option. A list of the users will be displayed. Select the user from&nbsp; the drop-down list; which osFinancials5/TurboCASH5 will use to generate document numbers.
2. Click **OK**.

| ![Image](<lib/admon-note.png>) | *If the "General numbers" field is not selected (ticked), osFinancials5/TurboCASH5 will generate the document numbers numerically.* |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *You may also select the user from the "Doc no" field on the **Documents** tab in **Setup → Access control** ([**Setup***](<setup-ribbon.md>) *ribbon).* ![Image](<lib/users-access-control-document-numbers.png>) |
