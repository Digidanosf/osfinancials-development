# Reports - systemreports - Reports menu file names

***

The standard reports on the **Reports** menu is stored in the “*...\\plug\_ins\\reports\\systemreports*” folder of your installation directory.&nbsp;

When printing reports using the “*Material*” theme, the icons in the “*Material*” theme folder will be displayed in menus and the icons in the “*Classic*” theme folder will be displayed on the report parameters screens and in Report man.

**To access the [Report designer**](<report-designer.md>)**:**

1. On the **Setup** ribbon, select [**Report designer**](<report-designer.md>). The [**Report designer**](<report-designer.md>) will list all "*systemreports*" files and report folders as follows:

&nbsp;![Image](<lib/report-designer-select.png>)
