# Uninstall osFinancials5

If you need to uninstall osFinancials5, go to the **Start** button on your Windows Taskbar and select osFinancials5.

| ***All apps option*** | ***Pinned apps options*** |
| --- | --- |
| ![Image](<lib/uninstall-windows-start-menu-all-apps.png>) | ![Image](<lib/uninstall-windows-start-menu-pinned-apps.png>) |


The **Program and features** - **Uninstall or change a program** of the Windows Control Panel will be launched.

![Image](<lib/uninstall-control-panel.png>)

| ![Image](<lib/admon-tip.png>) | *You may also double-click on the "Uninstall000.exe" file in the Installation folder of osFinancials5.* |
| --- | --- |


**To** **uninstall osFinancials5:**

1. Select **osFinancials5** and click on the **Uninstall** option.

![Image](<lib/Uninstall-osfinancials-confirm.png>)

2. Click on the **Yes** button to uninstall osFinancials5.&nbsp;
2. Wait for the process to complete. Once finished, the following information screen is displayed.&nbsp;

![Image](<lib/Uninstall-osfinancials-confirmed.png>)

4. Click on the **OK** button.&nbsp;
4. Once the Uninstall process is finished, you need to use File Explorer or your system's default file browser to select the osFinancials5 (or other installation directory).&nbsp;

| ![Image](<lib/admon-note.png>) | *The files which have been created, or edited by the user after the original installation, will not be deleted or removed. Some examples are;* *books folder - Any Sets of Books created, copied, saved, etc.* *Bin/Themes - Any changes to themes (e.g. background pictures, icons, etc.).* *plug\_ins/reports/ - any reports added after the original installation, etc.* *Registration.ini file - Only if you have registered this version of osFinancials5.* *You may need to copy these files somewhere else on your system. If you do not wish to keep these files, you may delete the entire osFinancials5 folder.*&nbsp; |
