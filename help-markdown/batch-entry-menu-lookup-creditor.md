# Batch entry - Context menu - Lookup creditor

***

In addition to the lookup function in the "*Account*" column in the "*Batch entry*" screen you may choose to lookup a creditor (supplier / vendor) account from the "*Creditor accounts*" screen. On the "*Creditor accounts*" screen, you can use the filter and search options to select a specific creditor (supplier / vendor) account.&nbsp;

**To select a creditor account in a batch:**

1. Select a transaction in the batch.
1. Right-click and select the **Lookup creditor** option on the context menu. The "*Creditor accounts"* list screen is displayed:&nbsp;

![Image](<lib/lookup-creditor.png>)

3. Select the creditor (supplier / vendor) account. (You may use the sort and filter options to locate a creditor (supplier / vendor) account).
3. Click on the **OK** or **Open** button. The selected account will be transferred to the account column.
