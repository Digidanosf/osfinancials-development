# Documents setup - Quotes

***

**To set documents - Quotes:**

1. On the **Setup** ribbon, select **Setup *→* Documents** **setup**.
1. Click on the **Quotes** tab.

![Image](<lib/setup-documents-quotes.png>)

3. Enter or select the following options:
1) &nbsp;

   1) **Next quote no.** - It is recommended that the Quote number be prefixed (e.g. QU000001) to identify the document type.&nbsp;
   1) **Layout file** - If you have a customised layout file, you may select the layout file.

| ![Image](<lib/admon-read.png>) | *The universal Layout file (one file to print all [document types*](<document-types.md>)*) is by default set for all Sets of Books.* *See - [Document layout files*](<document-layout-files.md>) *to assist you in selecting a layout files.* |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | *The Layout files set in the Documents setup, is a global setting. If you need to set a different layout file for a specific debtor (customer / client), you may select it on the **Delivery address** tab of **Debtors** (**Default**-ribbon).* *You may also change the layout file when printing documents.* |
| --- | --- |


1. &nbsp;
   3. **Layout file 2** - This field is optional. If a layout file is selected, the document will print twice using this layout file. If you have a customised layout file, you may select the layout file. Otherwise, you may select one of the layout files listed above.
   3. **Quote heading** - This field is not used on the document layout files, but it is recommended that Quote should be displayed or entered.
   3. **Quote message** - Enter a global message to be printed on all Quotes.

4) Click on the other tabs, if necessary. Once finished, click on the **OK** button to save your changes.
