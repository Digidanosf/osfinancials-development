# Delete Set of Books

***

If you want to remove a Set of Books permanently from your osFinancials5/TurboCASH5 system, you should be absolutely certain that you want to do so, as all data contained within the Set of Books will be permanently lost. Before deleting a Set of Books from your system, it is important to ensure that you have printed all necessary reports and retained any supporting documents, such as slips, invoices, etc., which will help substantiate your profit (or loss) from the business. This information may be required for your tax returns or other legal purposes. Therefore, it is crucial to make sure that you have all the necessary documentation before deleting a Set of Books.

| ![Image](<lib/admon-warning.png>) | *This operation can not be reversed\! If you have deleted a Set of Books by mistake, you need to restore it from a backup file\!*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-warning.png>) | *It is important to remember that even if you are a small business, you are required to keep the necessary records for at least five years starting from the date you received the last applicable tax return. This is to comply with audit and record-keeping requirements.*&nbsp; *Additionally, if there are other statutory prescriptions that apply to your business, you should also comply with them. Therefore, it is crucial to maintain accurate and complete records and ensure that you retain them for the required period of time to meet legal obligations.* |
| --- | --- |


| ![Image](<lib/admon-important.png>) | *After you have taken all necessary steps and are absolutely certain that you want to proceed with the deletion of a Set of Books, you may then delete it from your system.* |
| --- | --- |


**To delete a Set of Books:**&nbsp;

1. On the [**Start**](<start-ribbon.md>) ribbon, select **Delete Set of Books**. The following "*Set of Books"* screen, reflecting the list of all Set of Books, is displayed:&nbsp;

![Image](<lib/books-delete-x.png>)![Image](<lib/books-d-x.png>)

| ![Image](<lib/admon-important.png>) | *&nbsp;The location or full path of the selected Set of Books is displayed.&nbsp; Before you click the **Delete** button, make absolutely sure that the correct Set of Books is selected.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-tip.png>) | ***Red - Font colour*** *This indicates that a Set of Books (elsewhere on your system) is deleted or the Set of Books folder is moved somewhere else on your system. It may also indicate that the drive or network on which the Set of Books is stored, is not available or mounted.*&nbsp; ***Blue - Font colour***&nbsp; *This indicates that a Set of Books is not located in the directory where osFinancials5/TurboCASH5 is installed, but elsewhere on your system.* |
| --- | --- |


2. Select the Set of Books, and click on the **Delete** button. The following confirmation message is displayed:&nbsp;

*Are you sure you want to delete the MY-BOOKS set of accounts?"*

3. If you are sure, click on the **Yes** button. Another confirmation message is displayed:&nbsp;

*The Set of Books MY-BOOKS will permanently be removed from disk\!*\
*Do you want to cancel the deletion?*

4. On this screen you have the option to cancel the deletion process. If you click on the **Yes** button, the process will be terminated. If you click on the **No** button, the Set of Books will be permanently deleted.
4. Wait for the deletion process to finish. The following information message is displayed:

*"Set of Books C:\\osFinancials5\\books\\MY-BOOKS\\books.fdb deleted\!" or* \
*"Set of Books C:\\TurboCASH5\\books\\MY-BOOKS\\books.fdb deleted\!" or*&nbsp;

6. Click on the **OK** button.

| ![Image](<lib/admon-note.png>) | *Once the Set of Books is deleted, the contents of the folder with the name of the Set of Books maybe emptied and the files of the Set of Books should be transferred to the Recycle Bin of your Operating System.* ![Image](<lib/books-delete-recycle-bin.png>) *You may need to: -*&nbsp; *The Set of Books should by default be deleted. You may check in the books folder (e.g."C:\\osfinancials5\\books\\MY-BOOKS") if the deleted Set of Books folder was deleted. You may delete the folder, if not deleted. (If this action is not done, you will not be able to create or save a Set of Books with exactly the same name as the Set of Books which was deleted).*&nbsp; *Empty the Recycle Bin of your Operating System.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-important.png>) | *If the Set of Books is not too large for your Recycle Bin, all the files for the deleted Set of Books will be transferred to your Recycle Bin. You then need to click on the **Recycle Bin** icon on your desktop and empty the Recycle Bin to make more space available on your hard disk.* &nbsp; |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *Should you have any Set of Books open on the system whilst deleting a Set of Books, osFinancials5 will automatically close any Set of Books on your system at this stage.*&nbsp; |
| --- | --- |
