# Context menu - Edit account

***

You may edit a debtor (customer / client) account for sales documents (i.e. Invoices, Credit notes or Quotes) or edit a creditor (supplier / vendor) account for purchase documents (i.e. Purchases, Supplier returns or Orders) using the document details.&nbsp;

Once the changes on an account is saved, it will be reflected in the document header of unposted documents.&nbsp;

| ![Image](<lib/admon-note.png>) | *If a document is posted and the account is edited, the changes will not be reflected in the document header.*&nbsp; |
| --- | --- |


# Edit a debtor (customer / client) account

**To edit a debtor (customer / client) account from the documents list screen:**

1. Select the Invoices, Credit notes or Quotes document type.
1. Select the document to edit the account.
1. Right-click and select **Documents → Edit account** option on the context menu. The "*Debtor accounts"* edit form with the details of the debtor (customer / client) account for sales documents (i.e. Invoices, Credit notes or Quotes) is displayed:

![Image](<lib/documents-context-menu-edit-debtor.png>)

4. Enter and/or edit the fields, as necessary.
4. Click on the **OK** or **Save** button.

# Edit a creditor (supplier / vendor) account

**To edit a creditor (supplier / vendor) account from the documents list screen:**

1. Select the Purchases, Supplier returns or Orders document type.
1. Select the document to edit the account.
1. Right-click and select **Documents → Edit account** option on the context menu. The "*Creditor accounts"* edit form with the details of the creditor (supplier / vendor) account for purchase documents (i.e. Purchases, Supplier returns or Orders) is displayed:

![Image](<lib/documents-context-menu-edit-credtor.png>)

4. Enter and/or edit the fields, as necessary.
4. Click on the **OK** or **Save** button.
