# Value stock account - Stock grid reports

The "*Value stock account*" report lists the total value of the stock control account. If the "*Show details*" option is selected, it will list the the value of the selected stock item. &nbsp;

# Value stock account options

![Image](<lib/stock-sidebar-value-stock-account-options.png>)

# Print single item (context menu)&nbsp;

Parameter - "*Show detail*" leave blank to print the only the total of the selected stock item.

Parameter - "*Show detail*" select to print the stock item code, item description, quantities and the total value of the selected stock item.

![Image](<lib/stock-sidebar-value-stock-account-parm.png>)

Prints directly to the Reportman preview screen.&nbsp;
