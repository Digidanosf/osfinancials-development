# Creditor - Context menu - Set Creditor Group 1/2

***

You may change the Creditor group 1 and/or Creditor 2&nbsp; for selected creditor (supplier / vendor) account(s).&nbsp;

**To change the Creditor group 1 or 2:**&nbsp;

1. On the **Creditors** (Creditors list screen) (**Default** ribbon), select the creditor (supplier / vendor) account(s).

| ![Image](<lib/admon-tip.png>) | *If you need to select multiple* creditor (supplier / vendor) account(s), *press the **Ctrl** or **Shift** key and click on each* creditor (supplier / vendor) account#8202;*, for which you need to change the creditor reporting group.* |
| --- | --- |


2. Right-click and select "*Creditor group 1*" or "*Creditor group 2*" on the context menu.&nbsp;
2. Select "*Empty*" or the available creditor group from the sub-menu. The "*Creditor group 1*" or "*Creditor group 2*" will change on reports.&nbsp;
