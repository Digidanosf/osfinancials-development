# Budgets - Budget tab (Reports ribbon)

***

**To process your Budget figures:**

1. On the **Reports** ribbon, select **Budgets**. The **Budget** tab will be displayed.
1. **Period** - Select a period on the **Per period** tab. You may click on the **Free selection** tab to select a specific date or a range of dates to include budget figures.&nbsp;

![Image](<lib/budget-02.png>)

3. **Refresh** - This button will populate the Budget figures for the selected period or selected dates.&nbsp;
3. **Cost centre 1** - Budgets may be entered for [Cost centres](<cost-centres.md>) (Cost centre 1), if activated.
3. **Cost centre 2** - Budgets may be entered for [Cost centres](<cost-centres.md>) (Cost centre 2), if activated.
3. **Projects** - Budgets may be entered for [Projects](<projects.md>), if activated.
3. **Reference** - "*BUDG*" is the default reference. You may enter your own reference (up to 15 characters).
3. **Description** - "*Auto budget*" is the default description. You may enter your own description (up to 35 characters).&nbsp;
3. **Percentage** - Tick this option and specify a percentage to adjust the budget figures.&nbsp;
3. **Delete all** - If any budget figures exists, a confirmation message&nbsp; will be displayed.&nbsp;

*"Delete data for this period?"*

If you click Yes, the budget figures for the selected period or dates will be deleted.

11. **Copy** -
1) &nbsp;

   1) **From Budget Previous year** -&nbsp; If you already have budget figures for the previous financial year, select this option to populate last year's budget figures.&nbsp;
   1) **From Actual Previous year** -&nbsp; If you already have posted batch and document transactions for the previous financial year, select this option to populate last year's the actual totals of posted transactions.&nbsp;
