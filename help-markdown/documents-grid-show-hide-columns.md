# Show / Hide columns

***

By default, all thirty-four (36) columns will be displayed and listed in the Document entry list screen for each document type.&nbsp;

The columns is grouped in to two (2) bands:

You may use the filter options to find a specific document, or all documents with the same values, in any of the thirty-six (36) columns. These columns are as follows:

1. **Left band** - Lists the first four (4) columns, i.e., id (document id), Document number, Date of document, and Account description.&nbsp;
1. **Data band** - List the rest of the columns Due date, Accid (Account id), Printed, Amount inclusive, Tax amount, Reference, Document group 1, Document group 2, Account no., Salesperson, Amount exclusive, Status, Message line 1, Message line 2, Message line 3, Delivery address 1, Delivery address 2, Delivery address 3, Delivery address country, Postal address 1, Postal address 2, Postal address 3, Postal code, Country, Invoice address Company, Invoice address Contact, Delivery address Company, Delivery address Contact, E-mail address, Project, and Due (due / outstanding amount).

# Hide / Show bands&nbsp;

**Hide left band** - Remove the tick in the first tick box. This will hide the id, Document no. Date and Description, columns.&nbsp;

**Show left band** - Tick the first block. This will add the hidden columns on the left band.

**Hide data band** - Remove the tick in the second tick box. This will hide all columns in the data band.

**Show data band** - Tick the second block. This will add all the hidden columns on the left band.

# Hide / Show columns

By default, all thirty-six (36) columns will be displayed and listed in the Document entry list screen for each document type.&nbsp;

You may select to hide or show any column, as required.

**To hide or show columns:**

Click on the second block - Column headings - A list of the fields will be listed.&nbsp;

Remove the ticks for the columns to hide or select the columns you need to display.&nbsp;
