# Context menu - Change account

***

You may change the account for unposted documents.

**Sales documents** - Select a different debtor (customer / client) account for unposted sales documents (i.e. Invoices, Credit notes or Quotes).&nbsp;

**Purchase documents** - Select a different creditor (supplier / vendor) account for unposted purchase documents (i.e. Purchases, Supplier returns or Orders). &nbsp;

# Change a debtor (customer / client) account

**To change a debtor (customer / client) account for a unposted document:**

1. Select the Invoices, Credit notes or Quotes document type.
1. Select the document to change the account for.
1. Right-click and select **Documents → Change account** option on the context menu. The "*Debtor accounts"* list screen will list only the active debtor (customer / client) accounts:

![Image](<lib/documents-select-debtor.png>)

4. Select a different account.&nbsp;
4. Click on the **Open** or **OK** button. &nbsp;

# Change the creditor (creditor / vendor) account

**To change the creditor (creditor / vendor) account for a unposted purchase document:**

1. Select the Purchases, Supplier returns or Orders document type.
1. Select the document to change the account for.
1. Right-click and select **Documents → Change account** option on the context menu. The "*Creditor accounts"* list screen will list only the active creditor (supplier / vendor) accounts:

![Image](<lib/documents-select-creditor.png>)

4. Select a different account.&nbsp;
4. Click on the **Open** or **OK** button.&nbsp;
