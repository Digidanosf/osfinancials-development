These spreadsheet reports are stored in the “*...\\plug\_ins\\reports\\excelreports*” folder of your installation directory.&nbsp;

Selected reports will be opened in the powerful spreadsheet feature in osFinancials5/TurboCASH5.&nbsp;

![Image](<lib/ribbon-reports-spreadsheet-menu.png>)

When printing reports using the “*Material*” theme, the icons in the “*Material*” theme folder will be displayed in menus and the icons in the “*Classic*” theme folder will be displayed on the report parameters screens and in "*Report man*".&nbsp;
