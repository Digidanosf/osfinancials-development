# Insert transactions / comments in documents

***

Sometimes you may have entered a number of transactions in a document, and you wish to insert a transaction or a line in a specific place in the document. You may need to do this to insert another transaction, or to insert a comment for a selected transaction.

**To insert a line for a transaction or a comment in a document:**

1. Select the row or line into which you wish to insert a transaction or enter a comment.
1. Click on the **\^N:Insert** icon (**Ctrl** + **N**). A new blank line or row above a selected transaction or comment line will be inserted.
1. Enter the transaction in the inserted row, or change the inserted row to a Comment.
