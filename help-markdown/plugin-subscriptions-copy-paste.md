# Subscriptions - Copy / Paste - Debtors

***

# Copy and Paste / Subscriptions&nbsp;

You may copy a subscription from a selected debtor account and paste it to another debtor account.&nbsp;

## Copy Subscriptions&nbsp;

On the **Debtors** grid (**Default** ribbon), select a debtor account and right-click on the account.&nbsp;

On the context menu, select **Copy from plugin → Subscriptions**.

![Image](<lib/subscriptions-copy.png>)

&nbsp;

## Paste subscriptions&nbsp;

Once you have copied subscriptions, go to the **Debtors** grid (**Default** ribbon), select a debtor account and right-click on the account.&nbsp;

On the context menu, select **Paste from plugin → Subscriptions**.

![Image](<lib/subscriptions-paste.png>)

A similar confirmation message will be displayed:

*Erase old data and copy for plugin \&Subscriptions from accountD1MD-AVI Mr. Mike Davies*

Click **Yes** to proceed or **No** to cancel the Paste option.

# Check / Edit Copied Subscriptions

Once Subscriptions is copied, it will add all the details of existing subscriptions to the **Subscriptions** tab of the Debtor account.&nbsp;

![Image](<lib/subscriptions-copied-debtor.png>)

You need to review the copied subscriptions:

**Delete subscriptions** which is not applicable to the debtor account.

**Edit copied subscriptions** applicable to the debtor account:

Select a subscription and check or edit the subscription. On the **Details** option of the selected subscription, different start date and settings may need to be changed. Change these where necessary.&nbsp;
