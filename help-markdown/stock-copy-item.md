# Copy from a Stock item

***

This option will open a Stock screen with the necessary data from a selected Stock item. Before you can create the new Stock item with the copied data, you need to at least enter a Stock code and Save it. You may enter and/or select the options for the blank fields, as necessary. If necessary you may also edit or change any of the copied data.&nbsp;

| ![Image](<lib/admon-note.png>) | *The data that will be copied from a selected Stock item, may depend on the on the Stock item type.*&nbsp; |
| --- | --- |


**To create a new stock item - Copy from item:**&nbsp;

1. On the **Default** ribbon, select **Stock items** (**F12**).&nbsp;
1. Select the Stock item from the list.
1. Make absolutely sure that it is the correct Stock item to be copied.

| ![Image](<lib/admon-tip.png>) | *&nbsp;To view more details of the selected Stock item, you may:*&nbsp; *Click on the **Edit** button - or double-click on the selected stock item.*&nbsp; *Use the horizontal scroll bar to view the data of the selected stock items in columns.*&nbsp; |
| --- | --- |


4. Click on the **Copy from item** button.&nbsp;

| ![Image](<lib/admon-tip.png>) | *&nbsp;You may right-click and select the **Copy → Stock item** from the context menu.*&nbsp; |
| --- | --- |


The Stock item screen with basically all the data from the selected Stock item screen will be displayed as follows: \
![Image](<lib/stock-copy-02.png>)

5. Enter the new Stock code.
5. Enter the necessary details in the fields, as required. You may also edit the copied data, where necessary.
5. Click on the **Save** button to save this stock item.

| ![Image](<lib/admon-note.png>) | ***It copy the following fields :***&nbsp; *Stock description* *Barcode* *Manufacturer*&nbsp; *Extra description*&nbsp; *Expected quantity (values will be set to 0 (zero)*&nbsp; *Type (You need to select the [stock item type*](<stock-item-types.md>)*)* *Units*&nbsp; *Apply invoice discount (tick)*&nbsp; *Stock group 1* *Stock group 2*&nbsp; *Preferred supplier 1 account and stock code.* *Preferred supplier 2 account and stock code.* *Ledger*&nbsp; *Tax classes (Sales account / Cost of sales account / Stock control account)*&nbsp; *Weight (Net and Gross)*&nbsp; *Selling prices 1/2/3*&nbsp; *Latest cost*&nbsp; *Reorder level*&nbsp; *Reorder at*&nbsp; *Minimum*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-important.png>) | ***What it does not copy:***&nbsp; *Stock code*&nbsp; *Ledger*&nbsp; *Average cost*&nbsp; *Total cost*&nbsp; *Quantity on hand*&nbsp; *Last invoice*&nbsp; *Document groups*&nbsp; *Transactions*&nbsp; *Image (does not copy the correct image but the default image)*&nbsp; *Multigroup*&nbsp; *Descriptions (Language) (copies the groups but not the Descriptions and the text).*&nbsp; |
| --- | --- |
