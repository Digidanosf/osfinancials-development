# Debtor statements

***

The following debtor statements are available:&nbsp;

* [**Debtor statements**](<debtor-statement.md>) - The statement will print all transactions. It does not use the Open item link.
* [**Debtor statements - Outstanding**](<debtor-statement-open-item.md>) - This statement will print only the outstanding transactions. Transactions (e.g. invoices, etc. Transactions linked in the Open item link, will not be listed).&nbsp;

| ![Image](<lib/admon-note.png>) | *The credit transactions which are linked to specific debit transactions of the same value, will not be displayed. If a transaction for a different amount is linked to a specific transaction, only the outstanding amount (difference) will be listed.* |
| --- | --- |


| ![Image](<lib/admon-note.png>) | *Before you wish to print Debtor statements, you may need to check the following:* *Statement messages are entered correctly.* *Banking details of your company is entered in the **Company info** (**Setup** ribbon).* *That no outstanding transactions which is not linked in [Open item processing*](<open-item-link-debtor-accounts.md>)*.* |
| --- | --- |


# Debtor statements - Global messages

**Enter global messages to print on all debtor statements:**

1. On the **Setup** ribbon, select **Documents setup**.
1. Select the **Statement setup** tab.

![Image](<lib/documents-setup-statement-setup.png>)

3. Enter the message in up to 3 lines to display on the Debtor statements.&nbsp;
3. Click on the **OK** button.&nbsp;

# Debtor statements - Messages for specific accounts

**Enter statement messages for specific debtor accounts:**&nbsp;

1. On the **Default** ribbon, select **Debtors** (**F4**).
1. Select the debtor (customer / client) account.
1. On the **Delivery address** tab, enter a message of up to 3 lines.&nbsp;
1. Click on the **Save** button. &nbsp;
