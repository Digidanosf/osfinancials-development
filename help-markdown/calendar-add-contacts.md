# Add / Create contacts - Calendar

***

**To add (create) a contact:**&nbsp;

1. Click on the **Contacts** icon (if the Contacts screen is not displayed).&nbsp;
1. To create a new contact, simply right-click on any area of the Contacts screen. On the context menu, select the **Add contact...** option. The Untitled - Contact screen is displayed:&nbsp;

![Image](<lib/calendar-contacts-new-main-tab.png>)

3. Enter a Name for the contact. This Name will be displayed in the title bar, and on the Contact screen once this contact is saved.&nbsp;
3. Enter the title, address, city, state zip / postal code, country, company, position, and select the category of the contact (business, clients, family, other or personal).&nbsp;
3. Click on the **Telephone** tab. The following screen is displayed:&nbsp;

![Image](<lib/calendar-contacts-new-telephone-tab.png>)

6. Enter up to two telephone numbers, a fax number and an e-mail address, if available / applicable. You may also enter two additional fields for the following:&nbsp;

   1. Number for - select one of the following: Mobile, Other, (phone), Other Fax, Pager, Primary, Radio, Telex, TTY/TDD, Work or Work Fax, then enter the number for the selected option.&nbsp;
   1. Number for - select one of the following: Assistant (default), Call-back, Car, Company, Home, Home Fax, IDSN or Mobile, then enter the number for the selected option.&nbsp;

6. Once finished entering the necessary details, click on the **Notes** tab, if you need to enter additional information for the contact. &nbsp;
6. Click on the **OK** button.&nbsp;
