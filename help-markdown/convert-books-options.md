# Converting Sets of Books

***

***Add Sets of Books***&nbsp;

***Already have Set of Books***&nbsp;

TurboCASH4, TurboCASH5 (4.0.0.969), osFinancials4 (4.0.0.1093) Sets of Books do not need to be converted to osFinancials5/TurboCASH5.&nbsp;

You may copy the Sets of Books using File Explorer to the "***books***" directory of your osFinancials5/TurboCASH5 installation. &nbsp;

You may also [Restore a Set of Books](<restore-set-of-books.md>) from a valid Backup file in osFinancials5/TurboCASH5.

## New Set of Books&nbsp;

If you are new to osFinancials5/TurboCASH5, you may download an existing template for your country and language from the [Let osFinancials help you to create a Set of Books - Download](<create-books-default-download.md>) option.&nbsp; See a list of available Firebird Sets[ of Books and linked languages available as a free download](<sets-of-books-download-templates.md>). &nbsp;
