# Export Documents - Export grid

***

The **Export** button of the sidebar of the "*Documents*" list screen, will create a text file (tab delimited), which will open in your system's default Microsoft Excel (if installed on your system).&nbsp;

# Export options

**To export documents:**

On **Documents** (**Default** ribbon), select the [document type](<document-types.md>).&nbsp;

Select the [document status](<document-status.md>); i.e. "*All, Posted, Unposted*" or "*Confirmed*".

| ![Image](<lib/admon-note.png>) | *Select the Filter and search options in the [Filter column*](<documents-filter-search-options.md>) *and click on the **Filter** button to include only those filter criteria to be included in the Export file.*&nbsp; |
| --- | --- |


&nbsp;

![Image](<lib/docs-sidebar-export-01.png>)

&nbsp;

Click on the **Export** button. The "*Save As*" screen is displayed:

![Image](<lib/docs-sidebar-export-02.png>)

&nbsp;

Enter a name to save the file.&nbsp;

| ![Image](<lib/admon-note.png>) | *"**Save as type :**" options will be "CSV". A text (\*.txt) file type will automatically be created with your file name.*&nbsp; |
| --- | --- |


| ![Image](<lib/admon-warning.png>) | *If you do not enter a unique name in the selected folder, and save the export.csv or export.txt file name, you may replace existing (previously exported) files.*&nbsp; *If an existing exported file is opened, and you are trying to save a export file with the same name, a similar error message as the following will be displayed:* *Cannot create file "F:\\exports\\your-file-name.txt". The process cannot access the file because it is being used by another process.* |
| --- | --- |


&nbsp;

Click on the **Save** button.&nbsp;

# Example : Exported text file - LibreOffice Calc

If the exported text file does not open in Microsoft Excel; or if you do not have Microsoft Excel installed on your system, you may open the file with OpenOffice, LibreOffice, Ultra Office, etc.&nbsp;

Locate the exported text file in your System's File Explorer.

In this example, **Open with \> [**LibreOffice Calc](<https://www.libreoffice.org/discover/calc/> "target=\"\_blank\"") is selected. The "*Text Import*" screen is selected.&nbsp;
